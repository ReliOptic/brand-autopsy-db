# 04. 콘텐츠 DNA — Amgen (AMGN)

> 섹터: Health Care - Biotechnology | 분석일: 2026-04-12

---

## 1. 콘텐츠 철학

Amgen의 콘텐츠는 **"과학이 먼저, 이야기는 그 다음"** 원칙으로 구성된다.
임상 데이터와 분자 생물학적 메커니즘이 모든 주장의 기반이 되며,
환자 스토리와 감성 콘텐츠는 그 위에 레이어로 쌓인다.
규제 환경(FDA/EMA 광고 가이드라인)이 콘텐츠 구조를 강하게 제약한다.

## 2. 콘텐츠 필러 (5대 주제 축)

| 필러 | 설명 | 주요 형식 | 오디언스 |
|------|------|----------|---------|
| **1. 과학·혁신** | 신약 개발 과정, 분자 기전, 플랫폼 기술 | 백서, 학술 포스터, R&D 파이프라인 업데이트 | 투자자, KOL, 언론 |
| **2. 임상 근거** | 피벗 임상 결과, 실사용 데이터(RWE), 안전성 프로파일 | 저널 리프린트, 디테일링 자료, 컨그레스 발표 | HCP, PBM |
| **3. 환자 경험** | 치료 전후 삶의 질 변화, 환자 증언 | 영상 케이스 스터디, 환자 앰배서더 인터뷰 | 환자·보호자, 대중 |

## 4. 내러티브 구조 (스토리 아키타입)

### 아키타입 A: "정복자 (Conqueror)" — 불치 질환을 이기는 과학
```
문제 설정: "이 질환은 오랫동안 치료 불가능했습니다"
→ 과학적 돌파구: "Amgen 연구팀은 새로운 분자 경로를 발견했습니다"
→ 임상 근거: "X 임상시험에서 Y% 감소를 입증했습니다"
→ 환자 결과: "[환자 이름]은 이제 [일상 활동]을 할 수 있습니다"
→ 미래 전망: "우리는 더 많은 환자를 위해 연구를 계속합니다"
```

### 아키타입 B: "접근성의 수호자 (Access Guardian)" — 치료 접근성 확대
```
문제 설정: "많은 환자가 치료를 받지 못하고 있습니다"
→ 근거: "비용/인식 부족/지리적 장벽"
→ Amgen 해결책: "환자 지원 프로그램 / 바이오시밀러 포트폴리오"
→ 결과: "X만 명의 환자가 치료를 받았습니다"
```

### 아키타입 C: "혁신의 개척자 (Innovation Pioneer)" — IR·채용 대상
```
문제 설정: "바이오테크의 다음 10년을 정의할 도전이 있습니다"
→ Amgen의 포지션: "우리는 40년 혁신의 기반 위에 서 있습니다"
→ 플랫폼 기술: "BiTE, 바이오시밀러, 소분자 경구제 확장"
→ 비전: "과학이 상상하는 모든 질환을 치료 가능하게"
```

## 5. 어조 가이드라인 (톤 오브 보이스)

### HCP 대상
- 주장 → 반드시 참조 문헌 번호 병기
- 비교 우위 주장 → Head-to-head 데이터 또는 "based on [study]" 명시
- 수식어 → "significant", "clinically meaningful", "statistically" 선호
- 금지 표현 → "best", "superior" (데이터 없이), "breakthrough" (단독)

### 환자 대상
- 문장 길이 → 20단어 이하 권장
- 의학 용어 → 반드시 괄호 내 쉬운 설명 병기
- 능동태 → "Prolia helps protect your bones" (수동태 기피)
- 감정 → 공감 우선, 해결책으로 마무리 (공포로 끝내지 않음)

### 투자자 대상
- 데이터 밀도 → 높음 (숫자, %, 시장 규모 필수)
- 미래 전망 → Safe Harbor 문구 선행 필수 (공식)
- 경쟁 언급 → 중립적·사실 기반
- 파이프라인 → 개발 단계 명확 표기 (Phase 1/2/3)

## 6. 금지어 / 필수어 (콘텐츠 버전)

### 금지어
1. **"Revolutionary"** — 과대 주장, 대신 "innovative" 또는 "novel"
2. **"100% effective"** — 절대적 효능 주장 금지
3. **"No side effects"** — 임상 데이터와 배치되는 단순화
4. **"Proven to cure"** — FDA 광고 규정 위반 리스크
5. **"Better than [경쟁사 제품명]"** — 비교 광고 규제 위반 (직접 지명 금지)

### 필수어
1. **"Important Safety Information"** — 모든 제품 광고 필수 병기 (공식)
2. **"Ask your doctor"** — DTC 광고 call-to-action 표준 문구 (공식)
3. **"In clinical trials"** — 효능 주장 전 반드시 맥락 제공
4. **"Results may vary"** — 환자 개인차 고지 의무
5. **"Amgen is committed to"** — 기업 약속 문구의 공식 오프닝 패턴
