# 06. 채널 플레이북 — Broadcom (AVGO)

## 채널 전략 개요

Broadcom의 채널 전략은 **직접 영업(Direct) 우위 + 기술 커뮤니티 중심 풀(Pull) 마케팅**의 이중 구조다.
최종 구매는 대부분 직접 계약으로 성사되지만, 기술 신뢰도는 컨퍼런스·개발자 문서·파트너 에코시스템을 통해 사전 구축된다.

---

## 채널 맵

```
인지 (Awareness)
├── 산업 컨퍼런스 (OFC, Hot Chips, SC, CES)
├── 기술 미디어 (EE Times, The Register, AnandTech)
└── 투자자 IR (어닝콜, 투자자 데이모)

고려 (Consideration)
├── broadcom.com 기술 백서·데이터시트 다운로드
├── 개발자 포털 / SDK 문서
├── 파트너 채널 (VAR, SI, 클라우드 마켓플레이스)
└── Gartner / IDC 리서치 인용

전환 (Conversion)
├── 직접 영업 팀 (Global Account Manager)
├── POC (Proof of Concept) 프로그램
└── 라이선스 협상 (엔터프라이즈 ELA)

유지·확장 (Retain & Expand)
├── 기술 지원 포털 (TAC)
├── Broadcom Support 커뮤니티
├── 연간 기술 컨퍼런스 (VMware Explore 등)
└── 갱신·업셀 영업 사이클
```

## 채널별 상세 플레이북

### 채널 1: 공식 웹사이트 (broadcom.com)

| 항목 | 내용 |
|------|------|
| 역할 | 기술 정보 허브, 제품 발견, 리드 생성 |
| 주요 방문자 | 엔지니어, 제품 평가 중인 아키텍트 |
| 핵심 콘텐츠 | 데이터시트, 백서, 애플리케이션 노트, SDK |
| CTA 우선순위 | 1) 문서 다운로드 2) 영업 문의 3) 파트너 찾기 |
| SEO 키워드 | "ethernet switch asic", "vmware licensing", "broadcom network chip" (추정) |
| 성과 지표 | 백서 다운로드 수, 영업 문의 전환율, 개발자 포털 등록 수 |

**Steal Sheet — 웹사이트 레이어 3개**
1. 제품 페이지 상단 "성능 수치 3개" 고정 배치 — 방문 5초 내 핵심 가치 전달
2. 데이터시트 다운로드를 게이트(이메일 수집) 없이 제공 — 엔지니어 마찰 제거 (추정)
3. 검색 기능을 부품 번호(Part Number)·모델명 기준으로 최적화 — 기술자 직접 탐색 지원

**Steal Sheet — 컨퍼런스 레이어 3개**

2. 전시 부스에 라이브 데모 장비 설치 — 엔지니어가 직접 만지고 확인하는 경험 제공
3. 컨퍼런스 직후 48시간 내 발표 자료·논문 공식 웹사이트 업로드 (추정)

### 채널 3: 직접 영업 (Direct Sales)

| 항목 | 내용 |
|------|------|
| 구조 | 글로벌 계정 매니저(GAM) + 기술 영업 엔지니어(TSE) 페어링 (추정) |
| 대상 고객 | 하이퍼스케일 CSP, 포춘 500 엔터프라이즈, Tier-1 통신사 |
| 영업 사이클 | 12–36개월 (엔터프라이즈 규모에 따라 상이) (추정) |
| 주요 도구 | POC 키트, ROI 계산기, 레퍼런스 아키텍처 패키지 (추정) |
| 핵심 전략 | 기술 챔피언(엔지니어) 확보 후 경영진 에스컬레이션 (추정) |

**Steal Sheet — 직접 영업 레이어 3개**
1. 초기 기술 미팅에 TSE 동행 필수 — 엔지니어링 신뢰 없이는 GAM 단독 진행 불가 (추정)
2. POC 단계에서 고객 환경 맞춤 벤치마크 수행 — 범용 수치 대신 고객사 워크로드 결과 제시 (추정)
3. 계약 갱신 6개월 전 업셀 플랜 착수 — VMware 라이선스 재편 이후 특히 중요 (추정)

### 채널 4: 파트너 에코시스템

| 파트너 유형 | 역할 | 주요 파트너 (예시) |
|------------|------|-------------------|
| 시스템 통합사 (SI) | 설계·구축·마이그레이션 서비스 | Accenture, Deloitte, Wipro (추정) |
| 부가가치 재판매사 (VAR) | 중소·중견 기업 채널 커버리지 | Arrow Electronics, Avnet (추정) |
| 클라우드 마켓플레이스 | SaaS/소프트웨어 유통 | AWS Marketplace, Azure Marketplace (추정) |
| OEM 파트너 | 칩 내장 서버·스위치 제조 | Dell, HPE, Cisco, Juniper (공식) |
| 기술 얼라이언스 | 공동 솔루션 검증 | NVIDIA, Intel, Red Hat (추정) |

**Steal Sheet — 파트너 채널 레이어 3개**
1. OEM 파트너의 제품 출시 발표와 Broadcom 칩 사양을 동시 공개 — 공동 마케팅 극대화
2. 파트너 인증 프로그램(Broadcom Advantage Partner) 단계별 혜택 명확화 — SI 투자 유인 (추정)
3. 클라우드 마켓플레이스 리스팅에 90일 무료 평가판 포함 — VMware 소프트웨어 신규 고객 진입 장벽 낮춤 (추정)
