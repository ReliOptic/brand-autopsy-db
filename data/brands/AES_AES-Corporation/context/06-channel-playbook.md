# AES Corporation — 채널 플레이북

## 채널 전략 개요

AES의 채널 믹스는 B2B·B2G 의사결정자 대상 전문성 구축과 기관 투자자 대상 신뢰 관리를 이중 축으로 운영된다. 대중 소비자 채널보다 전문 미디어·직접 영업·IR 채널에 우선 투자하며, 디지털은 콘텐츠 배포보다 관계 심화 도구로 활용한다.

---

## 채널 아키텍처

```
Owned (소유 채널)
├── AES.com (기업 웹사이트)
├── LinkedIn (기업 페이지)
├── YouTube (기술·프로젝트 영상)
├── 연간보고서·지속가능성 보고서
└── Investor Relations 포털

Earned (획득 채널)
├── 에너지 전문 미디어 (Bloomberg, S&P Global, Wood Mac)
├── ESG 평가사 (MSCI, Sustainalytics, CDP)
├── 컨퍼런스 연설·패널
└── 애널리스트 리포트

Paid (유료 채널)
├── LinkedIn 스폰서 콘텐츠
├── 에너지 전문지 디지털 광고
├── 컨퍼런스 스폰서십
└── Google Search (브랜드·기술 키워드)

Partner (파트너 채널)
├── Fluence 공동 마케팅
├── Siemens 기술 파트너십 PR
└── 현지 자회사 (AES Indiana, AES Ohio 등) 채널
```

## 채널별 상세 플레이북

### 채널 1: 기업 웹사이트 (aes.com)

**역할**: 브랜드 허브, 기술 정보 제공, IR 데이터 집약

**핵심 방문자 유형**
- 잠재 C&I 고객 (솔루션 탐색)
- 기관 투자자 (IR 데이터 수집)
- 정부 파트너 (회사 신뢰도 검증)
- 채용 후보자 (기업문화 평가)

**Steal Sheet — 웹사이트**

| 항목 | 내용 |
|------|------|
| 최우선 CTA | "솔루션 상담 신청" / "Investor Relations" |
| 콘텐츠 우선순위 | 프로젝트 케이스 스터디 → 기술 솔루션 → ESG 데이터 |
| SEO 핵심 키워드 | "renewable energy PPA", "battery storage", "clean energy solutions" (추정) |
| 업데이트 주기 | 뉴스룸 주 2회, 프로젝트 페이지 분기 1회 (추정) |
| 성과 지표 | 솔루션 페이지 체류 시간, IR 문서 다운로드 수, 상담 신청 전환율 |
| 금지 패턴 | 슬라이더 중심 홈페이지, 수치 없는 마케팅 카피 |

### 채널 2: LinkedIn

**역할**: B2B 리드 너처링, 기술 사고 리더십, 인재 브랜딩

**계정 유형**
- AES Corporation 기업 페이지 (공식)
- AES 임원 개인 계정 (CEO·CSO 등 사고 리더십)

**콘텐츠 믹스 (추정)**

| 유형 | 비율 | 예시 |
|------|------|------|
| 프로젝트 마일스톤 | 30% | "X국 Y MW 프로젝트 완공" |
| 기술 인사이트 | 25% | "배터리 저장이 그리드 안정성에 미치는 영향" |
| ESG·정책 논평 | 20% | COP·IEA 보고서 반응 |
| 채용·문화 | 15% | 직원 스토리, 다양성 지표 |
| 파트너십·수상 | 10% | Fluence 공동 발표, 업계 어워드 |

**Steal Sheet — LinkedIn**

| 항목 | 내용 |
|------|------|
| 최적 게시 시간 | 화·수·목 오전 8~10시 (미 동부 기준, 추정) |
| 포스트 형식 | 리치 텍스트 + 단일 이미지 > 링크 프리뷰 |
| 해시태그 전략 | #CleanEnergy #EnergyTransition #Sustainability #AES (공식) |
| 임원 증폭 | CEO 게시물 24시간 내 직원 리쉐어 독려 (추정) |
| 성과 지표 | 팔로워 성장률, 인게이지먼트율(목표 3%+), 리드 생성 수 |
| 금지 패턴 | 실적 발표 없는 성과 자랑, 정치적 논쟁 개입 |

### 채널 3: YouTube / 영상 콘텐츠

**역할**: 기술 복잡성 해소, 프로젝트 현장감 전달, 채용 브랜딩

|----------|------|------|------|

| "Project Spotlight" | 5~8분 현장 다큐 | 분기 1회 | 정부·투자자 |
| "People of AES" | 3~5분 직원 인터뷰 | 월 1회 | 채용 후보자 |
| Investor Day 녹화 | 60~90분 발표 | 연 1회 | 기관 투자자 |

**Steal Sheet — 영상 콘텐츠**

| 항목 | 내용 |
|------|------|
| 첫 5초 훅 | 수치 또는 현장 충격 컷 (정전 예방 순간 등) |
| 자막 | 100% 제공 (접근성 + 무음 시청 대응) |
| 썸네일 스타일 | AES Blue 배경 + 흰 텍스트 + 현장 이미지 |
| CTA | 영상 말미 "솔루션 더 알아보기" 링크 |
| 성과 지표 | 시청 완료율(목표 40%+), 클릭률, 구독자 전환 |
| 금지 패턴 | 과도한 기업 홍보 영상, 실제 현장 없는 CG 전용 |

- ESG / CDP 연간 보고서 (공식)
- NDR (Non-Deal Roadshow) — 투자자 직접 미팅 (추정)

**Steal Sheet — IR 채널**

| 항목 | 내용 |
|------|------|
| 핵심 메시지 프레임 | 재생에너지 전환 → 수익 가시성 향상 |
