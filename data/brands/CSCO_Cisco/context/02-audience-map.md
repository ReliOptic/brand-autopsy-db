# 02. 오디언스 맵 — 시스코 (Cisco Systems, CSCO)

## 오디언스 개요

시스코의 구매 의사결정은 단일 인물이 아닌 "구매위원회(Buying Committee)" 구조로 작동한다. CIO·CISO·네트워크 아키텍트·조달팀이 각기 다른 언어로 승인 과정에 참여한다. 각 페르소나에 맞는 메시지 레이어링이 필수다.

---

## 핵심 페르소나

|------|------|
| 직함 | Chief Information Security Officer (CISO) |
| 연령/배경 | 45~55세, 금융·통신·제조 대기업, 보안 컨설팅 또는 군/경찰 사이버 부서 출신 |

| 시스코와의 접점 | Cisco SecureX → XDR, Cisco Talos 구독, Firepower NGFW, Umbrella DNS 보안 |

## 금지어 리스트 (오디언스 커뮤니케이션)
- "완벽한 보안" — CISO에게 허위 약속은 신뢰 즉시 붕괴. "침해 면적 최소화"로 대체 (추정, CISO 인터뷰 패턴 기반)
- "쉬운 설정" — 네트워크 아키텍트는 복잡도를 이미 알고 있다. "자동화로 반복 작업 제거"로 대체 (추정)
- "ROI 보장" — CIO에게 검증 불가 약속은 역효과. "고객사 A의 22% TCO 절감 사례"로 대체 (추정)
- "업계 표준" — 모든 벤더가 동일 표현 사용. 구체적 표준명(ISO 27001, IETF RFC)으로 대체
- "미래 지향적" — 추상적 미래 언어. 도입 후 6개월·12개월 기대 성과로 구체화

## 필수어 리스트 (오디언스 커뮤니케이션)
- "[고객사명] 사례" — 모든 클레임에 실제 레퍼런스 동반. 익명이라도 업종·규모 명시 (공식, Cisco Case Studies)
- "Talos 인텔리전스" — CISO 대화의 신뢰 앵커. 하루 2,000억 건 위협 분석 수치 동반 (공식)
- "MTTR / MTTD" — 아키텍트·CISO 공통 KPI 언어. 수치 비교 없이 단독 사용 금지
- "TCO" — CIO 설득의 핵심 단위. 라이선스+운영+교육+통합 비용 전체를 포함해 산출 (추정)
- "규제 컴플라이언스" — CISO·CIO 공통 구매 트리거. 적용 규제명(ISMS-P, GDPR, SOC2) 명시 (공식)

### Steal Sheet — 마케터가 훔쳐갈 3가지

**1. 동일 제품을 3개 언어로 팔아라 — 페르소나별 번역 레이어**
Cisco Catalyst Center는 아키텍트에게 "YANG/REST API 자동화"로, CISO에게 "정책 일관성으로 침해 면적 축소"로, CIO에게 "운영 인력 30% 절감"으로 팔린다(추정). 제품은 하나, 메시지는 셋이다. **액션**: 우리 핵심 제품의 핵심 가치를 기술(How)·보안(Risk)·비즈니스(ROI) 3개 언어로 각각 재작성하라.

**2. 구매위원회의 '거부권자'를 먼저 설득하라**
B2B 구매는 가장 열정적인 지지자가 아닌 가장 소극적인 반대자에 의해 막힌다. 시스코는 CISO(보안 리스크), CFO(예산), 법무팀(컴플라이언스) 각각을 위한 별도 문서를 제공한다(공식 Trust Center, Compliance Hub). **액션**: 우리 세일즈 팩에 "거부권자 FAQ 시트"를 추가하라. 반대 이유 5가지와 데이터 기반 반박을 한 장에 정리한다.

**3. '기술 챔피언'을 무료로 키워라 — 자격증이 마케팅이다**
Cisco Certification(CCNA·CCNP·CCIE)은 전 세계 수십만 명의 시스코 전도사를 만든다(공식). 자격증 보유자는 이직해도 시스코를 추천한다. 교육 투자가 장기 브랜드 자산이 된다. **액션**: 우리 제품의 무료 인증 과정이나 배지 프로그램을 만들어라. 기술 챔피언 1만 명은 세일즈 인력 100명보다 강하다.
