# 06 채널 플레이북 — Marriott International (MAR)

> 섹터: Consumer Discretionary | 세그먼트: Hotels & Lodging

---

## 1. 채널 전략 개요

**MAR 채널 철학 (추정)**
> "올바른 채널에서, 올바른 타이밍에, 올바른 메시지를 — 모든 터치포인트가 Bonvoy 생태계로 수렴한다."

**채널 우선순위 원칙**
1. **직접 채널 우선** (Direct > OTA): OTA 수수료 절감, 고객 데이터 확보
2. **로열티 중심**: 모든 채널이 Bonvoy 가입·활성화로 연결
3. **브랜드별 채널 특화**: 단일 채널 전략이 아닌 브랜드 DNA에 맞는 채널 믹스

---

## 2. 채널 아키텍처

```
MAR 채널 생태계
│
├── 소유 채널 (Owned)
│   ├── marriott.com + 브랜드별 도메인
│   ├── Marriott Bonvoy 앱
│   ├── 이메일 마케팅 (CRM 기반)
│   └── 호텔 내 디지털 접점 (키오스크, 앱)
│
├── 유료 채널 (Paid)
│   ├── 검색 광고 (Google Ads, Naver 등)
│   ├── 소셜 광고 (Meta, Instagram, TikTok)
│   ├── 디스플레이 / 프로그래매틱
│   └── OTA 유료 노출 (Expedia, Booking.com)
│
├── 획득 채널 (Earned)
│   ├── PR / 미디어 커버리지
│   ├── 인플루언서 협업
│   ├── UGC (사용자 생성 콘텐츠)
│   └── 리뷰 (TripAdvisor, Google Reviews)
│
└── 파트너 채널 (Partner)
    ├── 항공사 FFP 연계 (AA, DL, UA 등, 공식)
    ├── 신용카드 파트너 (Chase Bonvoy, Amex, 공식)
    ├── 여행사 / MICE 에이전시
    └── 기업 출장 관리 (Concur, SAP 등)
```

## 3. 채널별 상세 플레이북

### 3-A. Marriott.com / 브랜드 웹사이트

**목적**: 직접 예약 전환, 브랜드 경험 제공

| 항목 | 내용 |
|------|------|
| 핵심 KPI | 직접 예약 비율, 전환율, 평균 예약 단가 |
| 타깃 | 고의향 탐색자, Bonvoy 멤버 |
| 콘텐츠 타입 | 목적지 영감, 특별 패키지, Bonvoy 혜택 강조 |
| 최적화 포인트 | 검색 필터 UX, 멤버 로그인 유도, 모바일 체크아웃 간소화 |
| 브랜드 차별화 | Ritz-Carlton: 에디토리얼 매거진 레이아웃 / W: 다크 테마 몰입형 |

**직접 예약 인센티브 (공식)**
- Bonvoy 포인트 추가 적립
- 회원 전용 요금 (Member Rate)
- 모바일 체크인/체크아웃
- 객실 업그레이드 요청

---

### 3-B. Marriott Bonvoy 앱

**목적**: 로열티 활성화, 여정 전반 서비스, 재방문 유도

| 항목 | 내용 |
|------|------|
| 핵심 KPI | MAU, 앱 내 예약 비율, 푸시 알림 오픈율 |
| 핵심 기능 (공식) | 모바일 키, 체크인/체크아웃, 포인트 조회, 컨시어지 |
| 개인화 엔진 | 과거 숙박 이력 기반 추천, 선호 객실 기억 |
| 푸시 알림 전략 | 체크인 24시간 전 업그레이드 오퍼, 목적지 도착 로컬 팁 |
| 충성도 피드백 루프 | 포인트 잔액 시각화, 다음 등급까지 박수 표시 |

---

### 3-C. 이메일 마케팅 (CRM)

**목적**: 재방문 유도, 포인트 활성화, 개인화 오퍼

| 항목 | 내용 |
|------|------|
| 세그먼트 | Bonvoy 등급별, 마지막 숙박 브랜드별, 선호 목적지별 |
| 발송 빈도 (추정) | 주 1–2회 (캠페인) + 트리거 이메일 (예약·체크아웃 후) |
| 핵심 이메일 타입 | 예약 확인, 체크인 전 환영, 체크아웃 후 감사, 포인트 만료 알림 |
| 개인화 레벨 | 이름 호칭 + 선호 브랜드 + 과거 목적지 + 포인트 현황 |
| A/B 테스트 우선순위 | 제목 라인, CTA 텍스트, 이미지 vs. 텍스트 헤비 |

**이메일 성과 벤치마크 (추정, 호텔 업계 기준)**
- 오픈율 목표: 25–35%
- CTR 목표: 3–6%
- 예약 전환율: 1–3%

### 3-E. 인플루언서 & 파트너십 채널

**인플루언서 티어 전략 (추정)**

| 티어 | 팔로워 규모 | 브랜드 적합 | 활용 방식 |
|------|-----------|-----------|---------|
| Mega (100만+) | 1–5명 | Ritz-Carlton, W | 대형 캠페인, 브랜드 앰배서더 |
| Macro (10–100만) | 10–20명 | 전 브랜드 | 목적지 론칭, 시즌 캠페인 |
| Micro (1–10만) | 50–100명 | Westin, Courtyard | 니치 웰니스·비즈니스 커뮤니티 |
| Nano (1만 이하) | 다수 | 전 브랜드 | 로컬 목적지, 진정성 UGC |

**파트너십 채널 (공식)**
- Chase Sapphire × Bonvoy: 신용카드 적립 포인트 연계
- 항공사 FFP: United MileagePlus, Delta SkyMiles, AA AAdvantage 등
- Lyft: 이동 포인트 적립 파트너
- Uber: 공항 이동 연계 (추정)

### 3-F. OTA (온라인 여행사) 채널

**OTA 전략 원칙 (추정)**
- OTA는 신규 고객 획득 채널로 활용 — 재방문은 직접 채널로 전환
- OTA 예약 고객에게 Bonvoy 가입 유도 (체크인 시)
- OTA 노출은 유지하되 직접 예약 대비 인센티브 차별화

| OTA 플랫폼 | 활용 전략 |
|-----------|---------|
| Expedia / Hotels.com | 프리미엄 파트너십, 노출 순위 최적화 |
| Booking.com | 유럽·아시아 신규 시장 획득 |
| Agoda | 아시아·태평양 특화 |
| Ctrip / Trip.com | 중국 시장 접근 |
