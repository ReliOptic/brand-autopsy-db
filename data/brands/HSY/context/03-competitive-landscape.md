# 03 경쟁 환경 분석 — Hershey Company (HSY)

## 경쟁 구도 개요

미국 제과·스낵 시장은 Hershey, Mars, Mondelez 3강 체제로 운영된다 (추정). Hershey는 미국 초콜릿 시장 점유율 약 36%로 1위를 유지하나 (추정), 글로벌 시장에서는 Mars와 Mondelez에 비해 규모가 작다. 최근 프리미엄 크래프트 초콜릿 브랜드와 건강 스낵 카테고리의 부상이 새로운 위협으로 등장 중이다.

## 주요 경쟁사 분석

### 1. Mars, Incorporated

| 항목 | 내용 |
|------|------|
| 유형 | 비상장 사기업 (공식) |
| 본사 | 버지니아주 맥린, 미국 (공식) |
| 주요 브랜드 | Snickers, M&M's, Milky Way, Twix, Skittles, Starburst (공식) |
| 추정 매출 | 약 $450억 (전체 사업부 포함) (추정) |
| 강점 | 글로벌 규모, 다각화된 포트폴리오, 마케팅 예산 (추정) |
| 약점 | 비상장으로 투명성 제한, Hershey 대비 미국 시장 집중도 낮음 (추정) |

**Hershey 대비 차별점**
- M&M's의 캐릭터 IP는 장기 브랜드 자산 (추정)
- Snickers의 "배고프면 스니커즈" 글로벌 캠페인은 단일 메시지 일관성의 교과서 (추정)
- 미국 내 Reese's vs. Snickers 구도에서 Hershey가 일부 우위 (추정)

### 2. Mondelez International (MDLZ)

| 항목 | 내용 |
|------|------|
| 유형 | 상장사 (NASDAQ: MDLZ) (공식) |
| 본사 | 일리노이주 시카고, 미국 (공식) |
| 주요 브랜드 | Oreo, Cadbury, Toblerone, Milka, Chips Ahoy!, Trident (공식) |
| 연간 매출 | 약 $362억 (2023) (공식) |
| 강점 | 진정한 글로벌 커버리지, 프리미엄-매스 듀얼 포지셔닝, Cadbury 브랜드 파워 (추정) |
| 약점 | 미국 내 초콜릿 점유율 Hershey 대비 낮음, 비스킷 의존도 높음 (추정) |

**Hershey 대비 차별점**
- Cadbury는 영국·아시아에서 압도적 감성 자산 보유 (공식)
- Oreo의 크로스카테고리 확장(아이스크림, 음료 콜라보) 전략 선진적 (추정)
- 미국 초콜릿 시장에서는 Hershey가 Mondelez를 크게 앞섬 (추정)

## Steal Sheet — 경쟁 환경 레이어

| # | 훔칠 전략 | 적용 방법 |
|---|----------|----------|
| 1 | **카테고리 언어 소유** — "PB+초콜릿 = Reese's"처럼 카테고리 자체를 브랜드명으로 만들기 | 신규 카테고리 진입 시 제품명이 아닌 카테고리 언어를 먼저 정의 |
| 2 | **글로벌 위협을 미국 향수로 방어** — Ferrero·Lindt 등 글로벌 프리미엄 공세에 "Made in America" 서사로 대응 | 미국 제조·고용·역사 스토리를 경쟁 방어 내러티브로 활용 |
| 3 | **배틀카드 기반 판매 지원** — 유통 채널(Walmart, Target, CVS) 바이어 대상 카테고리 데이터로 선반 점유율 방어 | 경쟁사 대비 카테고리 기여도 데이터를 슬롯 협상 무기로 사용 |
