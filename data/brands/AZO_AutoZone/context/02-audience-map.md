# AutoZone (AZO) — 오디언스 맵

## 오디언스 세그먼트 개요

AutoZone의 고객군은 크게 두 축으로 나뉜다: **DIY(자가 수리)** 고객과 **DIFM(전문가 수리 의뢰)** 고객. 매출 비중은 DIY 약 80%, DIFM(상업용) 약 20%로 추정된다. (추정)

---

## 페르소나

### 페르소나 1: 주말 DIY 정비사 — "손으로 하는 김민준"

| 항목 | 내용 |
|------|------|
| 나이 | 32–45세 |
| 직업 | 중간 관리직 / 기술직 (연봉 5만~8만 달러) |
| 거주지 | 교외 주거지역, 차고 보유 |
| 차량 | 2015–2020년식 픽업트럭 또는 SUV |
| 학력 | 전문대 또는 대졸 |

**심리 프로파일**

- 차를 직접 고치는 것에서 성취감과 절약 만족감을 동시에 얻는다
- YouTube 튜토리얼을 먼저 보고 매장을 방문한다
- 브랜드보다 **부품 호환성**과 **가격**을 우선시한다
- "내가 할 수 있다"는 자기효능감이 구매 동기의 핵심

**구매 여정**

```
문제 인식 → YouTube/Reddit 검색 → AutoZone 앱/웹사이트 확인 → 매장 방문 → AutoZoner 상담 → 구매 → 설치
```

**핵심 니즈**

- 정확한 차량-부품 호환성 확인 도구 (Loan-A-Tool 포함)
- 합리적 가격 + 보증
- 설치 가이드 제공

**채널 선호도**

- YouTube 광고 (하우투 콘텐츠 주변)
- AutoZone 앱 푸시 알림
- 매장 내 디스플레이

**불만 트리거**

- 재고 없음 → 경쟁사로 이탈
- 직원의 전문성 부족
- 웹사이트 부품 검색 오류

### 페르소나 2: 소규모 정비소 오너 — "프로 정비사 이상철"

| 항목 | 내용 |
|------|------|
| 나이 | 40–58세 |
| 직업 | 독립 자동차 정비소 운영 |
| 거주지 | 도시 외곽 상업지구 |
| 차량 | 업무용 밴 + 개인 픽업 |
| 학력 | 직업훈련학교 졸업 |

**심리 프로파일**

- 시간이 곧 돈 — 부품 조달 속도가 매출에 직결된다
- 장기 공급처 관계를 중시, 신뢰할 수 있는 파트너를 원한다
- AutoZone Commercial 계정을 통해 대량 구매 및 외상 결제 이용 (추정)
- 가격보다 **재고 가용성**과 **배송 속도**가 더 중요

**구매 여정**

```
고객 차량 입고 → 진단 → AutoZone Commercial 앱 재고 확인 → 당일 배송 주문 → 수령 → 정비 완료 → 청구
```

**핵심 니즈**

- 당일 배송 보장
- 상업용 계정 혜택 (할인율, 외상 한도)
- 부품 반품 정책 유연성

**채널 선호도**

- AutoZone Commercial 전용 앱
- 영업 담당자 직접 연락
- 이메일 프로모션 (대량 주문 할인)

**불만 트리거**

- 배송 지연 → 정비 일정 차질
- 상업용 할인 혜택 불투명
- O'Reilly/NAPA 대비 재고 열세

### 페르소나 3: 예산 절약형 첫 차 오너 — "알뜰 운전자 박지현"

| 항목 | 내용 |
|------|------|
| 나이 | 22–30세 |
| 직업 | 사회초년생 / 프리랜서 |
| 거주지 | 도시 근교 아파트, 차고 없음 |
| 차량 | 2010–2016년식 중고 세단 (혼다, 도요타 등) |
| 학력 | 대재 또는 대졸 |

**심리 프로파일**

- 자동차 지식이 부족하고 수리점 방문이 불안하다
- 비용이 가장 큰 의사결정 요인
- 스마트폰으로 모든 정보를 먼저 탐색한다
- 처음 방문하는 AutoZone에서 AutoZoner의 친절에 크게 영향받는다

**구매 여정**

```
경고등 켜짐 → 구글/틱톡 검색 → AutoZone 무료 진단 서비스 인지 → 매장 방문 → AutoZoner 상담 → 간단한 DIY 부품 구매 또는 전문점 연결
```

**핵심 니즈**

- 무료 배터리/엔진 진단 서비스
- 쉬운 설명 + 비전문가 친화적 언어
- 저렴한 OEM 대체 부품

**채널 선호도**

- TikTok / Instagram (자동차 관리 팁 콘텐츠)
- 구글 로컬 검색 ("내 근처 자동차 부품")
- AutoZone 앱 (첫 방문 후 재구매)

**불만 트리거**

- 기술 용어 남발로 인한 소외감
- 필요 이상의 부품 추천 (불신 유발)
- 주차 불편, 매장 접근성 부족
