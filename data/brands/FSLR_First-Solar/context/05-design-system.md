# 05 디자인 시스템 — First Solar (FSLR)

## 디자인 철학

First Solar의 디자인 언어는 **"기술적 정밀함 + 환경적 책임감"** 의 시각적 표현이다. 과도한 장식을 배제하고, 데이터와 정보의 명확한 전달을 우선한다. 네이비-화이트 중심의 팔레트는 신뢰와 전문성을 상징하며, 앰버 계열 액센트가 에너지와 태양광을 연상시킨다.

### 컬러 사용 우선순위

**Primary Palette (핵심 3색)**
- `#00355F` Deep Navy — 브랜드 정체성의 중심
- `#FFC423` Solar Amber — 에너지·태양광 연상 액센트
- `#F8F9FA` Off White — 클린 배경

**Secondary Palette (보조 4색)**
- `#212529` Charcoal — 텍스트
- `#0D6EFD` Action Blue — 인터랙션
- `#198754` Success Green — 성과 지표
- `#6C757D` Medium Gray — 보조 정보

### 타이포그래피 원칙

- **헤딩**: HelveticaNowDisplay / Acumin Pro — 현대적이고 기술적인 정밀함
- **본문**: Franklin Gothic URW 계열 — 가독성 높은 산세리프, 기업 신뢰감
- **데이터/코드**: 모노스페이스 — 기술 문서의 정확성 강조
- **컨덴스드 서체 활용**: 좁은 레이아웃(배너, 네비게이션)에서 정보 밀도 유지

## 디자인 원칙

| 원칙 | 설명 |
|------|------|
| 데이터 우선 (Data-First) | 수치와 차트가 디자인의 중심, 장식은 최소화 |
| 클린 레이아웃 | 충분한 여백, 정보 계층 명확화 |
| 신뢰 시그널 | 로고, 인증 마크, 파트너 로고를 전략적 배치 |
| 반응형 우선 | 태블릿/모바일에서도 기술 데이터 가독성 유지 |
| 접근성 | 고대비 텍스트(`#212529`/`#F8F9FA`), WCAG AA 준수 (추정) |
