# 06. 채널 플레이북 — Colgate-Palmolive (CL)

## 채널 전략 개요

Colgate-Palmolive는 **"대중 도달 + 전문가 신뢰 + 디지털 전환"** 의 3중 채널 전략을 구사한다.
- **오프라인 대중 채널**: TV·마트·약국 — 브랜드 인지 및 구매 전환
- **전문가 채널**: 치과·동물병원 — 권위 보증 및 처방 전환
- **디지털 채널**: SNS·이커머스·앱 — 관여도 유지 및 재구매 Lock-in

## 채널별 상세 플레이북

### 채널 1 — TV / OTT (공중파·케이블·넷플릭스 광고)

| 항목 | 내용 |
|------|------|
| 목적 | 브랜드 인지, 신뢰 구축 |
| 주요 포맷 | 30초 CF (치과의사 등장 스토리 포맷) |
| 타깃 | 35~55세 가족 구매 의사결정자 |
| 빈도 | 연간 2~3회 캠페인 플라이트 (공식 기반 추정) |
| KPI | GRP, 브랜드 인지도 리프트, 구매 의향 |
| 핵심 메시지 | "치과의사 No.1 추천 — 가족 모두를 위해" |
| 주의사항 | 공포 소구 최소화, 전문성+따뜻함 밸런스 유지 |

**Steal Sheet — TV 채널**
| 상황 | 실행 포인트 |
|------|------------|
| 신제품 론칭 캠페인 | 치과의사 등장 + 임상 수치 화면 자막 |
| Hill's 브랜드 캠페인 | 수의사 + 보호자 감동 스토리 30초 포맷 |
| Palmolive 시즌 캠페인 | 가족 식사 장면 + 청결·향기 감성 컷 |

### 채널 2 — 디지털 (유튜브·인스타그램·틱톡·메타)

| 플랫폼 | 포맷 | 타깃 | 목적 | KPI |
|--------|------|------|------|-----|
| 유튜브 | 6s 범퍼 + 15s 프리롤 + 롱폼 | 25~45세 | 제품 교육·전환 | 조회 완료율, CTR |
| 인스타그램 피드 | 정방형 이미지·카드뉴스 | 20~40세 여성 | 브랜드 호감도 | 저장 수, 팔로우 |
| 인스타그램 릴스 | 15~30s 숏폼 | 18~35세 | 바이럴·챌린지 | 재생 수, 공유 |
| 틱톡 | 15~60s 네이티브 광고 | 18~28세 Z세대 | 인지·트렌드 편승 | 영상 완성률 |
| 메타 DA | 다이나믹 카탈로그 광고 | 리타겟팅 | 재구매 전환 | ROAS, CPA |
| 네이버 (한국) | 브랜드 검색광고 + 쇼핑 | 전연령 | 구매 전환 | 클릭률, 판매 |

**콘텐츠 빈도 가이드 (추정)**

| 플랫폼 | 권장 포스팅 빈도 | 베스트 타임 |
|--------|----------------|------------|
| 인스타그램 | 주 3~5회 | 오전 7~9시, 저녁 7~9시 |
| 틱톡 | 주 3~7회 | 저녁 6~10시 |
| 유튜브 | 월 2~4회 (롱폼) | 목~토 오전 |
| 페이스북 | 주 2~3회 | 낮 12~2시 |

**Steal Sheet — 디지털 채널**
| 상황 | 실행 포인트 |
|------|------------|
| Colgate Optic White 틱톡 캠페인 | "미백 챌린지" UGC 유도 + 인플루언서 시딩 |
| Hill's 인스타그램 콘텐츠 | 수의사 Q&A 릴스 + 반려동물 Before/After |
| 쿠팡 메타 리타겟팅 | 장바구니 이탈 → 번들 할인 쿠폰 DA |

### 채널 3 — 이커머스 (쿠팡·네이버·아마존·공식몰)

| 항목 | 내용 |
|------|------|
| 핵심 플랫폼 | 쿠팡 로켓배송, 네이버 스마트스토어, 아마존 (글로벌) |
| 전략 | 정기배송 구독 Lock-in, 번들 상품 구성 |
| 상세페이지 원칙 | 임상 데이터 인포그래픽 + 성분 비교표 + 치과/수의사 추천 배지 |
| 리뷰 관리 | 별점 4.5+ 유지, 부정 리뷰 48시간 내 대응 |
| 프로모션 | 첫 구독 30% 할인, 3개 이상 묶음 할인 |
| KPI | ROAS, 구독 전환율, 리뷰 평점 |

**Steal Sheet — 이커머스 채널**
| 상황 | 실행 포인트 |
|------|------------|
| 신제품 쿠팡 등록 | A+ 콘텐츠: 성분 인포그래픽 + 임상 수치 배너 필수 |
| Hill's 정기배송 전환 | "첫 달 무료 + 수의사 추천 리뷰 팩" 구성 |
| 시즌 프로모 (어린이날·추석) | 가족 패키지 번들 + 증정 기프트 구성 |

### 채널 4 — 전문가 채널 (치과·동물병원)

| 항목 | Colgate (치과) | Hill's (동물병원) |
|------|---------------|-----------------|
| 채널 목적 | 전문가 추천 → 소비자 신뢰 전이 | 처방 처방 → 구매 전환 |
| 주요 자료 | 임상 샘플, 브로셔, 치위생사 교육 키트 | 처방 가이드, 수의사 웨비나, 병원 진열대 |
| 관계 관리 | 치과학회 스폰서, 학술 세미나 | 수의사 학회, CME 지원, 처방 인센티브 |
| 디지털 연계 | 치과 앱 연동 쿠폰 | 병원 처방 QR → Hill's 정기배송 연결 |
| KPI | 치과 추천률, 원내 제품 사용률 | 병원 처방 건수, 처방 후 재구매율 |

**Steal Sheet — 전문가 채널**
| 상황 | 실행 포인트 |
|------|------------|
| 신규 치과 파트너 온보딩 | 샘플 키트 + 환자 배포용 쿠폰북 + 디지털 추천 링크 |
| 동물병원 Hill's 처방 확대 | 의사결정자(수의사) 웨비나 초청 + 처방 건수별 리베이트 구조 |
| 시즌 구강 건강 캠페인 | 치과 대기실 POP + QR 코드 → Colgate 구독 랜딩페이지 |

### 채널 5 — 오프라인 유통 (대형마트·약국·올리브영)

| 유통 채널 | 주요 브랜드 | 전략 |
|---------|-----------|------|
| 이마트·롯데마트·홈플러스 | Colgate, Palmolive, Softsoap | 브랜드 블록 진열, 엔드캡 확보 |
| 약국 (올리브영 포함) | Colgate Sensitive, Softsoap | 처방 연계 + 피부과 추천 포지셔닝 |
| 편의점 | Colgate 여행용 소용량 | 여행객·즉시 구매 수요 |
| 펫 전문점 (펫프렌즈) | Hill's | 수의사 추천 POP + 샘플 증정 |
| 동물병원 매점 | Hill's Prescription Diet | 처방 즉시 구매 유도 |

**진열 원칙 (추정)**
```
Colgate 브랜드 블록:
  - 시선 높이(120~150cm) 우선 확보
  - Colgate Total → Optic White → Sensitive → Kids 순 배치
  - 가격 앵커링: 프리미엄 라인 좌측 배치

Hill's 펫몰 진열:
  - 수의사 추천 배지 POP 스탠드 설치
  - Science Diet (일반) 앞단 + Prescription Diet 후단 배치
```

**Steal Sheet — 오프라인 채널**
| 상황 | 실행 포인트 |
|------|------------|
| 이마트 엔드캡 캠페인 | "치과의사 No.1" 대형 배너 + QR 쿠폰 + 번들 번들팩 |
| 올리브영 신규 입점 | Softsoap 체험 샘플 + 피부과 테스트 완료 POP |
| 펫프렌즈 Hill's 행사 | "수의사 추천 시식 이벤트" + SNS 공유 쿠폰 |
