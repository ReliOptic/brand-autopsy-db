## 이미지 & 사진 방향성

### 사람 이미지 원칙
- 실제 환자·HCP 모델 사용 (스톡 이미지 최소화) (추정)
- 다양한 인종·연령·체형 포함 — 글로벌 포용성 반영 (공식 방향)
- 자연광·야외 또는 밝은 실내 배경 선호
- 시선: 카메라 직시(신뢰감) 또는 활동 중 자연스러운 모습

### 금지 이미지
- 과장된 Before/After 체중 사진
- 약물 투여 클로즈업 (공포 유발 방지)
- 병원 침대·IV 라인 등 질병 고통 이미지 (희망 프레임 훼손)

### 데이터 시각화 원칙
- 그래프 배경: `#FFFFFF` 또는 `#F4F4F4`
- 주요 데이터 포인트: 컬러 `#003865` 또는 `#E8501A` 강조
- 오차 막대(error bar) 반드시 포함 — 과학적 신뢰성
- 출처 표기: 하단 우측, 폰트 8pt, `#6E6E6E`

## Steal Sheet — 디자인 레이어별 훔칠 수 있는 전략

### Steal Sheet 1: 컬러로 신뢰 구축

| 요소 | NVO 방식 | 적용 인사이트 |
|------|---------|--------------|
| 딥 네이비 프라이머리 | `#003865` — 권위·안정감 | 의료·금융 등 신뢰 중심 브랜드의 컬러 심리학 활용 |
| 제품별 컬러 분리 | 오렌지/퍼플/그린으로 포트폴리오 구분 | 복잡한 제품군을 색으로 즉시 구분 — 인지 부하 감소 |
| 데이터 컬러 일관성 | 4색 시리즈 고정 (`#003865`~`#7BB3D6`) | 반복 노출로 "이 색 = 신뢰할 수 있는 데이터" 연상 훈련 |

### Steal Sheet 2: 의료 브랜드 타이포 전략

| 요소 | NVO 방식 | 적용 인사이트 |
|------|---------|--------------|
| 헤드라인 산세리프 + 본문 세리프 | 모던함 + 학술 신뢰감 동시 확보 | 두 폰트 계열 조합으로 이중 감성 소구 |
| 데이터 수치 Bold 강조 | "15%" 숫자만 대형 Bold 처리 | 핵심 클레임을 1초 안에 전달하는 시각 계층 |
| 면책 문구 스타일 일관성 | 회색·소형 고정 스타일 | 규제 준수이면서 시각적 방해 최소화 |

### Steal Sheet 3: 제품 패키징 컬러 심리학

| 요소 | NVO 방식 | 적용 인사이트 |
|------|---------|--------------|
| Ozempic 오렌지(`#E8501A`) | 에너지·행동 촉구, 처방 긴박감 | 오렌지는 의료 맥락에서 희귀하게 사용 → 즉각 주목 |
| Wegovy 퍼플(`#6A3EA1`) | 프리미엄·변화·비만치료 고급화 | 퍼플로 "새로운 카테고리" 위상 부여 |
| 인슐린 블루 계열 | 안전·안정·친숙함 | 오래된 처방약일수록 청색 계열로 신뢰 강화 |
