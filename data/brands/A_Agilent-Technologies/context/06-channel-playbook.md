## 채널별 상세 플레이북

### 채널 1 — 웹사이트 (agilent.com)

**목표**: 기술 정보 제공 → 리드 전환 → 영업 연결

**핵심 페이지 구조 (공식)**
- 홈 → 제품 카테고리 → 세그먼트별 솔루션 → 애플리케이션 노트 → 문의/견적

**KPI (추정)**
- 애플리케이션 노트 다운로드 수 (월별)
- 제품 페이지 → 견적 요청 전환율
- 유기 검색 트래픽 성장률

**콘텐츠 업데이트 주기**
- 신제품 페이지: 출시 당일 (공식)
- 애플리케이션 노트: 월 10~20건 (추정)
- 블로그: 주 2~3회 (추정)

**최적화 포인트 (추정)**
- 제품 필터 → 세그먼트·규격·가격대 멀티 필터
- 모바일 기기 접속 최적화 (현장 엔지니어 대상)
- 채팅봇 → 기술 FAQ 자동 응답
