# RTX Corporation — 디자인 시스템

## 디자인 철학

RTX의 비주얼 아이덴티티는 "정밀(Precision) + 신뢰(Trust) + 진보(Progress)"를 시각적으로 구현한다. 과도한 장식을 배제하고 기술적 명확성을 우선하며, 방산·항공 양 시장에서 동일한 전문성 인상을 유지하는 것이 핵심 원칙이다.

## 로고 시스템

### 마스터 로고 (공식)
- RTX 워드마크: 대문자 "RTX" + 풀 네임 "Corporation" 하단 또는 우측 배치
- 로고 심볼: 기어·익형(翼形) 모티프를 추상화한 기하학적 마크 (추정)
- 최소 크기: 디지털 48px / 인쇄 12mm (추정)
- 클리어스페이스: 로고 높이 × 0.5 사방 확보

### 로고 사용 금지 사항
- 로고 색상 임의 변경 금지
- 로고 회전·왜곡·기울기 적용 금지
- 저해상도 래스터 이미지 사용 금지 (SVG/EPS 권장)
- 복잡한 사진 배경 위 직접 배치 금지 (반드시 컬러 박스 또는 화이트 배경 위 배치)

### 사업부 로고 관계
- Collins Aerospace, Pratt & Whitney, Raytheon 각각 독립 로고 보유
- 모든 사업부 마케팅물 하단에 "An RTX Business" 표기 필수 (공식)

### 아이콘 스타일 (추정)
- 선형(Line) 아이콘 우선 — 2px 스트로크, 직각 코너 50%, 라운드 코너 50%

- 인포그래픽: 클린 플랫 디자인, RTX 팔레트 내 3색 이내
- 3D 렌더링: 사실적 제품 렌더 선호, 과도한 CG 효과 지양

## Steal Sheet — 디자인 레이어별 3가지 전술

### 레이어 1: 브랜드 일관성
1. **컬러 토큰 시스템 도입**: `--rtx-navy`, `--rtx-red` 등 CSS 변수로 다국어 사이트 전체 일관성 유지
2. **"An RTX Business" 배지 자동화**: 사업부 마케팅 시스템에 RTX 마스터 브랜드 배지 자동 삽입 규칙 적용
3. **로고 파일 중앙 배포 시스템**: SVG/EPS 최신 버전을 단일 DAM(디지털 에셋 관리) 시스템에서 전 사업부 배포

### 레이어 2: 사용자 경험
1. **기술 다이어그램 템플릿 라이브러리**: 블루프린트 스타일 다이어그램 재사용 템플릿 제공으로 제작 속도 3배 향상 (추정)
2. **다크모드 대응 팔레트**: 방산 프레젠테이션용 다크 테마 공식화 — 네이비 배경 + 라이트 블루 텍스트
3. **모바일 퍼스트 에어쇼 랜딩페이지**: 전시회 현장 QR 코드 연결용 초경량 모바일 페이지 템플릿

### 레이어 3: 인식·차별화
1. **블루프린트 비주얼 시그니처**: 기술 다이어그램·청사진 스타일을 RTX 고유 비주얼 언어로 확립
2. **데이터 시각화 브랜딩**: 차트·그래프에 RTX 팔레트 자동 적용 Tableau/Power BI 테마 배포
3. **영상 인트로/아웃트로 스팅어**: 모든 RTX 브랜드 영상에 3초 로고 스팅어 의무화로 브랜드 기억도 향상
