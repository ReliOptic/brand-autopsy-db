# 05. 디자인 시스템 — Berkshire Hathaway (BRK.B)

> 섹터: 금융 — 다각화 복합기업 | 티커: BRK.B

---

## 1. 디자인 철학

버크셔 해서웨이의 시각적 아이덴티티는 **의도적 절제**다. 화려한 브랜딩을 피하고 내용 자체로 말한다는 원칙이 디자인에도 그대로 반영된다. 공식 웹사이트는 1990년대 스타일을 2020년대에도 유지하며, 이것이 오히려 "우리는 마케팅에 돈을 낭비하지 않는다"는 메시지가 됐다.

### 2-3. 컬러 사용 원칙

- 네이비(`#1A2744`) + 크림(`#F5F2EA`) = 기본 조합, 99%의 문서에 적용
- 골드(`#C9A84C`)는 **핵심 수치 강조**에만 제한적 사용
- 버건디(`#7B2333`)는 **리스크·경고** 맥락에만 사용
- 그린(`#2D5016`)은 **성장·긍정** 맥락에만 사용
- 총 사용 컬러 수 3개 이하 원칙 (절제의 시각화)
