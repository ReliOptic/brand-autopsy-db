# 03 경쟁 환경 분석 — Merck / Keytruda (MRK)

> 섹터: 헬스케어 — 제약 | 티커: MRK | 업데이트: 2026-04-12

---

## 1. 경쟁 구도 개요

Keytruda(pembrolizumab)는 PD-1/PD-L1 면역관문억제제 시장의 글로벌 리더.
경쟁은 세 레이어에서 동시 발생:

```
레이어 1: 직접 경쟁 (PD-1/PD-L1 계열)
  → BMS Opdivo, AZ+Pfizer Imfinzi/Bavencio, Roche Tecentriq

레이어 2: 간접 경쟁 (ADC, CAR-T, 이중항체 등 차세대 IO)
  → AZ Enhertu, BMS Breyanzi, Roche Polivy, J&J Talvey

레이어 3: 바이오시밀러 위협 (2028년 이후 특허 만료)
  → Samsung Bioepis, Celltrion, Amgen 등 바이오시밀러 파이프라인
```

## 2. 주요 경쟁사 배틀카드

---

### 배틀카드 1: BMS Opdivo (nivolumab) vs. Keytruda

| 항목 | Keytruda (MRK) | Opdivo (BMS) |
|------|----------------|--------------|
| 성분명 | pembrolizumab | nivolumab |
| 기전 | PD-1 차단 | PD-1 차단 |
| FDA 적응증 수 | 40+ (2025) | 27+ (2025) |
| 투여 방식 | 3주 또는 6주 간격 IV | 2주, 4주, 6주 IV |
| 대표 적응증 강점 | NSCLC 1차, 삼중음성유방암, 자궁경부암 | HCC, 위암, 흑색종 |
| 바이오마커 전략 | PD-L1 + TMB + MSI-H | PD-L1 (TMB 적응증 제한적) |
| 2024년 매출 (추정) | ~$25B | ~$9B |
| 병용 파트너 전략 | 화학요법, Lenvima, 방사선 | Yervoy (ipilimumab) CTLA-4 병용 강점 |

**Keytruda 우위 포인트 (추정)**
- 적응증 수에서 압도적 리드 (40 vs 27)
- 6주 간격 투여로 편의성 향상
- KEYNOTE 브랜드 인지도 임상의 사이 최고 수준

**Opdivo 우위 포인트 (추정)**
- CTLA-4 병용 (Opdivo+Yervoy) 생존 데이터 일부 적응증에서 강점
- HCC(간세포암), RCC(신장암) 세부 데이터 경쟁력
- BMS의 공격적 급여 협상력

**대응 메시지 (추정)**
> "가장 많은 적응증, 가장 많은 환자 데이터 — KEYNOTE 시리즈가 증명합니다."

---

### 배틀카드 2: AstraZeneca Imfinzi (durvalumab) vs. Keytruda

| 항목 | Keytruda (MRK) | Imfinzi (AZ) |
|------|----------------|--------------|
| 성분명 | pembrolizumab | durvalumab |
| 기전 | PD-1 차단 | PD-L1 차단 |
| 핵심 적응증 강점 | NSCLC 전단계부터 전이성까지 전 주기 | 절제불가 3기 NSCLC(PACIFIC 연구) |
| 병용 전략 | 화학요법 광범위 병용 | Imjudo(tremelimumab) 병용 |
| 투여 편의 | 6주 간격 IV 가능 | 4주 간격 IV |
| 바이오마커 의존도 | PD-L1 고의존 (일부 적응증) | PD-L1 무관 적응증 보유 |
| 2024년 매출 (추정) | ~$25B | ~$4B |

**Keytruda 우위 포인트 (추정)**
- 전체 매출·적응증 폭에서 월등
- NSCLC 1차 단독 치료 근거 (PD-L1≥50%) 확고

**Imfinzi 우위 포인트 (추정)**
- 절제불가 3기 NSCLC에서 PACIFIC 연구 5년 생존 데이터 차별화
- 방사선 동시화학요법 후 유지요법이라는 독특한 포지셔닝

**대응 메시지 (추정)**
> "NSCLC의 모든 단계를 커버하는 유일한 선택 — 1기부터 전이성까지."

### 배틀카드 3: 바이오시밀러 위협 (2028년 이후)

| 항목 | Keytruda 오리지널 | 예상 바이오시밀러 |
|------|-------------------|-------------------|
| 특허 만료 (미국) | 2028~2030년 (추정) | 출시 예상 2029년+ |
| 가격 프리미엄 | 연간 약 $15~18만 (미국) (추정) | 오리지널 대비 30~50% 할인 예상 (추정) |
| 개발사 | — | Samsung Bioepis, Celltrion, Amgen, Fresenius Kabi (추정) |
| Merck 대응 전략 | 차세대 병용요법 포트폴리오 구축 | subcutaneous 제형 전환으로 특허 연장 시도 |
| 브랜드 방어 | KEYNOTE 데이터 축적 + 의사 신뢰도 | — |

**Keytruda 방어 포인트 (추정)**
- 30년 이상 KEYNOTE 데이터 → 의사 처방 습관 형성
- 피하주사(SC) 제형 개발로 특허 재설정 시도
- 환자지원프로그램(PAP)으로 가격 감수성 완화

## 4. Steal Sheet — 경쟁 전략 탈취 포인트

> 경쟁사 전략에서 Keytruda가 학습·역용할 수 있는 포인트 3개

| # | 경쟁사 전략 | Steal 포인트 | Keytruda 적용 방안 |
|---|------------|-------------|-------------------|
| 1 | BMS Opdivo+Yervoy 병용 캠페인 — "두 가지 면역 무기" 스토리텔링 | 병용요법을 단일 브랜드 내러티브로 묶는 방식 | Keytruda+Lenvima 병용을 "이중 공략 (Dual Attack)" 스토리로 통합 브랜딩 (추정) |
| 2 | AZ의 PACIFIC 연구 — 단일 임상 중심 브랜드 구축 | 한 편의 연구가 카테고리를 재정의할 수 있음 | 특정 적응증에서 KEYNOTE 연구 하나를 집중 소구하여 "그 연구 = Keytruda" 인식 강화 (추정) |
| 3 | Roche Tecentriq의 바이오마커 무관 전략 일부 성공 | 모든 환자에게 기회를 열어두는 메시지 | MSI-H/TMB-H 적응증 확대 과정에서 "바이오마커가 없어도 희망을" 보조 메시지 구성 (추정) |
