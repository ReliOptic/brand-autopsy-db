# 06 채널 플레이북 — Stryker (SYK)

## 채널 전략 개요

**채널 철학 (추정)**
> Stryker는 B2B 의료기기 기업으로, 소비자 대중 채널보다 전문의·병원 의사결정자에 직접 도달하는 전문가 채널을 우선한다. 디지털 전환은 영업 조직 효율화의 수단이지, 대체재가 아니다.

## 채널별 상세 플레이북

### 채널 1 — 직접 영업 (Field Sales)

**역할:** 관계 기반 최종 의사결정 지원
**담당 팀:** 지역별 영업 담당자 + 임상 전문가(Clinical Specialist)

**활동 유형 (추정)**
- 1:1 제품 데모 및 수술실 현장 지원
- 병원 구매위원회 프레젠테이션
- 경쟁사 전환 고객 대상 마이그레이션 지원

**핵심 도구**
- MAKO 이동식 데모 유닛
- 디지털 ROI 계산기 태블릿 앱
- 임상 케이스 스터디 프레젠테이션 덱

**KPI (추정)**
- 신규 계정 수주율
- 기존 고객 업셀 비율
- 시연 후 계약 전환율

### 채널 2 — stryker.com (웹사이트)

**역할:** 모든 채널의 정보 허브 및 리드 캡처 포인트
**주요 섹션 구조 (추정)**

```
홈 → 제품/솔루션 → 임상 근거 → 교육/트레이닝 → 뉴스룸 → 고객지원
                      ↓
              제품 카테고리
              ├── 관절 치환
              ├── 수술 기구
              ├── 신경기술
              └── 디지털 헬스
```

**콘텐츠 우선순위**
1. 제품 사양 + 임상 데이터 PDF 다운로드
2. MAKO 데모 신청 폼
3. Stryker University 등록 링크
4. 뉴스룸 & 투자자 관계(IR)

**SEO 핵심 키워드 (추정)**

- "MAKO robotic system"

- "surgical instruments supplier"

## Steal Sheet — 채널 플레이북 (레이어별 3개)

### Layer 1: 채널 통합 전략
1. **오프라인-온라인 루프 설계** — 학회 부스 명함 수집 즉시 CRM 연동 → 48시간 내 개인화 이메일 발송 → LinkedIn 연결 요청 자동화로 오프라인 접점을 디지털 육성으로 전환
2. **영업-마케팅 핸드오프 프로토콜** — 웹사이트 리드 스코어 80+ 달성 시 자동으로 담당 영업에게 알림 + 지금까지의 콘텐츠 소비 이력 전달, 콜드 아웃리치 제거
3. **채널별 CTA 계층화** — LinkedIn(인식) → 웹사이트(고려) → 데모 신청(의도) → 영업 미팅(전환)으로 채널별 단일 CTA 집중, 혼선 방지

### Layer 2: 콘텐츠 배포 전략
1. **학회 앵커 콘텐츠 캘린더** — 연간 5대 학회를 앵커로 설정, 각 학회 전 4주부터 관련 콘텐츠 적층 배포 → 학회 현장 최고조 → 이후 2주 팔로업 콘텐츠 배포 사이클 구축
2. **임상 데이터 멀티 포맷 재활용** — 논문 1편 → LinkedIn 인포그래픽 + 이메일 요약 + 웹 케이스 스터디 + 영업 덱 슬라이드로 동시 5채널 배포, 제작 비용 대비 노출 극대화
3. **KOL 공동 콘텐츠 앰플리파이** — KOL 공동 발표 영상을 Stryker YouTube에 게시 후 해당 KOL의 LinkedIn에도 태그 배포, KOL 개인 네트워크를 Stryker 콘텐츠 유통망으로 활용

### Layer 3: 측정 & 최적화 전략
1. **채널별 기여도(Attribution) 모델** — 마지막 터치 기여가 아닌 선형(Linear) 기여 모델 적용, 학회·교육 등 긴 구매 사이클에서 상위 퍼널 채널의 기여 정당하게 평가
2. **Clinical Specialist 활동 데이터화** — 현장 방문 횟수·시연 건수·팔로업 이메일 개봉률을 CRM에 통합, 영업 효율 최적 패턴 발굴 후 전사 표준화
3. **A/B 테스트 상시 운영** — 이메일 제목줄(임상 데이터 vs. ROI 앵글), LinkedIn 포스트 형식(인포그래픽 vs. 텍스트 카드)을 분기별 순환 테스트해 오디언스 반응 데이터 축적
