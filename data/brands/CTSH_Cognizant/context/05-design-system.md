# 05 디자인 시스템 — Cognizant (CTSH)

## 디자인 철학

Cognizant의 시각 언어는 **"명확성, 신뢰, 기술적 정밀함"** 세 축을 중심으로 구성된다. 엔터프라이즈 B2B 환경에 최적화되어 있으며, 과도한 장식 없이 정보 전달 효율을 극대화하는 클린 모던 스타일을 지향한다.

## 레이아웃 원칙 (추정)

- **그리드:** 12컬럼 반응형 (Bootstrap 기반 — CSS 변수 `var(--bs-body-font-family)` 확인됨)
- **여백:** 섹션 간 80~120px 수직 여백으로 호흡감 확보
- **이미지 스타일:** 다양한 인종·문화의 전문가 이미지 적극 활용 (DE&I 반영)
- **다크 섹션:** `#001468` 또는 `#00002B` 배경으로 고급감 연출
- **Swiper/캐루셀:** swiper-icons 폰트 감지 → 모바일 최적화 슬라이더 사용

## Steal Sheet — 디자인 시스템 3종

1. **Gellix 폰트 일관성 점검:** 모든 마케팅 자산(PPT·PDF·웹·소셜)에서 Gellix 우선 사용 — 브랜드 폰트 미적용 시 신뢰도·일관성 즉시 하락. 외부 대행사 브리핑 시 폰트 파일 및 사용 지침 필수 제공.

2. **컬러 접근성(A11y) 검증:** `#FFD200`(노란색) + 화이트 조합은 WCAG 대비비 부족 가능 — 다크 텍스트(`#001468`)와 페어링 권장. 모든 신규 디지털 자산 출시 전 명암비 4.5:1 이상 검증 의무화.

3. **아이콘 라이브러리 활용:** Cognizant-Icons 커스텀 폰트를 모든 UI·프레젠테이션에 일관 적용 — 외부 아이콘 라이브러리(FontAwesome 등) 혼용 금지. 브랜드 시각 언어 통일성 확보의 핵심 요소.
