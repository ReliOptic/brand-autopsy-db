# 02 오디언스 맵 — Align Technology (ALGN)
> 섹터: 헬스케어 / 치과 | 티커: ALGN

---

## 1. 오디언스 구조 개요

Align Technology는 **이중 시장 구조(Dual Market Structure)**를 운영한다.
- **B2C**: 교정 치료를 원하는 최종 환자 (소비자)
- **B2B**: Invisalign을 처방·시술하는 치과의사 및 교정 전문의

두 오디언스는 메시지, 채널, 의사결정 동인이 완전히 다르며, 브랜드는 이 둘을 동시에 설득해야 한다.

---

## 2. B2C 페르소나

### 페르소나 1: 자신감을 원하는 직장인 여성 — "소영"

| 항목 | 내용 |
|------|------|
| 나이 | 28~35세 |
| 직업 | 대기업 마케터 / 프리랜서 디자이너 |
| 거주 | 서울 강남·마포·성수 (추정) |
| 소득 | 연 4,000~7,000만 원 (추정) |
| 교정 동기 | 앞니 삐뚤어짐 오래 신경 쓰임, 발표·미팅 때 입 가리는 습관 |
| 핵심 니즈 | 눈에 안 띄는 교정 장치, 치료 기간 예측 가능, 음식 제한 최소화 |
| 정보 탐색 | Instagram 전후 사진, 유튜브 브이로그, 카카오톡 오픈채팅 후기 |
| 결정 장벽 | 비용 부담, 통증 우려, "정말 효과 있나?" 의심 |
| 브랜드 기대 | "나처럼 바쁜 사람도 할 수 있다"는 공감 + 전문가 신뢰 |
| 메시지 훅 | "점심시간에 뺐다가 회의 때 끼면 돼요" |

### 페르소나 2: 자녀 교정을 고민하는 부모 — "준혁 아빠"

| 항목 | 내용 |
|------|------|
| 나이 | 40~50세 |
| 직업 | 중소기업 임원 / 자영업자 |
| 거주 | 수도권 신도시 / 지방 광역시 (추정) |
| 자녀 나이 | 초등 고학년~중학생 (8~14세) |
| 교정 동기 | 자녀 유치·영구치 교환기, 학교 친구들도 시작, 치과 의사 권유 |
| 핵심 니즈 | 아이 순응도 (착용 거부 최소화), 장기 효과, 사후 관리 편의 |
| 정보 탐색 | 맘카페, 네이버 블로그, 치과 의사 직접 상담 |
| 결정 장벽 | 높은 비용, "아이가 잘 끼울지" 걱정, 철 교정 대비 효과 의심 |
| 브랜드 기대 | 임상 안전성 데이터, 아동 전용 제품 라인 존재 확인 |
| 메시지 훅 | "Invisalign First — 아이 성장에 맞춘 단계별 교정" |

---

### 페르소나 3: 재교정을 원하는 중년 남성 — "병철"

| 항목 | 내용 |
|------|------|
| 나이 | 42~55세 |
| 직업 | 의사 / 변호사 / 금융권 임원 |
| 거주 | 서울 강남·서초 / 부산 해운대 (추정) |
| 교정 동기 | 20대에 철 교정 했으나 유지 장치 미착용으로 재발, 비즈니스 미팅 인상 관리 |
| 핵심 니즈 | 완전한 비가시성, 짧은 내원 주기, 고급스러운 치료 경험 |
| 정보 탐색 | 지인 추천, 병원 홈페이지, LinkedIn 광고 |
| 결정 장벽 | "나이 들어 교정이 의미 있나?", 바쁜 일정 |
| 브랜드 기대 | 성인 교정 전문성, VIP 서비스 느낌, 빠른 결과 |
| 메시지 훅 | "철 교정 없이, 아무도 모르게, 당신의 미소를 되찾으세요" |

## 3. B2B 오디언스 — 치과 의사 / 교정 전문의

### 오디언스 세그먼트

| 세그먼트 | 특성 | 핵심 니즈 |
|---------|------|----------|
| 교정 전문의 (Orthodontist) | Invisalign 최다 처방자, 기술 수용 빠름 | 최신 어플라이언스 옵션, 임상 케이스 지원 |
| 일반 치과의사 (GP Dentist) | 경증~중등도 케이스 처방, 수익 다각화 목적 | 사용 쉬운 소프트웨어, 환자 전환 마케팅 툴 |
| 치과 대학원생 / 레지던트 | 미래 처방자, 조기 브랜드 충성도 확보 타깃 | 교육·인증 프로그램, 저비용 입문 패키지 |
| 그룹 치과 / DSO | 대량 구매, 표준화 프로세스 선호 | 볼륨 할인, 통합 디지털 워크플로우 |

## 4. 오디언스 의사결정 여정 (Patient Journey)

```
인지 (Awareness)
    ↓ SNS 광고 / 지인 추천 / 치과 내원
관심 (Interest)
    ↓ 인스타그램 전후 사진 / 유튜브 브이로그 / 병원 홈페이지
고려 (Consideration)
    ↓ ClinCheck 3D 시뮬레이션 상담 / 비용 견적
결정 (Decision)
    ↓ 치과 의사 최종 추천 / 할부 옵션 확인
치료 (Treatment)
    ↓ 2주 교체 주기, 앱 리마인더, 정기 내원
옹호 (Advocacy)
    ↓ 완료 후 SNS 공유, 지인 추천, 리뷰 작성
```

## 5. 오디언스 통증 지점 (Pain Points) 매핑

| 단계 | B2C 통증 지점 | B2B 통증 지점 |
|------|-------------|-------------|
| 인지 | "투명 교정이 효과 있나?" 의심 | 환자 수 정체, 차별화 어려움 |
| 고려 | 고가 비용, 치료 기간 불확실성 | 복잡한 케이스 처방 부담 |
| 치료 | 착용 불편, 음식 제한, 분실 | 소프트웨어 학습 곡선 |
| 완료 후 | 유지 장치 관리 | 환자 재방문율 확보 |

## 6. Steal Sheet — 오디언스 맵 레이어

| # | 훔쳐야 할 전략 | 실행 방법 |
|---|--------------|---------|
| 1 | **치과 의사를 공동 마케터로 활용** — 처방의가 브랜드 앰배서더 역할을 하도록 iTero + ClinCheck 툴로 환자 상담 경험 자체를 브랜딩 | 병원 내 디지털 상담 키오스크, 3D 시뮬레이션 첫 경험이 구매 결정의 80%를 차지하도록 설계 (추정) |
| 2 | **페르소나별 진입 메시지 분리** — "바쁜 직장인", "자녀 교정 부모", "성인 재교정" 각각 완전히 다른 랜딩 페이지·광고 크리에이티브 운영 | 동일 브랜드 우산 아래 3개 서브 캠페인 병렬 운영 |
| 3 | **B2B 교육 프로그램이 곧 마케팅** — Invisalign University, Go Live 프로그램으로 의사를 교육하면 처방 충성도가 자동 생성됨 | 치과 의사 인증 레벨 시스템(Preferred→Elite→Diamond) 으로 경쟁 심리 자극 (공식) |
