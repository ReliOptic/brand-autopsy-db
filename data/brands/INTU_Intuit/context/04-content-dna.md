# 04. 콘텐츠 DNA — 인튜이트 (Intuit / INTU)

## 콘텐츠 철학

> "재무 지식은 소수의 전유물이 아니다. 인튜이트의 콘텐츠는 세금·회계·신용이라는 전문 영역을 '다음 단계만 알면 되는' 행동 가이드로 분해한다."

인튜이트 콘텐츠의 핵심 구조: **공포 → 단계 분해 → 행동 촉구 → 보장 확인**
- 공포: "이걸 모르면 세금을 더 낸다"
- 분해: "3단계로 끝내는 자영업자 세금 신고"
- 행동: "지금 바로 확인하세요" CTA
- 보장: "최대 환급 보장, 걱정 마세요"

## 콘텐츠 톤 & 스타일 규칙

### 해야 하는 것
- **숫자로 시작하라**: "47 deductions", "Average refund: $3,252", "10 hours saved/month"
- **단계로 쪼개라**: 3-step, 5-step 구조. 독자가 어디까지 왔는지 항상 알게 하라
- **보장 언어를 뒤에 배치하라**: 정보 제공 → 행동 유도 → 보장 확인 순서
- **"you"를 주어로 고정하라**: "You can file in minutes", "Your refund is ready"
- **전문 용어는 괄호 병기**: "Schedule C (자영업 소득 신고 양식)"처럼 처음 등장 시 풀어쓰기

### 하지 말아야 하는 것
- 세금 코드 번호만 나열 ("IRC Section 162" 단독 사용 금지)
- 부정적 결과만 강조 ("감사 받을 수 있습니다" → "감사 가능성을 최소화하는 방법")
- 조건 없는 최상급 ("The best tax software ever" — 근거 없는 최상급 금지)
- 수동태 남발 ("It has been determined that..." → "We found...")

### Steal Sheet — 마케터가 훔쳐갈 3가지

**1. "공포 → 단계 → 보장"의 3막 구조를 모든 콘텐츠에 삽입하라**
인튜이트의 모든 콘텐츠는 사용자의 불안을 먼저 인정하고(공포), 실행 가능한 단계로 분해하고(분해), 마지막에 안전망을 확인시킨다(보장)(추정, TurboTax 랜딩페이지 구조 분석). 이 3막은 금융·의료·법률처럼 불안이 높은 도메인에서 전환율을 극대화하는 검증된 구조다. **액션**: 우리 핵심 랜딩페이지의 히어로 섹션을 "공포 인정 → 단계 제시 → 보장 확인" 순서로 재구성하라.

**2. 세금 달력을 콘텐츠 발행 달력과 1:1 매핑하라**
TurboTax의 검색 트래픽은 3~4월에 연간 최고점을 찍는다. 이 시기 역방향으로 6~8주 전부터 콘텐츠·이메일 시퀀스를 가동하면 "검색이 폭발하기 전에 이미 고객의 머릿속에 들어가 있을" 수 있다(추정, Google Trends 패턴 기반). **액션**: 우리 산업의 "의도 폭발 주기"를 찾고, 피크 8주 전에 SEO 콘텐츠를 발행해 크롤링·색인 시간을 확보하라.

**3. 인앱 콘텐츠를 가장 비용 효율적인 업셀 채널로 전환하라**
인튜이트는 TurboTax 내 "이 공제를 놓치셨을 수 있습니다" 인앱 팁을 통해 Deluxe·Premium 업그레이드를 유도한다(추정, 제품 UX 분석). 외부 광고 없이도 이미 사용 중인 고객에게 가치를 더 보여줄 수 있다면, 업셀 전환율이 콜드 리드 전환율의 3~5배가 된다(추정, SaaS 업셀 벤치마크). **액션**: 제품 내 "컨텍스트 기반 팁"을 콘텐츠 팀의 책임으로 편입하고, 각 팁에 업셀 트리거를 연결하라.
