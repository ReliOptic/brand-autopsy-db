## 1. 경쟁 구도 개요

Google은 단일 시장이 아닌 **검색·광고·클라우드·AI·디바이스** 등 복수의 전선에서 동시에 경쟁한다.

| 전선 | 주요 경쟁사 | Google 점유율 (추정) |
|------|-------------|----------------------|
| 검색 엔진 | Microsoft Bing, Perplexity AI, ChatGPT Search | ~89% (글로벌, 2026 추정) |
| 디지털 광고 | Meta, Amazon Ads, TikTok Ads | ~28% (글로벌 디지털 광고) |
| 클라우드 인프라 | AWS, Microsoft Azure | ~12% (IaaS, 3위) |
| AI 파운데이션 모델 | OpenAI, Anthropic, Meta AI | 경합 중 |
| 스마트폰 OS | Apple iOS | Android ~72% (글로벌) |
| 생산성 스위트 | Microsoft 365 | Workspace 40억+ 계정 (공식) |
| 동영상 플랫폼 | TikTok, Netflix, Meta Reels | YouTube ~34% (글로벌 동영상 시청) |

## 2. 배틀카드

### 배틀카드 1: Google vs. Microsoft (Bing + Azure + Copilot)

| 항목 | Google | Microsoft |
|------|--------|-----------|
| 검색 점유율 | ~89% | ~4% (Bing) |
| AI 모델 | Gemini 2.0 | GPT-4o (OpenAI 파트너십) |
| 클라우드 | GCP 3위, 강성장 | Azure 2위, 엔터프라이즈 강세 |
| 생산성 | Google Workspace | Microsoft 365 (Office 포함) |
| AI 통합 | AI Overviews, Gemini in Workspace | Copilot M365, Copilot+ PC |
| 가격 정책 | Workspace Business Starter $6/월 | M365 Business Basic $6/월 |
| 약점 | 엔터프라이즈 전환 비용 높음 | 검색 UX 열세, 광고 생태계 미성숙 |

**Google 승리 조건**: 검색 광고 번들 ROI, Gemini AI 품질 우위, Android 생태계 락인
**Microsoft 승리 조건**: Office 의존 기업 유지, Azure OpenAI 서비스, Teams 통합

---

### 배틀카드 2: Google vs. OpenAI (ChatGPT + Operator)

| 항목 | Google | OpenAI |
|------|--------|--------|
| 주력 모델 | Gemini 2.0 Flash/Pro | GPT-4o, o3 |
| 검색 통합 | AI Overviews (Search 내장) | ChatGPT Search |
| 멀티모달 | 텍스트·이미지·오디오·비디오 | 텍스트·이미지·오디오 |
| 에이전트 | Project Astra, Gemini Live | Operator, Deep Research |
| 배포 규모 | 구글 서비스 전체 (Search·Maps 등) | API + ChatGPT 앱 |
| 월간 활성 사용자 | Gemini 15억+ (추정) | ChatGPT 5억+ (공식, 2025) |
| 수익화 | 광고 + 구독 + API | 구독 + API + 엔터프라이즈 |
| 약점 | 브랜드 인지도 후발, 속도 이슈 | 인프라 비용 구조 취약 |

**Google 승리 조건**: 검색 연동 AI로 일상 진입점 장악, 가격 경쟁력
**OpenAI 승리 조건**: 에이전트 UX 선점, 엔터프라이즈 Operator 채택

---

### 배틀카드 3: Google Cloud vs. AWS

| 항목 | Google Cloud | AWS |
|------|--------------|-----|
| 클라우드 점유율 | ~12% | ~32% |
| AI/ML 서비스 | Vertex AI, TPU | SageMaker, Bedrock |
| 데이터 분석 | BigQuery (시장 선도) | Redshift, Athena |
| 컨테이너/쿠버네티스 | GKE (K8s 원조) | EKS |
| 글로벌 리전 수 | 40+ (공식) | 32+ (공식) |
| 가격 정책 | 자동 지속 사용 할인 | 예약 인스턴스 중심 |
| 강점 | BigQuery·K8s·Gemini API 통합 | 생태계 성숙도·파트너 수 |
| 약점 | 엔터프라이즈 영업력·파트너 생태계 | AI 모델 자체 개발 역량 |

**Google 승리 조건**: AI-native 워크로드 타겟팅, BigQuery + Gemini 번들
**AWS 승리 조건**: 기존 고객 잔류, 멀티클라우드 락인
