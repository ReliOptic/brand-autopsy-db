# AON plc — 콘텐츠 DNA

## 콘텐츠 철학

AON의 콘텐츠는 "사고 리더십(Thought Leadership)"을 핵심 엔진으로 삼는다. 단순 서비스 홍보보다 업계 리스크 트렌드를 먼저 정의하고, AON의 데이터와 분석이 그 기준점이 되도록 설계된다. 모든 콘텐츠는 고객이 AON 없이는 얻을 수 없는 인사이트를 제공한다는 전제 아래 생산된다.

| 필러 | 설명 | 대표 형식 |

| 리스크 인텔리전스 | 글로벌 리스크 데이터 분석, 트렌드 예측 | 연간 리포트, 지수 발표 |
| 사이버 & 기술 리스크 | 사이버 공격 비용, 보험 커버리지 격차 | 백서, 케이스 스터디 |

## 콘텐츠 형식 & 톤 가이드

### 장문 콘텐츠 (Long-Form)

**연간 리스크 리포트 (예: Global Risk Management Survey)**
- 분량: 40-80페이지 (추정)
- 구조: 요약 → 데이터 → 지역별 분석 → 권고안
- 톤: 학술적 엄밀성 + 경영진 언어 병행
- 필수 요소: 통계 수치 선행, 업종별 교차 분석

**백서 (White Paper)**
- 분량: 8-20페이지
- 구조: 문제 정의 → 시장 데이터 → AON 프레임워크 → 실행 방안
- 톤: 권위적, 단언형 문장 선호
- 금지: 과도한 AON 자사 홍보 (신뢰성 훼손)

### 단문 콘텐츠 (Short-Form)

**데이터 브리프**
- 분량: 1-2페이지
- 구조: 핵심 수치 3개 → 함의 → 다음 단계
- 톤: 직접적, 숫자 중심

**LinkedIn 포스트 (추정)**
- 분량: 150-300자 (한국어 기준)
- 구조: 충격적 수치 선행 → 인사이트 1줄 → 링크
- 톤: 경영진 개인 어조 활용 (임원 퍼스널 브랜딩)

## 콘텐츠 톤 & 보이스 상세 규칙

### 문장 구조 원칙

1. **수치 선행 원칙**: 첫 문장 또는 첫 문단에 구체적 수치 배치
   - (예) "글로벌 기업의 78%가 사이버 사고 이후 보험 커버리지 격차를 경험했다."
2. **단언형 동사**: "~할 수 있습니다" 대신 "~합니다", "~해야 합니다" 사용
3. **산업 용어 유지**: 번역투 순화 금지 — "리스크 트랜스퍼", "언더라이팅 사이클" 등 원어 병기
4. **수동태 최소화**: "리스크가 관리됩니다" → "기업이 리스크를 관리합니다"

### 금지 표현 상세

| 금지 표현 | 이유 | 대체 표현 |
|---------|------|---------|
| "저렴한 보험" | 프리미엄 포지셔닝 훼손 | "최적화된 리스크 자본 배분" |
| "완벽한 보호" | 과도한 약속, 법적 리스크 | "포괄적 커버리지 설계" |
| "쉽게 해결" | 전문성 경시 | "체계적 접근 방식" |
| "걱정 마세요" | 감성적·비전문적 | "리스크 노출을 정량화합니다" |
| "업계 최고" | 근거 없는 우위 주장 | "글로벌 Top 3 리스크 자문사" |

### 필수 포함 요소

| 필수 요소 | 적용 규칙 |
|---------|---------|
| 출처 명시 | 모든 통계에 조사 출처·연도 병기 |
| 지역 컨텍스트 | 글로벌 데이터에 아시아-태평양 분리 언급 |
| 행동 촉구 | 모든 콘텐츠 말미 "다음 단계" 또는 전문가 상담 유도 |
| AON 방법론 언급 | 자체 프레임워크·지수 최소 1회 언급 |

## 스토리텔링 프레임워크

AON 콘텐츠의 서사 구조는 일관되게 다음 패턴을 따른다 (추정):

```
1. 위협 정량화
   "X% 기업이 Y 리스크에 노출되어 있습니다"
        ↓
2. 대응 격차 노출
   "그러나 Z%만이 적절한 커버리지를 보유합니다"
        ↓
3. 비용 가시화
   "이 격차는 연간 $N의 잠재적 손실을 의미합니다"
        ↓
4. AON 프레임워크 제시
   "Aon의 [방법론명] 접근 방식은..."
        ↓
5. 행동 촉구
   "지금 리스크 진단을 시작하십시오"
```

## Steal Sheet — 콘텐츠 DNA 레이어

### Layer 1: 형식 도용 전략

| 항목 | AON 전략 | 훔쳐야 할 인사이트 |
|------|---------|-----------------|
| 연간 리포트 발행 | 업계 기준 설문 데이터 독점 발행 | 자체 설문으로 미디어 인용 유도 — PR 비용 절감 |
| 계층적 콘텐츠 | 리포트→백서→브리프 축약 버전 3단계 제공 | 독자 심도에 따른 다중 진입점 설계 |
| 데이터 시각화 | 복잡한 리스크 데이터를 인포그래픽 선변환 | 수치 콘텐츠는 반드시 시각화 버전 병행 |
