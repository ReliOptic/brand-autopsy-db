# 02. 오디언스 맵 — The Coca-Cola Company (KO)

> 섹터: Consumer Staples / Beverages | 티커: KO (NYSE)

---

## 1. 타겟 오디언스 전체 구조

```
Primary Core (핵심)
├── 13-24세 Z세대 / 젊은 밀레니얼 (볼륨 드라이버)
├── 25-44세 부모·가족 단위 (브랜드 충성 전수자)
└── 45-65세 헤리티지 팬 (향수·감성 소비층)

Secondary (보조)
├── 스포츠·축제 오케이전 소비자
└── 글로벌 신흥시장 진입층 (아시아·아프리카·남미)
```

## 2. 페르소나 상세

### 페르소나 1: "소셜 미나" (Social Mina)

| 항목 | 내용 |
|------|------|
| 이름 | 박미나 |
| 나이 | 22세 |
| 직업 | 대학교 4학년, 파트타임 카페 아르바이트 |
| 거주지 | 서울 마포구 |
| 소득 | 월 80만원 (아르바이트) |
| 관계 | 친구 그룹 6-8명, 인스타그램 팔로워 2,300명 |
| 핵심 동기 | 그룹 내 분위기 메이커, 공유·인증 욕구 |
| 코카-콜라 사용 맥락 | 친구들과 치킨 시킬 때, 스터디 카페, 축제·콘서트 |
| 소비 빈도 | 주 3-4회 (추정) |
| 브랜드 인식 | 코카-콜라 = "분위기" 음료, 인스타그램 감성 소품 |
| 고통점 | 비싼 외식비, 건강에 대한 막연한 죄책감 |
| 메시지 반응 | "함께 마시자", 한정판 패키지, 아이돌 협업 |
| 주요 채널 | 인스타그램, 유튜브 쇼츠, 틱톡 |
| 전환 장벽 | 설탕 함량 우려, Pepsi/Monster 등 대안 |

**핵심 인사이트**: 미나에게 코카-콜라는 제품이 아닌 사회적 소품(social prop)이다. 구매 결정의 40%는 또래 집단의 선택에 의해 영향을 받는다 (추정).

### 페르소나 2: "패밀리맨 준호" (Family Man Junho)

| 항목 | 내용 |
|------|------|
| 이름 | 이준호 |
| 나이 | 38세 |
| 직업 | 중견기업 팀장, IT 분야 |
| 거주지 | 경기도 수원 |
| 소득 | 월 500만원 (세후) |
| 가족 | 배우자, 초등학생 자녀 2명 |
| 핵심 동기 | 가족 행복, 아이들과의 특별한 순간 |
| 코카-콜라 사용 맥락 | 주말 피자·치킨 외식, 가족 나들이, 명절 |
| 소비 빈도 | 주 1-2회 가족 단위 구매 (추정) |
| 브랜드 인식 | 어릴 때부터 마셔온 익숙한 맛 = 안전한 선택 |
| 고통점 | 아이들 건강 걱정, 당류 섭취 제한 |
| 메시지 반응 | 가족 행복, 크리스마스/명절 감성, "추억" 코드 |
| 주요 채널 | 유튜브, 네이버, 카카오TV |
| 전환 장벽 | 자녀 건강 이슈로 Zero Sugar 전환 고려 중 |

**핵심 인사이트**: 준호는 코카-콜라를 "허락된 인덜전스(allowed indulgence)"로 소비한다. 완전 금지보다 특별한 날의 보상으로 포지셔닝하면 구매 유지율이 높다 (추정).

### 페르소나 3: "레거시 팬 순자" (Legacy Fan Sunja)

| 항목 | 내용 |
|------|------|
| 이름 | 김순자 |
| 나이 | 58세 |
| 직업 | 자영업 (작은 식당 운영) |
| 거주지 | 부산 남구 |
| 소득 | 월 250만원 (순이익 기준) |
| 가족 | 성인 자녀 2명, 손자 1명 |
| 핵심 동기 | 안정감, 오랜 취향 유지, 손님 접대 |
| 코카-콜라 사용 맥락 | 식당 음료로 제공, 손님 접대, 가족 모임 |
| 소비 빈도 | 업소 단위 대량 구매, 주 2회 발주 (추정) |
| 브랜드 인식 | "역시 코카-콜라지" — 대체 불가 원조 |
| 고통점 | 납품 단가, 경쟁 음료 가격 공세 |
| 메시지 반응 | 헤리티지·클래식 이미지, 신뢰성, 변하지 않는 맛 |
| 주요 채널 | TV 광고, 지역 신문, 유통 담당자 접점 |
| 전환 장벽 | 없음. 브랜드 충성도 최상위. 가격만 변수 |

**핵심 인사이트**: 순자는 수십 년 누적 신뢰의 산물이다. 브랜드 변경 시도는 역풍을 부르며(1985 New Coke 사례 참조), 헤리티지 코드 유지가 최우선이다.

## 4. 고객 여정 맵

```
인지(Awareness)
  └── TV·OTT·유튜브 광고, 매장 진열, 행사 협찬
        ↓
관심(Interest)
  └── 한정판 패키지, 콜라보 캠페인, 인플루언서 콘텐츠
        ↓
구매(Purchase)
  └── 편의점/마트 충동구매, 음식점 페어링, 자판기
        ↓
경험(Experience)
  └── 맛, 탄산감, 사회적 공유 경험
        ↓
옹호(Advocacy)
  └── SNS 인증, 지인 추천, 브랜드 굿즈 수집
```
