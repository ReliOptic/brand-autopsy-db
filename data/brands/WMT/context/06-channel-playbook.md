## 1. 채널 전략 개요

Walmart의 채널 전략은 **"모든 접점에서 절약을 경험하게 한다"**는 단일 원칙 아래 물리 채널과 디지털 채널을 통합 운영한다. 채널 간 메시지는 일관성을 유지하되, 각 채널의 맥락에 맞는 포맷과 빈도로 최적화된다.

## 2. 채널별 상세 플레이북

---

### 채널 1: 오프라인 매장 (Brick & Mortar)

**역할:** 매출의 약 80% 이상 발생, 브랜드 경험의 물리적 구현 (추정)

| 항목 | 내용 |
|------|------|
| 주요 접점 | 입구 엔드캡, 가격 사이니지, 계산대, 고객 서비스 데스크 |
| 핵심 메시지 포맷 | 롤백 배지, 멀티바이 특가, EDLP 표시 |
| 직원 커뮤니케이션 | 파란 조끼 + 웃음 — 친근한 이웃 이미지 강화 (공식) |
| 레이아웃 원칙 | 식료품 외곽 배치, 생필품 내부 — 동선 극대화 |
| 계절 변환 주기 | 6~8주마다 주요 매장 구역 테마 전환 (추정) |
| Scan & Go | Sam's Club 선도, Walmart 매장 확대 중 (공식) |

**Steal Sheet — 오프라인 3가지**
1. 픽업 드라이브스루 전용 동선을 매장 입구 최근접 위치에 배치 — 온라인→오프라인 연결 마찰 제거
2. 엔드캡 상단 3피트는 항상 주간 롤백 최고 절약 상품 — 시선 고정 후 카트 유도
3. 처방약 픽업 카운터 인근에 건강식품·비타민 크로스셀링 구역 배치

### 채널 2: Walmart 앱 & Walmart.com

**역할:** e커머스 매출 성장 엔진, Walmart+ 가입 허브, 픽업 예약 플랫폼 (공식)

| 항목 | 내용 |
|------|------|
| 월간 활성 사용자 | 약 1억 5천만 명 (추정) |
| 핵심 기능 | 쇼핑 리스트, 픽업 예약, Walmart+, 가격 비교, Scan & Go |
| 개인화 | 구매 이력 기반 추천, 위치 기반 재고 안내 (공식) |
| 앱 푸시 빈도 | 주 3~5회 (과도한 푸시 → 알림 차단 위험) (추정) |
| SEO/ASO 핵심 키워드 | "grocery pickup", "walmart deal", "save money" |

**푸시 메시지 공식:**
```
[오늘만] {카테고리} {절약 금액} 절약 → 지금 픽업 예약
예: [오늘만] 신선 채소 $8 절약 → 지금 픽업 예약
```

**Steal Sheet — 앱 3가지**
1. 앱 홈 상단 "이번 주 내가 절약한 금액" 위젯 — 재방문·Walmart+ 업그레이드 유도
2. 픽업 예약 완료 화면에서 "친구에게 공유하면 $5 쿠폰" CTA 삽입
3. 비로그인 사용자에게 위치 기반 재고 확인 허용 → 앱 가치 체험 후 가입 전환

### 채널 3: 이메일 마케팅

**역할:** 기존 고객 재활성화, 주간 딜 안내, Walmart+ 전환 (추정)

| 항목 | 내용 |
|------|------|
| 발송 빈도 | 주 2~3회 (추정) |
| 세그먼트 기준 | 구매 카테고리, 지역, Walmart+ 여부 |
| 평균 오픈율 | 22~28% (추정) |
| 이메일 유형 | 주간 딜 / 개인화 추천 / 시즌 이벤트 / 재구매 유도 |
| 제목 최적 길이 | 35~45자 (모바일 미리보기 완전 표시) |

**이메일 구조 템플릿:**
```
제목: [카테고리]에서 $XX 절약하세요 — 오늘 마감
프리헤더: 이번 주 롤백 상품 [N]개 + 픽업 무료

[히어로 이미지: 절약 금액 대형 표시]
[CTA 버튼: "지금 장보기" — Blue #0071CE]

딜 1 / 딜 2 / 딜 3 (카드형 3열)

[푸터: Walmart+ 업그레이드 유도 배너]
```

**Steal Sheet — 이메일 3가지**
1. 제목에 "$" 금액 직접 기입 — "최대 절약" 대신 "오늘 $12 아끼세요"로 오픈율 +15% (추정)
2. 픽업 기반 고객에게는 "내일 오후 3시 픽업 가능" 시간 맞춤 발송
3. 6개월 미구매 휴면 고객 재활성화: "$5 웰컴백 쿠폰" 단독 발송 후 30일 내 구매 추적

### 채널 4: TV / 영상 광고

**역할:** 브랜드 인지도 유지, 시즌 캠페인 감성 전달, 신규 서비스 인지 (추정)

| 항목 | 내용 |
|------|------|
| 주요 매체 | 지상파 TV, 케이블(ESPN·Hallmark), CTV/OTT |
| 광고 길이 | 15초(브랜드 리마인더) / 30초(상품 프로모) / 60초(감성 시즌) |
| 집행 피크 | 블랙프라이데이 전 6주, 크리스마스 시즌, 신학기 |
| 크리에이티브 방향 | 실제 가족 출연, 절약 순간의 감정적 클라이맥스 |
| 브랜드 식별 요소 | 로고 노출 마지막 3초 + "Save Money. Live Better." 태그라인 |

**Steal Sheet — TV 광고 3가지**
1. 30초 광고 오프닝 5초에 절약 금액 수치 자막 — 채널 서핑 중 시선 고정
2. OTT(스트리밍) 광고에 QR코드 삽입 → 앱 다운로드 또는 딜 페이지 직접 연결
3. 블랙프라이데이 60초 감성 필름을 YouTube에 풀버전 공개 → 바이럴 확산 유도

### 채널 6: Walmart Connect (리테일 미디어 광고)

**역할:** 공급업체 광고 수익화, 검색·디스플레이 광고, 1st-party 데이터 활용 (공식)

| 항목 | 내용 |
|------|------|
| 광고 형태 | 스폰서드 프로덕트, 스폰서드 브랜드, 디스플레이 |
| 타깃 데이터 | Walmart 구매 이력 1st-party 데이터 활용 (공식) |
| 성장률 | 연 30%+ 광고 매출 성장 (추정) |
| 경쟁 포지션 | Amazon Ads 대비 식료품 구매 데이터 우위 주장 (공식) |

**Steal Sheet — Walmart Connect 3가지**
1. 공급업체에게 "검색 → 구매" 전환 데이터 투명 공개 — Amazon Ads 이탈 유인
2. 오프라인 매장 내 디지털 사이니지와 온라인 광고 번들 패키지로 옴니채널 광고 단가 프리미엄 부여
3. 소규모 브랜드 전용 셀프서브 광고 플랫폼 접근성 강화 — 롱테일 광고 매출 확대

---

### 채널 7: 주간 광고지 (Weekly Circular)

**역할:** 고령·저디지털 고객 핵심 접점, 지역 매장 트래픽 유도 (추정)

| 항목 | 내용 |
|------|------|
| 발행 주기 | 주 1회 (일요일 기준) |
| 배포 방식 | 신문 삽지, 우편, 앱 내 디지털 버전 |
| 핵심 구성 | 주간 롤백 상품, 식료품 특가, 계절 이벤트 |
| 디자인 원칙 | 대형 가격 숫자, Yellow·Orange 배지, 제품 사진 중심 |

**Steal Sheet — 광고지 3가지**
1. 디지털 광고지에 "앱으로 리스트에 추가" 원터치 버튼 삽입 — 오프라인→디지털 브리지
2. 광고지 표지 상품을 앱 홈 히어로 배너와 동기화 — 통합 캠페인 일관성 강화
3. 고령 고객 밀집 지역 우편 발송 광고지에는 QR코드 + 전화 주문 번호 병기
