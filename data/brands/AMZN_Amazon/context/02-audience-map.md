## 1. 오디언스 세그먼트 전체 지도

Amazon은 B2C·B2B·B2D(개발자) 세 축으로 오디언스가 구분된다.

```
┌─────────────────────────────────────────────────────┐
│                   AMAZON 오디언스                    │
├─────────────┬─────────────────┬─────────────────────┤
│   B2C       │      B2B        │       B2D           │
│ 소비자      │  기업/판매자    │   개발자/엔지니어   │
├─────────────┼─────────────────┼─────────────────────┤
│ Prime 회원  │ 3P 셀러         │ AWS 사용자          │
│ 스트리밍    │ AWS 기업 고객   │ Alexa 개발자        │
│ Alexa 유저  │ 광고주          │ 오픈소스 커뮤니티   │
│ Kindle 독자 │ 물류 파트너     │                     │
└─────────────┴─────────────────┴─────────────────────┘
```

## 2. 핵심 페르소나

### 페르소나 1: "편의 중심 소비자" — 박지연 (35세)

| 항목 | 내용 |
|------|------|
| 직업 | 맞벌이 직장인, 마케팅 팀장 |
| 거주 | 서울 강남구 (아파트) |
| 소득 | 연 8,000만 원 (가구 합산 1.4억) |
| 사용 서비스 | Amazon Prime, Prime Video, Kindle, Alexa Echo |
| 핵심 동기 | 시간 절약, 번거로움 제거, 프리미엄 편의성 |
| 결정 트리거 | 무료 익일 배송, 고객 리뷰 별점 4.5+, Prime 할인 |

| 공포 | 배송 지연, 가품, 개인정보 침해 |
| 즐겨 쓰는 채널 | Amazon App, Alexa, Prime Video, 이메일 뉴스레터 |
| 광고 반응 | 개인화 추천에 높은 반응 (추정) |

**Quote (추정):** "리뷰 보고 클릭 두 번이면 내일 아침 문 앞에 있어요. 그게 전부예요."
