# 05 디자인 시스템 — Illinois Tool Works (ITW)

## 디자인 철학

ITW의 시각적 아이덴티티는 **"기능이 형태를 이끈다(Form Follows Function)"** 원칙을 따른다. 장식적 요소를 최소화하고 정보 전달 명확성을 극대화한다. 산업 제조업의 신뢰성과 정밀성이 시각 언어로 표현된다. (추정)

### 컬러 사용 원칙

- **Primary Blue `#003087`**: 전사 브랜드 요소에만 사용. 희석 또는 그라디언트 금지.
- **Accent Blue `#009FDA`**: 전체 화면의 20% 이하로 제한.
- **배경 컬러**: 순백(`#FFFFFF`) 또는 라이트 그레이(`#F3F4F6`) 외 배경색 사용 지양.
- **텍스트 가독성**: 최소 대비 비율 WCAG AA 기준(4.5:1) 준수 권장.

## 로고 시스템

### 로고 구조 (공식)
- **ITW 워드마크**: 'Illinois Tool Works'의 이니셜 조합 블록 레터링
- 전용 네이비 `#003087` 배경 또는 화이트 배경 위 네이비 텍스트 버전
- 서브브랜드 로고: 각 브랜드 독립 로고 + ITW 패밀리 심볼 병기 (추정)

### 로고 금지 사항 (추정)
- 로고 색상 임의 변경 금지
- 로고 비율 왜곡 금지
- 로고 최소 크기 이하 사용 금지 (디지털 32px / 인쇄 10mm 이하)
- 로고 배경에 복잡한 사진 직접 배치 금지
