# 06. 채널 플레이북 — Regeneron Pharmaceuticals (REGN)

> 산업: 헬스케어 - 바이오테크놀로지 | 분석 기준일: 2026-04

---

## 1. 채널 전략 개요

Regeneron의 채널 전략은 두 개의 주요 오디언스(HCP, 환자)를 위한 **이중 트랙 구조**로 운영된다.

- **HCP 트랙:** 과학 데이터·임상 근거 중심, 규제 환경 준수
- **환자 트랙 (DTC):** 감성 공감·삶의 변화 중심, 접근성 강조

모든 채널은 미국 FDA PhRMA 가이드라인, KFDA 및 현지 규제 기준을 준수한다. (공식)

## 3. 채널별 상세 플레이북

### 채널 1: 메디컬 리아종 (MSL — Medical Science Liaisons)

| 항목 | 내용 |
|------|------|
| 역할 | 임상 전문가가 HCP에게 과학 데이터 1:1 전달 |
| 주요 콘텐츠 | 임상시험 결과, 케이스 스터디, 파이프라인 업데이트 |
| 빈도 | 고처방의 (HVC) 월 1~2회, 일반 분기 1회 (추정) |
| KPI | 처방 의향 변화, HCP 엔게이지먼트 스코어 |
| 주의 | 처방 유도 불가, 교육 목적만 허용 (공식) |

**Steal Sheet:**
1. Type 2 염증 통합 관리 미팅 어젠다 — 피부과·알레르기과 HCP 대상 공동 교육 미팅
2. 경쟁사 배틀카드 즉시 배포 — 경쟁 제품 라벨 변경 시 72시간 내 반박 자료 전달
3. KOL 공동 연구 제안 — RWE 데이터 공동 수집으로 처방 관계 심화

---

### 채널 2: 학술 컨퍼런스 & 심포지엄

| 항목 | 내용 |
|------|------|
| 주요 행사 | AAD (미국피부과학회), AAAAI (알레르기면역학회), ATS (흉부학회), AAO (안과학회), ASCO (종양학회) |
| 참가 형태 | 메인 스폰서, 위성 심포지엄, 포스터·구연 발표 |
| 핵심 목표 | 임상 데이터 선점 발표, KOL 네트워킹, 경쟁사 대응 |
| 콘텐츠 형식 | Late-breaking 데이터, 헤드투헤드 서브그룹 분석 |

**Steal Sheet:**
1. Late-Breaking Session 선점 — 핵심 임상 결과를 컨퍼런스 최우선 세션에 발표 예약
2. 위성 심포지엄 독점 개최 — 경쟁사 대비 넓은 공간·식사 제공으로 참석률 극대화
3. 소셜 미디어 실시간 중계 — 컨퍼런스 데이터를 Twitter/LinkedIn 실시간 업데이트

---

### 채널 3: HCP 디지털 포털

| 항목 | 내용 |
|------|------|
| URL 예시 | hcp.dupixent.com, hcp.eylea.us (공식) |
| 콘텐츠 | 처방 정보, 투약 가이드, 환자 보험 지원 자료, 임상 데이터 라이브러리 |
| 기능 | 샘플 주문, 처방 보조 도구, 환자 등록 프로그램 연결 |
| 접근 제어 | HCP 면허 인증 필요 (공식) |

**Steal Sheet:**
1. AI 처방 보조 도구 통합 — 환자 프로파일 입력 시 적합한 Dupixent 용량·용법 추천 알고리즘
2. 보험 사전승인 자동화 — Prior Authorization 서류 자동 작성 툴로 HCP 행정 부담 제거
3. CME 크레딧 통합 — 포털 내 교육 콘텐츠 이수 시 CME 학점 자동 부여

### 채널 4: TV / CTV (Direct-to-Consumer)

| 항목 | 내용 |
|------|------|
| 주요 시장 | 미국 (DTC 광고 허용), 뉴질랜드 |
| 포맷 | 30초·60초 TV 광고, CTV (스트리밍 광고), 온라인 동영상 |
| 타깃팅 | 아토피 환자 연령대 (25~54세), 천식·COPD 환자 (40~65세) |
| 핵심 메시지 | 증상 완화 + "의사에게 Dupixent에 대해 물어보세요" |
| 규제 요건 | ISI (Important Safety Information) 음성·화면 고지 의무 (공식) |

**톤 & 매너:**
- 전반부: 증상으로 인한 고통 공감 (어두운 톤)
- 중반부: Dupixent 투약 장면 (희망적 전환)
- 후반부: 일상 복귀 장면 (밝고 따뜻한 톤)
- 엔딩: 브랜드 로고 + "의사에게 상담하세요" CTA

**Steal Sheet:**
1. 환경별 맞춤 CTV 타깃팅 — 알레르기 관련 앱·웹사이트 방문자에게 Dupixent CTV 광고 노출
2. 시즌 캠페인 — 봄·가을 알레르기 시즌 광고비 집중 집행으로 ROI 극대화
3. 환자 증언 시리즈 — 다양한 인종·연령·적응증별 환자 스토리로 광범위한 공감대 형성

| 채널 | 전략 | 콘텐츠 유형 | 빈도 |
|------|------|------------|------|
| YouTube | Dupixent 환자 스토리, 질환 교육 | 2~5분 동영상 | 월 2~4편 (추정) |
| Instagram | 환자 커뮤니티, 인식 제고 | 카드뉴스, 릴스 | 주 3~5회 (추정) |

| LinkedIn | 기업 브랜드, 채용, R&D 업데이트 | 파이프라인 뉴스, 임직원 스토리 | 주 5회 (추정) |
| Twitter/X | 학술 컨퍼런스 실시간, 업계 뉴스 | 트윗, 스레드 | 상시 (추정) |

**Steal Sheet:**
1. 해시태그 커뮤니티 소유 — #아토피이겨내기 #DupixentLife 등 브랜드 해시태그로 환자 UGC 집결
2. 인플루언서 의학 크리에이터 — 피부과·알레르기 전문의 의사 유튜버 파트너십 (HCP-환자 경계 콘텐츠)
3. 리타깃팅 퍼널 설계 — 질환 정보 검색 → 치료 옵션 탐색 → 의사 상담 예약 3단계 디지털 퍼널

### 채널 6: 환자 지원 프로그램

| 프로그램 | 내용 | 대상 |
|----------|------|------|
| Dupixent MyWay® | 코페이 지원, 자가투여 교육, 간호사 지원 | 미국 상업 보험 환자 (공식) |
| 보험 지원 서비스 | Prior Authorization 보조, 급여 내비게이션 | 전체 환자 |
| 환자 스타터 키트 | 첫 투약 교육 자료, 주사 연습 장치 | 신규 처방 환자 |
| 무료 샘플/브릿지 | 보험 승인 대기 중 무료 투약 지원 | 경제적 취약 환자 (추정) |

**Steal Sheet:**
1. 디지털 코치 앱 — 투약 알림, 증상 추적, HCP 소통 통합 앱으로 치료 지속률 향상
2. 커뮤니티 이벤트 — 환자 지원 그룹 오프라인 미팅 스폰서십으로 브랜드 충성도 강화
3. 조기 접근 프로그램 — 보험 승인 전 신속 브릿지 제공으로 첫 투약 경험 극대화
