## 2. 페르소나

### 페르소나 1: 기관 투자자 — "배당왕을 믿는 펀드매니저"

| 항목 | 내용 |
|------|------|
| 이름 | 박준혁 (추정) |
| 나이 | 45세 |
| 직책 | 대형 연기금 포트폴리오 매니저 |
| 지역 | 서울 / 뉴욕 이원 활동 |
| 학력 | MBA, CFA |
| 연봉 | 2억~5억 원 (추정) |
| 핵심 목표 | 안정적 배당 수익 + 자본 보전 + ESG 스코어 충족 |
| 주요 고민 | 에너지전환 리스크, 규제 강화, 좌초자산 우려 |
| XOM과의 접점 | 분기 실적 콜, IR 자료, Bloomberg 터미널 |
| 구매 트리거 | 높은 FCF, 주주환원 증가, 저탄소 사업 성장 증거 |
| 기피 메시지 | 모호한 ESG 목표, 부채 증가, 탐사 실패 뉴스 |
| 선호 어조 | 데이터 중심, 간결, 장기 전망 명확 |

**핵심 인사이트 (추정):** 이 페르소나는 XOM의 배당 연속성(40년+)과 CCS·수소 투자 확대에서 안도감을 얻는다. 그러나 IEA 2050 넷제로 시나리오와의 충돌을 지속적으로 의식한다.

### 페르소나 3: 정책 결정자 — "에너지 안보를 지키려는 산업부 국장"

| 항목 | 내용 |
|------|------|
| 이름 | 이수연 (추정) |
| 나이 | 52세 |
| 직책 | 산업통상자원부 에너지정책국장 |
| 지역 | 서울 세종 |
| 학력 | 행정학 석사, 에너지 경제 연수 |
| 연봉 | 공무원 7~9급 관리직 기준 (추정) |
| 핵심 목표 | 에너지 공급 안정, 탄소중립 정책 이행, 산업 경쟁력 유지 |
| 주요 고민 | 재생에너지 간헐성, 에너지 안보 vs 탈탄소 균형 |
| XOM과의 접점 | 공식 정책 브리핑, 산업 포럼, CCS 협력 논의 |
| 구매 트리거 (협력 트리거) | CCS 기술 실증, 에너지전환 로드맵 협력 제안 |
| 기피 메시지 | 규제 회피 인식, 환경 사고, 로비 의혹 |
| 선호 어조 | 정책 정합성 강조, 장기 파트너십, 공공 이익 프레이밍 |
