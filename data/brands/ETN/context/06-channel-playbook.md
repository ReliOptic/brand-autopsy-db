# 06 채널 플레이북 — Eaton Corporation (ETN)

## 채널 전략 개요

Eaton의 채널 전략은 **"전문가에게 전문가로"** 원칙을 기반으로 한다. 대중 소비자 채널보다 B2B 전문 채널, 파트너 네트워크, 기술 미디어에 집중한다. 디지털 채널은 기술 정보 배포와 리드 너처링에, 오프라인 채널은 신뢰 구축과 대형 계약 클로징에 활용한다. (추정)

## 채널별 상세 플레이북

### 1. 직접 영업 (Direct Sales)

**역할:** 연간 계약 $1M 이상의 전략적 고객 직접 관리
**팀 구조 (추정):** 산업별 Key Account Manager + 기술 세일즈 엔지니어 페어링
**프로세스:**
```
1단계: 잠재 고객 발굴 (LinkedIn, 전시회, 파트너 레퍼럴)
2단계: 기술 니즈 파악 (현장 방문, 기술 심사)
3단계: 솔루션 설계 (기술팀 협업, ROI 산출)
4단계: 제안서 제출 (48시간 내 초안)
5단계: 협상 및 클로징
6단계: 설치 후 관리 (90일 최적화 프로그램)
```
**콘텐츠 지원:** 경쟁 배틀카드, 케이스 스터디, ROI 계산기, 기술 백서
**금지 행동:** 가격 단독 협상 (가치 기반 협상 원칙 준수)

---

### 2. 파트너/대리점 채널

**역할:** 중소형 프로젝트, 지역 밀착 서비스, 한국·아시아 현지화
**파트너 티어 구조 (추정):**

| 티어 | 조건 | 혜택 |
|------|------|------|
| Platinum | 연 매출 $5M+ | 전담 AM, 마케팅 펀드, 독점 지역 |
| Gold | 연 매출 $1M+ | 기술 교육 우선, 공동 마케팅 |
| Silver | 인증 보유 | 기본 기술 지원, 교육 자료 |

**파트너 지원 도구 (추정):**
- 파트너 포털: 기술 문서, 가격표, 마케팅 자료
- 기술 교육 프로그램 (온라인 + 현장)
- 공동 마케팅 펀드 (MDF)
- 리드 공유 및 등록 시스템

### 3. 웹사이트 (eaton.com)

**역할:** 기술 정보의 원천, 글로벌 24/7 영업 사원
**구조 (추정):**
```
홈 → 인더스트리별 솔루션 → 제품 카탈로그 → 기술 자료
                         → 케이스 스터디
                         → 전문가 상담 CTA
```
**SEO 전략 (추정):**
- 기술 키워드 중심: "industrial UPS", "power distribution unit", "busway system"
- 인더스트리 키워드: "data center power management", "EV charging infrastructure"
- 롱테일: "how to calculate UPS capacity for data center"

**전환 최적화 포인트:**
- 제품 페이지마다 "전문가 상담" CTA
- 기술 자료 다운로드 게이팅 (이메일 수집)
- 채팅 또는 콜백 예약 플로팅 버튼

## Steal Sheet — 채널 레이어별 3개

### Steal Sheet: 인지(Awareness) 채널 레이어

1. **LinkedIn 업계 보고서 "전력 관리 현황" 연간 발간**
   Eaton 자체 조사 + 외부 데이터로 산업 트렌드 보고서 발간. LinkedIn Document Ad로 배포, 다운로드 리드 수집. 미디어 픽업으로 SEO 백링크 확보. Schneider의 "Sustainability Report"에 대응하는 전문성 앵커 콘텐츠.

2. **기술 팟캐스트 스폰서십 또는 자체 채널**
   "Power Engineering Today" 류의 산업 팟캐스트 스폰서 또는 Eaton 자체 팟캐스트 운영. 엔지니어·시설관리자의 통근·이동 시간 점유. 경쟁사 대비 음성 채널 선점.

3. **유튜브 "전력 인프라 사고 분석" 시리즈**
   실제 발생한 전력 사고를 기술적으로 분석하는 교육 영상. "다크 에듀테인먼트" 포맷으로 높은 조회수 + 브랜드 전문성 동시 달성. 재생에너지·데이터센터 확장 시대의 높은 검색 수요 포착.

### Steal Sheet: 고려(Consideration) 채널 레이어

1. **파트너 포털 "셀프서비스 기술 견적" 툴**
   파트너가 직접 기술 사양 입력 → 즉시 가이드 견적 생성. 영업 지원 속도 향상, 소형 프로젝트 자동화. 경쟁사 대비 파트너 만족도 차별화.

2. **"산업별 전력 설계 웨비나" 월 1회 시리즈**
   데이터센터, 반도체, 배터리공장, 재생에너지 등 핫 세그먼트 순환 웨비나. 등록자는 고품질 MQL. 강연자: Eaton 기술 임원 + 고객 레퍼런스 공동 발표.

3. **LinkedIn 기술 뉴스레터 "Eaton Power Digest"**
   월 1회 LinkedIn 뉴스레터 발행. 전력 업계 주요 뉴스 + Eaton 관점 코멘터리. 구독자 기반 nurture, 영업 대화 진입점 생성.

### Steal Sheet: 전환(Conversion) 채널 레이어

1. **"현장 기술 감사" 무료 제공 캠페인**
   대형 잠재 고객 대상 무료 현장 전력 인프라 감사 서비스 제공. 실제 문제 발견 → 솔루션 제안 자연스러운 연결. 영업 초기 진입 장벽 제거, 경쟁사보다 먼저 고객 현장 장악.

2. **"프로젝트 클로징 킷" 영업 지원 패키지**
   고객 내부 승인용 자료 패키지 (ROI 분석, 기술 스펙, 인증서, 레퍼런스 리스트). 고객이 내부에서 Eaton을 대신 "팔 수 있게" 지원. 구매 결정 사이클 단축.

3. **리타게팅 광고 — LinkedIn + Google Display**
   웹사이트 특정 제품 페이지 방문자 대상 리타게팅. 케이스 스터디 + "전문가 상담 예약" CTA 광고. 고려 단계에서 이탈한 잠재 고객 재획득.

## 채널 성과 측정 프레임워크

### 퍼널 단계별 KPI

| 단계 | 채널 | 주요 KPI |
|------|------|---------|
| 인지 (Awareness) | LinkedIn, YouTube, PR | 도달 범위, 브랜드 검색량, 미디어 언급 |
| 관심 (Interest) | 웹사이트, 이메일, 이벤트 | 세션 시간, 자료 다운로드, 웨비나 등록 |
| 고려 (Consideration) | 이메일 너처링, LinkedIn | MQL 수, 콘텐츠 소비 깊이 |
| 의도 (Intent) | 직접 영업, 파트너 | SQL 수, RFQ 접수 |
| 전환 (Conversion) | 직접 영업 | 계약 체결 수, 계약 규모, Win Rate |
| 유지 (Retention) | 이메일, 고객 포털, 영업 | 갱신율, 추가 구매율, NPS |
| 추천 (Advocacy) | 케이스 스터디, 이벤트 | 레퍼런스 고객 수, 인바운드 레퍼럴 |

### 채널 투자 수익률 기준 (추정)

- MQL당 비용 목표: $200~500 (B2B 산업재 기준)
- SQL 전환율 목표: MQL의 15~25%
- 영업 사이클: 평균 6~18개월
- 고객 생애 가치(LTV): 초기 계약의 3~5배 (유지보수·업그레이드 포함)
