# 04. 콘텐츠 DNA — Texas Instruments (TXN)

## 콘텐츠 철학

TI의 콘텐츠는 "판매"가 아니라 "교육"을 목적으로 한다. 엔지니어가 문제를 해결하는 과정에서 TI 콘텐츠를 발견하고, 그 과정에서 자연스럽게 TI 제품을 채택하도록 설계된다. 이는 인바운드 마케팅의 교과서적 실행이자, B2B 기술 브랜드의 가장 강력한 콘텐츠 전략이다. (추정)

핵심 원칙: **"Teach first. Sell never."** — 직접적 판매 언어 없이, 기술적 권위로 구매를 유도한다.

---

## 콘텐츠 유형 분류

### Tier 1 — 핵심 기술 자산 (Core Technical Assets)

**데이터시트 (Datasheet)**
- TI 브랜드의 가장 중요한 콘텐츠 단위. (공식)
- 평균 수십~수백 페이지, 실측 그래프·회로도·열 특성 데이터 포함.
- 작성 원칙: 과장 없이 측정 가능한 수치만. 최상급 표현은 반드시 측정 조건을 병기.
- SEO 효과: 엔지니어가 "op amp low noise datasheet" 검색 시 TI 데이터시트가 상위 노출.

**앱노트 / 애플리케이션 노트 (Application Note)**
- 특정 설계 문제(예: "LLC 컨버터 설계 방법")를 해결하는 심층 기술 문서.
- 실제 회로 시뮬레이션, BOM 예시, 레이아웃 가이드라인 포함.
- 가장 높은 기술 신뢰도를 제공하는 콘텐츠 형식.

**레퍼런스 디자인 (Reference Design)**
- 실제 동작하는 완전한 회로 설계를 무료로 공개. (공식)
- 엔지니어가 "그대로 복사해서 쓸 수 있는" 검증된 설계.
- BOM, 거버 파일, 테스트 결과까지 포함.

### Tier 2 — 교육·온보딩 콘텐츠

**TI E2E Community 포럼**
- 엔지니어가 직접 TI 전문가에게 기술 질문을 올리고 답변받는 공개 포럼. (공식)
- 검색 엔진 최적화 효과: 수백만 건의 Q&A가 롱테일 검색 트래픽을 생성.
- 커뮤니티 톤: 기술 문서보다 완화된 동료 간 대화체 허용.

**트레이닝 / 웹세미나 (Training & Webinar)**
- TI Training 포털에서 무료 온라인 강좌 수백 개 제공. (공식)
- 주제: 전력 설계, 모터 드라이브, 센서, 무선통신, 펑셔널 세이프티 등.
- 수료 후 인증서 발급 → 엔지니어 경력 개발 동기 부여.

**LaunchPad / 개발 키트**
- 하드웨어 콘텐츠. $10~$50 수준의 평가보드로 TI 생태계 무료 체험. (공식)
- Code Composer Studio(무료 IDE)와 연동. 소프트웨어 생태계 진입 장벽 제거.

### Tier 3 — 시장·트렌드 콘텐츠

**백서 / 설계 가이드**
- "자동차 전장의 전력 효율 설계 트렌드" 같은 시장 레벨 교육 자료.
- 구매 의사결정자·CTO급 오디언스 대상.
- 기술 깊이보다 비즈니스 임팩트(비용 절감, 설계 기간 단축) 강조.

**YouTube TI Channel**
- 제품 소개 영상보다 "HOW-TO" 기술 시연 중심. (추정)
- 예: "How to design a boost converter with TPS61023" 형식의 실습 영상.

## 콘텐츠 보이스 가이드라인

### 문장 구조 원칙
- **수치 우선**: 형용사 앞에 측정값을 놓는다.
  - X: "매우 낮은 정지전류"
  - O: "정지전류 0.9µA — 업계 최저 수준"
- **조건 명기**: 최상급 표현에는 항상 측정 조건을 병기한다.
  - X: "최고의 효율"
  - O: "VIN=12V, IOUT=2A 조건에서 95.2% 효율"
- **수동태 최소화**: 엔지니어링 문서도 능동태로 간결하게.
  - X: "고려될 수 있는 파라미터"
  - O: "다음 파라미터를 확인하라"

### 금지 표현 (콘텐츠 레벨)
- 검증 불가 형용사 단독 사용: "탁월한", "혁신적인", "세계적인"
- 감정 호소형 CTA: "지금 바로 느껴보세요"
- 경쟁사 직접 비방: 수치 비교는 허용, 브랜드 공격은 금지

### 필수 요소 (콘텐츠 레벨)
- 관련 레퍼런스 디자인 링크
- 관련 E2E 포럼 스레드 링크
- 평가보드/LaunchPad 구매 링크
- 데이터시트 다운로드 링크

## 콘텐츠 성과 지표 (KPI, 추정)

| 지표 | 의미 |
|------|------|
| 데이터시트 다운로드 수 | 설계 채택 가능성의 선행 지표 |
| E2E 포럼 월간 활성 스레드 | 기술 권위 및 커뮤니티 건강도 |
| 레퍼런스 디자인 다운로드 | 실제 설계 채택 의향 |
| LaunchPad 판매량 | 차세대 설계 엔지니어 파이프라인 |
| 기술 웨비나 등록/완료율 | 교육 깊이 및 오디언스 품질 |

### Steal Sheet — 3가지

1. **롱테일 SEO 전략으로서의 기술 문서.** TI의 수백만 건 데이터시트·앱노트·포럼 Q&A는 "op amp single supply rail-to-rail" 같은 초구체적 검색어에서 상위 노출된다. 콘텐츠 마케팅을 블로그 포스팅이 아니라 문제 해결 문서로 설계하면 B2B 기술 세그먼트에서 무한한 유기적 트래픽이 발생한다.

2. **콘텐츠 깊이가 신뢰를 만든다.** 깊이 없는 콘텐츠를 대량 생산하는 경쟁사 대비, TI는 한 개의 앱노트에 수십 페이지 분량의 실측 데이터를 담는다. 엔지니어는 콘텐츠 두께로 브랜드 신뢰도를 판단한다. "한 페이지 요약"보다 "50페이지 앱노트"가 더 강력한 마케팅 자산이다.

3. **하드웨어 콘텐츠(평가보드)는 디지털 콘텐츠보다 강력하다.** LaunchPad 개발 키트를 엔지니어의 손에 쥐어주는 순간, TI 생태계의 코드·툴·데이터시트·포럼이 한꺼번에 소비된다. 물리적 제품을 "콘텐츠"로 재정의하면 체험 기반 브랜드 락인이 가능하다.
