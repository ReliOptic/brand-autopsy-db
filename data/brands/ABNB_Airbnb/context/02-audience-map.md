# 02 오디언스 맵 — Airbnb (ABNB)

## 타깃 오디언스 개요

Airbnb는 여행자(게스트)와 공간 제공자(호스트) 양측 시장을 동시에 공략하는 플랫폼이다.
핵심 타깃은 경험 중심 밀레니얼·Z세대 여행자와 부업 수입을 원하는 도시 거주 주택 소유자이다.

---

### 페르소나 1: 경험 수집가 밀레니얼

**이름:** 김지수 (32세, 서울 마포구 거주, 콘텐츠 마케터)

| 항목 | 내용 |
|------|------|
| 소득 | 연 5,500만 원 (추정) |
| 여행 빈도 | 연 3~5회 해외여행, 국내 월 1회 |
| 채널 | Instagram, Pinterest, YouTube 여행 브이로그 |
| 페인포인트 | 체인 호텔의 획일성, 진짜 현지 경험 부재 |
| 동기 | SNS 공유 가능한 독특한 공간, 현지인처럼 살아보기 |
| 결정 기준 | 사진 퀄리티, 리뷰 신뢰도, 위치 독창성 |
| Airbnb 사용 | 연간 4~6회 예약, 평균 3박 이상 |
| 브랜드 기대 | "이곳에서만 할 수 있는 경험"의 발견 |

**핵심 메시지:** "호텔에선 찍을 수 없는 사진, Airbnb에서 찾으세요."

### 페르소나 2: 워케이션 디지털 노마드

**이름:** Alex Chen (28세, 베를린 거주, 프리랜서 UX 디자이너)

| 항목 | 내용 |
|------|------|
| 소득 | 월 4,000~6,000 USD (추정) |
| 여행 패턴 | 한 도시 2~4주 장기 체류, 월평균 이동 1~2회 |
| 채널 | LinkedIn, Nomad List, Reddit r/digitalnomad |
| 페인포인트 | 안정적 Wi-Fi 확보, 주방 사용 필요, 호텔의 높은 장기 비용 |
| 동기 | 생활비 절감 + 현지 문화 몰입 + 업무 효율 |
| 결정 기준 | Wi-Fi 속도 명시, 주방·세탁기 구비, 월간 할인 여부 |
| Airbnb 사용 | 장기 숙박(28박+) 필터 적극 활용 |
| 브랜드 기대 | 신뢰할 수 있는 편의시설 정보와 유연한 취소 정책 |

**핵심 메시지:** "일하면서 살아보는 여행, 장기 할인으로 더 스마트하게."

| 동기 | 유휴 자산 수익화, 유연한 운영 스케줄 |
| 결정 기준 | 수수료 구조 투명성, 호스트 보호 정책, 스마트 가격 기능 |

**핵심 메시지:** "내 집이 수익을 낸다. 호스팅은 Airbnb가 도와드립니다."

## 오디언스 인사이트 요약

- 게스트의 **71%**는 Airbnb 예약 전 SNS에서 숙소 영감을 얻는다. (추정)
- 장기 숙박(28박+) 세그먼트는 전체 예약의 **20%+**를 차지하며 팬데믹 이후 급성장. (공식 IR 기반 추정)
- 호스트 이탈의 주요 원인은 운영 복잡성과 게스트 신뢰 불안. (추정)
- Z세대 여행자는 "체험 예약(Experiences)"에 밀레니얼 대비 1.4배 높은 참여율. (추정)
