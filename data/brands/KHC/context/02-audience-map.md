## 3. 오디언스 여정 맵 (수진 기준)

```
인지 → 고려 → 구매 → 사용 → 충성
  |        |        |       |       |
TV/SNS   비교앱   매장    가정    재구매
캠페인   가격확인  진열대   식탁    구독
향수     영양표시  쿠폰     레시피  추천
```

| 단계 | 핵심 채널 | 메시지 포커스 | 마찰 요인 |
|------|-----------|--------------|-----------|
| 인지 | TV, Instagram | 감성·향수 스토리 | 광고 과부하 |
| 고려 | 쇼핑앱, 리뷰 | 가격·성분 비교 | 경쟁사 프로모 |
| 구매 | 매장 진열대, 온라인 | 진열 위치, 쿠폰 | 대체품 유혹 |
| 사용 | 제품 패키지, Pinterest | 레시피 활용 팁 | 기대 불일치 |
| 충성 | 이메일, 앱 | 멤버십 혜택, 신제품 | 가격 인상 |
