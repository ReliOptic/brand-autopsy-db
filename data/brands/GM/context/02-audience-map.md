# 02 오디언스 맵 — General Motors (GM)

## 타깃 오디언스 개요

GM은 단일 브랜드가 아닌 멀티브랜드 포트폴리오이므로, 오디언스는 브랜드별로 분산되어 있다.
아래는 GM 모 브랜드 및 핵심 서브브랜드를 관통하는 통합 오디언스 지형도이다.

---

## ### 페르소나 1 — "실용적 가장" 마이클

| 항목 | 내용 |
|------|------|
| 이름 | 마이클 (Michael) |
| 나이 | 42세 |
| 직업 | 건설 현장 감독 / 중소 도급업체 운영 |
| 거주지 | 미국 중서부 (오하이오, 미시건, 인디애나) |
| 소득 | 연 $75,000–$110,000 (추정) |
| 주요 브랜드 접점 | Chevrolet Silverado, GMC Sierra |
| 구매 동기 | 적재 용량, 견인력, 내구성, 잔존가치, 딜러 네트워크 접근성 |
| 정보 탐색 채널 | YouTube 리뷰, 딜러 방문, 지인 추천, 트럭 전문 포럼 |
| 핵심 가치관 | 가족 부양, 열심히 일한 결과로 좋은 트럭 소유, 미국산 제품 선호 |
| 브랜드 기대 | "내 일을 방해하지 않는 신뢰할 수 있는 도구" |
| 불안 요소 | EV 트럭의 실제 견인 항속거리, 충전 인프라 부재 지역 거주 |
| GM 관련도 | 핵심 볼륨 고객군. Silverado EV 전환의 최대 설득 난제 페르소나 |

**페르소나 인사이트:** 마이클은 GM의 현재 매출을 지탱하는 핵심이지만, EV 전환의 가장 큰 저항 세력이기도 하다. "내 트럭이 충전소 없는 현장에서 멈추면 안 된다"는 실용적 공포가 구매 결정을 지배한다.

## ### 페르소나 2 — "미래 지향 얼리어답터" 지나

| 항목 | 내용 |
|------|------|
| 이름 | 지나 (Gina) |
| 나이 | 34세 |
| 직업 | 테크 스타트업 마케팅 디렉터 |
| 거주지 | 미국 서부 (캘리포니아, 워싱턴주) |
| 소득 | 연 $130,000–$180,000 (추정) |
| 주요 브랜드 접점 | Cadillac LYRIQ, Chevrolet Blazer EV |
| 구매 동기 | 지속가능성, 기술 혁신, 디자인 미학, 사회적 정체성 표현 |
| 정보 탐색 채널 | Instagram, EV 전문 미디어(Electrek, InsideEVs), TED, 팟캐스트 |
| 핵심 가치관 | 환경 책임, 기술 얼리어답터 정체성, 프리미엄 경험 |
| 브랜드 기대 | "내 라이프스타일과 가치관을 반영하는 차" |
| 불안 요소 | GM이 Tesla만큼 소프트웨어·OTA 업데이트가 빠른가에 대한 회의 |
| GM 관련도 | EV 전환의 미래 성장 고객군. 현재는 Tesla 이탈 방지가 핵심 과제 |

**페르소나 인사이트:** 지나는 GM을 "따라가는 브랜드"로 인식할 위험이 크다. GM이 이 페르소나를 확보하려면 소프트웨어 경험, OTA 업데이트 주기, 브랜드 스토리의 진정성이 Tesla 대비 명확한 차별점을 만들어야 한다.

## ### 페르소나 3 — "브랜드 충성 베테랑" 로버트

| 항목 | 내용 |
|------|------|
| 이름 | 로버트 (Robert) |
| 나이 | 58세 |
| 직업 | 지역 은행 지점장 / 은퇴 준비 단계 |
| 거주지 | 미국 남부 (텍사스, 조지아, 테네시) |
| 소득 | 연 $95,000–$140,000 (추정) |
| 주요 브랜드 접점 | Buick Enclave, GMC Yukon, Cadillac CT5 |
| 구매 동기 | 편안함, 안전성, 브랜드 유산, 가족 공간, 딜러 서비스 품질 |
| 정보 탐색 채널 | 자동차 잡지, TV 광고, 딜러 방문, 지인 네트워크 |
| 핵심 가치관 | 가족 중심, 안정성, 오랜 브랜드 관계 유지, 미국적 가치 |
| 브랜드 기대 | "내가 믿어온 브랜드가 계속 나를 실망시키지 않기를" |
| 불안 요소 | GM의 리콜 반복, EV 전환으로 인한 친숙한 모델 단종 우려 |
| GM 관련도 | 고수익 유지 고객. 충성도는 높으나 EV 전환 시 이탈 위험 존재 |

**페르소나 인사이트:** 로버트는 GM의 과거 자산이자 현재 수익의 핵심이다. 하지만 EV 전환 과정에서 익숙한 모델이 사라지거나 딜러 경험이 변하면 Toyota/Lexus로 이탈할 가능성이 높다. 브랜드 연속성 메시지가 필수적이다.
