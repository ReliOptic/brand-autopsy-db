# 06. 채널 플레이북 — Pfizer (PFE)

> 섹터: Health Care — Pharmaceuticals | 티커: PFE

---

## 1. 채널 전략 개요

Pfizer는 규제 환경(FDA, EMA, PMDA 등)에 따른 채널 제약이 크다.
미국 외 대부분 국가에서 DTC(소비자 직접 광고)가 금지되므로,
HCP 채널과 디지털 질환 인식 채널을 이원화해 운용한다. (공식)

## 2. 채널별 상세 플레이북

### 채널 1: Pfizer.com — 공식 웹사이트

| 항목 | 내용 |
|------|------|
| 1차 목적 | 브랜드 허브, 뉴스룸, 파이프라인 정보, IR |
| 주요 섹션 | 제품·파이프라인, 뉴스룸, 과학·혁신, 기업 책임, 투자자 관계 |
| 콘텐츠 업데이트 주기 | 뉴스룸: 수시 / 파이프라인: 분기 / ESG: 연간 |
| KPI | 방문자 수, 체류 시간, 파이프라인 페이지 CTR, IR 다운로드 수 |
| 톤 | 공식적·전문적, MLR 심의 완료 콘텐츠만 게재 (공식) |
| 성공 지표 예시 | 투자자 IR 자료 다운로드 전월 대비 +20% (추정) |

**Steal Sheet 포인트:** 뉴스룸을 단순 보도자료 저장소가 아닌 콘텐츠 퍼블리싱 허브로 운용 — 검색 트래픽과 미디어 픽업 동시 확보.

## 5. 규제 컴플라이언스 체크리스트 (채널 공통)

- [ ] MLR(Medical-Legal-Regulatory) 심의 완료 여부 확인

## Steal Sheet — 채널 플레이북 레이어

| # | 전략 요소 | 실행 힌트 |
|---|-----------|----------|
| 1 | **컨퍼런스 전후 콘텐츠 앰플리파이 사이클** | 학술 발표 전 예고 → 현장 라이브 → 발표 후 요약 영상 3단계로 하나의 데이터를 3배 활용 |
| 2 | **임직원 어드보커시로 유기 도달 확장** | LinkedIn 팔로워보다 임직원 개인 네트워크 합산이 더 큰 경우 많음 — 과학자·임원 계정 활성화가 광고비 절감의 핵심 |
| 3 | **규제 컴플라이언스 체크리스트를 채널 런칭 게이트로** | 모든 채널 콘텐츠 배포 전 체크리스트 통과를 프로세스에 내재화 → 리콜·규제 제재 리스크 선제 차단 |
