# 06. 채널 플레이북 — Amphenol (APH)

## 채널 전략 개요

Amphenol의 채널 전략은 B2B 기술 기업의 전형을 따른다. 직접 영업(Direct Sales)과 유통 파트너(Distribution)의 이중 구조를 운영하며, 디지털 채널은 주로 기술 정보 제공과 설계 채택 촉진에 집중한다. (추정)

---

## 채널 맵

```
[채널 생태계]

디지털 채널
├── 자사 웹사이트 (amphenol.com 및 사업부 도메인)
├── 이메일 마케팅 (뉴스레터, 제품 업데이트)
├── LinkedIn (기업 페이지, 임원 콘텐츠)
├── YouTube (제품 영상, 웹세미나 아카이브)
└── 기술 문서 포털 (데이터시트, CAD 파일)

유통 파트너 채널
├── Arrow Electronics
├── Avnet
├── DigiKey
├── Mouser Electronics
└── 지역 전문 유통사 (국내: 삼화전자, LS전선 계열 등 — 추정)

직접 영업 채널
├── 글로벌 계정 관리자 (GAM)
├── 지역 필드 영업
└── 내부 기술 영업 (Inside Sales)

파트너/협력 채널
├── OEM 설계 파트너십
├── EMS(전자 제조 서비스) 파트너
└── 기술 표준화 기관 (IEC, IEEE, JEDEC 참여)

이벤트 채널
├── DesignCon, CES, electronica
├── 고객사 기술 세미나
└── 대학·연구기관 협력 프로그램
```

## 페르소나별 채널 우선순위

### 페르소나 1 (설계 엔지니어 박재현) 채널 여정

```
인식: LinkedIn 기술 인사이트 포스트 → 자사 웹 기술 문서
고려: 데이터시트 다운로드 → Arrow/DigiKey 재고 확인
평가: 샘플 요청 → 기술 세미나 참가 → 내부 기술영업 미팅
구매: 유통사 PO 또는 직접 계약
```

### 페르소나 2 (조달 관리자 이수미) 채널 여정

```
인식: GAM 정기 방문 → QBR 미팅
고려: Arrow/Avnet 가격 비교 → 공급망 BCP 문서 요청
평가: 계약 협상 → ERP/EDI 통합 검토
구매: 장기 공급 계약 (직접 영업)
```

### 페르소나 3 (AI 스타트업 CTO 마이클 탠) 채널 여정


## Steal Sheet — 채널 플레이북 레이어

**Steal 1: 설계 채택 단계 "디지털 셀프서비스" 강화**
Samtec은 웹에서 엔지니어가 영업 개입 없이 사양 확인 → 샘플 요청 → CAD 다운로드 → 구매까지 완결할 수 있는 플로우를 구축했다. APH는 유통사 의존도가 높아 자사 웹에서의 전환율이 낮다 (추정). 자사 웹 내 완결형 샘플링 플로우를 구축하면 설계 채택 속도와 데이터 수집 모두 개선된다.

**Steal 2: LinkedIn 엔지니어 바이라인 콘텐츠 시리즈**
Keysight, NI(National Instruments)는 현직 엔지니어의 이름을 전면에 내세운 "기술 인사이트 시리즈"를 LinkedIn에서 운영하며 팔로워 인게이지먼트를 업종 평균 3배 이상 달성한다 (추정). APH의 LinkedIn은 기업 보도자료 중심이다 (추정). 내부 엔지니어 사고 리더십 콘텐츠로 전환해야 한다.

**Steal 3: 유통사 공동 웹세미나 시리즈**
Arrow, Avnet은 주요 반도체·커넥터 공급사와 공동으로 설계 엔지니어 대상 기술 웹세미나를 월 1~2회 운영하며 등록자 리드를 공급사에 공유한다. APH도 유통 파트너와의 공동 웹세미나 시리즈를 체계화하면 신규 리드와 기술 브랜딩을 동시에 달성할 수 있다.
