# 06. 채널 플레이북 — Altria Group (MO)

> 소비재 · 담배 | 대표 브랜드: Marlboro, IQOS, on!

---

## 1. 채널 전략 개요

담배 광고는 미국에서 TV·라디오·야외 광고가 MSA(담배 마스터 합의) 및 FDA 규정에 의해 광범위하게 제한된다. 이에 따라 Altria의 채널 전략은 **규제 허용 범위 내 최대 효율**을 목표로, 소매 채널과 성인 인증 디지털 플랫폼을 중심으로 구성된다. (공식/추정 혼합)

## 3. 채널별 전술 플레이북

### 3-A. 편의점·소매점 POS (Point of Sale)

**역할**: 성인 소비자의 최종 구매 결정 지점. 가장 높은 ROI 채널. (추정)

| 전술 요소 | 내용 |
|-----------|------|
| 진열 위치 | 계산대 뒤 상단 고정 위치 — 성인 직원 관리 구역 (공식) |
| 가격 태그 | 멀티팩 할인, 번들 가격 표시 (추정) |
| POS 자재 | 브랜드 가격 표지, 제품 디스플레이 스탠드 (공식) |
| 쿠폰 | 계산대 쿠폰, 성인 인증 앱 연동 쿠폰 (추정) |
| on! 전시대 | 별도 파우치 디스플레이 유닛, 플레이버 코딩 명시 (추정) |
| 핵심 지표 | 선반 점유율(Share of Shelf), 재고 회전율 |

---

### 3-B. 성인 인증 디지털 플랫폼

**역할**: 제품 교육, 전환 유도, 고객 데이터 수집. (공식)

| 플랫폼 | URL (추정) | 주요 기능 |
|--------|------------|-----------|
| IQOS 공식 사이트 | iqos.com | 성인 인증 → 제품 정보 → 구매 → CRM 등록 |
| on! 공식 사이트 | on.com | 플레이버 탐색, 구독 구매, 로열티 프로그램 |
| Altria 기업 사이트 | altria.com | IR, ESG, 기업 정보, 규제 커뮤니케이션 |

**디지털 퍼널 설계**:
```
방문 → 성인 인증 (생년월일) → 제품 탐색 → 무료 트라이얼 신청
→ CRM 등록 → 이메일 넛지 → 정기 구독 전환
```

---

### 3-C. 이메일 CRM

**역할**: 등록된 성인 고객 대상 재구매 촉진·신제품 안내. (공식)

| 이메일 유형 | 발송 빈도 (추정) | 목적 |
|-------------|----------------|------|
| 웰컴 시리즈 | 등록 후 D+0, D+3, D+7 | 온보딩·제품 이해 |
| 프로모션 | 월 2–3회 | 할인·번들 구매 전환 |
| 신제품 출시 | 출시 시 | 카테고리 확장 인지 |
| 재활성화 | 90일 비구매 시 | 이탈 방지 |
| 규제/뉴스 업데이트 | 필요 시 | 브랜드 신뢰도 유지 |

**이메일 규칙**:
- 수신자 전원 성인 인증 완료 고객만 해당 (공식)
- 수신 거부(Unsubscribe) 명확 표기 필수 (공식)
- 건강 이점 직접 주장 금지 (공식)

## 6. 채널 통합 원칙

1. **성인 인증 일관성**: 모든 소비자 접점에서 성인 확인 절차 유지 (공식)
2. **메시지 정합성**: 채널별 톤이 달라도 핵심 메시지(스모크프리 전환, 책임 있는 선택) 일관 유지 (추정)
3. **데이터 통합**: CRM·소매 데이터 통합으로 개인화 수준 향상, 단 개인정보 보호 규정 준수 (추정)
4. **규제 모니터링**: 채널별 규제 변화를 실시간 모니터링하여 즉각 대응 체계 유지 (공식)
