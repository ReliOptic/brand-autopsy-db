# 06 채널 플레이북 — Allstate (ALL)

## 채널 전략 개요

Allstate는 전통적인 에이전트 채널의 신뢰 자산을 유지하면서, 디지털 다이렉트 채널을 병행 확장하는 **하이브리드 옴니채널 전략**을 구사한다. 고객 여정 단계별로 채널 역할을 분리하여 인지 → 비교 → 구매 → 유지 → 클레임의 각 단계를 최적화한다. (추정)

## 채널별 상세 전략

### 1. 에이전트 네트워크 (핵심 채널)

**현황:** 전국 약 10,000명 이상 전속 에이전트 + 독립 에이전트 (공식)

**역할 분담**
- 전속 에이전트: 복잡한 리스크 상담, 번들 판매, 클레임 지원
- 독립 에이전트: 가격 민감 고객 대상 비교 판매 보완

**활성화 전략**
- 에이전트 디지털 툴킷 지원: CRM, 견적 자동화, 고객 포털 접근
- 지역 마케팅 공동 펀드(MDF) 지원으로 지역 광고 집행
- Drivewise 데이터 기반 고위험 고객 재상담 트리거 자동화

**측정 지표**
- 에이전트당 신규 계약 건수
- 번들 전환율 (자동차 → 자동차+주택)
- 에이전트 유지율 및 생산성

---

### 2. 유료 검색 (Google/Bing Ads)

**전략 우선순위**
- 브랜드 키워드 방어: "Allstate insurance", "Allstate quote" 독점 입찰
- 경쟁 키워드 공략: "Progressive vs Allstate", "car insurance near me"
- 지역 타게팅: 에이전트 서비스 가능 지역 반경 입찰 가중치 조정

**광고 포맷**
- 반응형 검색 광고 (RSA): 3개 이상 헤드라인 조합 A/B 테스트
- 스마트 디스플레이: 리타게팅 견적 미완료 사용자
- 콜 전용 광고: 시니어 세그먼트 타게팅

**랜딩 페이지 원칙**
- 견적 시작까지 2단계 이내
- 신뢰 배지 (AM Best 등급, 수상 내역) 상단 배치
- 채팅봇 + 전화 CTA 동시 노출

### 3. TV·OTT 광고

**Mayhem 캠페인 (공식 장기 캠페인)**
- 캐릭터 "Mayhem" (딘 윈터스 분): 일상적 위험을 코믹하게 의인화
- 보험 미가입·부실 가입의 결과를 유머로 전달
- 강력한 브랜드 회상율 기반 인지도 유지

**OTT 전략 (추정)**
- Hulu, Peacock, Amazon Prime 광고 지면 활용
- 15초 범퍼 광고 + 30초 풀 스팟 믹스
- 데이터 기반 타게팅: 신규 주택 구매자, 신차 구매자 세그먼트

**효과 측정**
- 브랜드 리프트 조사 (인지·선호·구매 의향)
- TV 광고 노출 후 검색량 변화 상관관계 분석

### 6. 자연 검색 (SEO)

**콘텐츠 SEO 우선순위**
- 허브 & 스포크 구조: 보험 카테고리 허브 페이지 → 세부 가이드 스포크
- E-E-A-T 강화: 실제 에이전트·전문가 바이라인, 인용 출처 명시
- 로컬 SEO: 에이전트별 Google Business Profile 최적화

**기술 SEO (추정)**
- Core Web Vitals 최적화 (LCP < 2.5초, FID < 100ms, CLS < 0.1)
- 구조화 데이터 마크업: FAQ, HowTo, Review 스키마
- 모바일 우선 인덱싱 완전 대응
