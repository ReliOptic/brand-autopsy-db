# 01. 브랜드 정체성 — Intuitive Surgical (ISRG)

> **카테고리**: 헬스케어 / 로봇 수술 시스템  
> **플래그십 제품**: da Vinci 수술 시스템  
> **본사**: 서니베일, 캘리포니아, 미국  
> **설립**: 1995년 (공식)  
> **티커**: NASDAQ: ISRG (공식)

---

## 1. 브랜드 에센스

| 항목 | 내용 |
|------|------|
| **브랜드 미션** | 최소 침습 수술을 통해 환자의 치유를 돕고, 외과의의 역량을 확장한다 (공식) |
| **브랜드 비전** | 수술 결과가 환자의 거주지·접근성과 무관하게 동일하게 탁월한 세상 (추정) |
| **핵심 가치** | 임상적 우수성, 기술 혁신, 환자 안전, 파트너십 (공식) |
| **포지셔닝** | "수술 로봇의 표준(Standard of Care)" — 임상 근거 기반 독점적 플랫폼 (공식) |
| **카테고리 역할** | 카테고리 창조자 & 방어자 |

|------|-----------|------|
| **권위 / Authority** | 9/10 | 임상 데이터·논문·학회 발표 중심의 근거 기반 어조 |
| **공감 / Empathy** | 7/10 | 환자 스토리·치유 여정을 선택적으로 활용 |
| **혁신 / Innovation** | 8/10 | 기술 진보를 선도적으로 표현하되 과장 지양 |
| **신뢰 / Trust** | 10/10 | 규제·임상·안전 언어로 최고 수준 신뢰 구축 |

| **간결성 / Conciseness** | 7/10 | 의료 규제 특성상 긴 면책조항이 수반되나 핵심 메시지는 압축적 |
| **글로벌리티 / Globality** | 8/10 | 현지 규제·언어를 존중하면서도 글로벌 단일 플랫폼 메시지 유지 |

## 5. 금지어 / 필수어

### 금지어 (절대 사용 금지)

1. **"로봇이 수술한다"** — 외과의의 역할을 대체하는 인상을 주므로 금지 (공식 가이드라인)
2. **"무결점"** — 의료기기 법적 책임상 절대적 완벽성 주장 불가
3. **"치료 보장"** — FDA·KFDA 규정상 임상 결과 보장 표현 불가
4. **"혁명(Revolution)"** — 근거 없는 과대 표현으로 규제 리스크 (추정)
5. **"경쟁사보다 우월"** — 직접 비교 광고 규제 및 브랜드 품격 손상

### 필수어 (반드시 사용)

1. **"외과의 주도(Surgeon-led)"** — 인간 중심성 강조, 규제 리스크 방어
2. **"최소 침습(Minimally invasive)"** — 카테고리 정의어
3. **"임상적 근거(Clinical evidence)"** — 신뢰 기반의 핵심 언어
4. **"정밀도(Precision)"** — da Vinci 기술 차별점 핵심
5. **"환자 결과(Patient outcomes)"** — 모든 커뮤니케이션의 최종 목적어

## 6. 브랜드 퍼스낼리티 프리즘 (Kapferer)

| 차원 | 내용 |
|------|------|
| **외적 특징 (Physique)** | 정밀 로봇 암, 수술실 조명, 3D 시각화 콘솔 |
| **성격 (Personality)** | 침착하고 정밀한 전문가, 학자형 혁신가 |
| **문화 (Culture)** | 실리콘밸리 엔지니어링 + 의료 윤리의 교차점 |
| **자기상 (Self-image)** | "나는 환자의 생명을 가장 안전하게 다루는 도구를 만든다" |
| **반영 (Reflection)** | 세계 최고 수준의 외과의가 선택하는 시스템 |
| **관계 (Relationship)** | 신뢰할 수 있는 장기 기술 파트너 |
