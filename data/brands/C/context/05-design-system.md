# 05. 디자인 시스템 — Citigroup (C)

## 브랜드 디자인 철학

Citi의 디자인 언어는 **"명확성(Clarity), 신뢰(Trust), 연결(Connection)"** 세 원칙을 시각적으로 번역한다. 복잡한 금융 정보를 단순화하여 전달하는 것이 최우선이며, 장식보다 기능성을 우선시한다. (추정)

## UI 컴포넌트 언어

### 버튼 스타일
- **Primary CTA:** 배경 `#0066A0`, 텍스트 `#FFFFFF`, 둥글기 4px (추정)
- **Secondary CTA:** 테두리 `#003B70`, 배경 투명, 텍스트 `#003B70` (추정)
- **Hover 상태:** Primary → `#003B70` 다크닝, 트랜지션 200ms (추정)

### 카드 컴포넌트
- 배경: `#FFFFFF`, 그림자: `0 2px 8px rgba(0,0,0,0.08)` (추정)
- 경계선: `1px solid #F0F0F0` (추정)
- 내부 패딩: 24px (추정)

### 아이콘 시스템
- 라인 아이콘 중심, 스트로크 2px (추정)
- 색상: `#0066A0` 또는 `#1A1A1A` (추정)
- 표준 크기: 16px, 24px, 32px (추정)
