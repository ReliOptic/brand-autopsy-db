# F5 Inc. (FFIV) — 디자인 시스템

CSS 추출 데이터 기반으로 분류한 F5 공식 팔레트다.

### 컬러 사용 원칙 (추정)

- **Primary Blue (`#0E6EB9`)**: 모든 인터랙티브 요소의 기본색. CTA 버튼, 내비게이션 활성 상태, 하이퍼링크에 일관 적용
- **배경 계층**: `#FAF9F7`(페이지) → `#F4F4F5`(섹션) → `#FFFFFF`(카드) 3단계 배경 위계
- **시맨틱 컬러**: 그린·옐로·오렌지·레드 계열은 보안 상태 표시기 전용으로만 사용
- **액센트 핑크 (`#F06680`)**: 위협 수준 표시, 긴급 CTA에 제한적으로 사용

## 디자인 원칙 (추정)

1. **데이터 밀도 우선**: 대시보드·리포트는 스크롤 없이 최대 정보 제공
2. **보안 시각화**: 위협 지도, 트래픽 플로우 다이어그램을 인터랙티브하게 구현
3. **일관된 아이콘 언어**: 제품 전반에 동일한 아이콘 세트 적용 (보안 방패, 클라우드, 서버 등)
4. **접근성 (WCAG 2.1 AA)**: `#0E6EB9`와 `#FAF9F7` 조합 대비율 4.5:1 이상 유지

## Steal Sheet — 디자인 레이어별

### Steal Sheet 1: 브랜드 인지도 강화

| 레이어 | 전술 |
|--------|------|
| 인식 | neusaNextProWide 와이드 자간을 광고·OOH에 적극 활용해 F5 특유의 시각적 목소리 구축 |
| 고려 | Primary Blue(`#0E6EB9`) 단색 배경의 보안 위협 인포그래픽으로 소셜 피드 점령 |
| 결정 | 제품 UI 스크린샷의 상태 컬러 코딩을 마케팅 자료에도 일관 적용, 제품 친숙도 선행 구축 |

### Steal Sheet 2: 기술 신뢰도 시각화

| 레이어 | 전술 |
|--------|------|
| 인식 | Roboto Mono 코드 스니펫을 소셜 카드에 배치 — "개발자가 쓰는 제품" 이미지 강화 |
| 고려 | 아키텍처 다이어그램을 인터랙티브 SVG로 제공, 기술 블로그 체류시간 증가 |
| 결정 | PoC 가이드 문서에 코드 하이라이팅 적용, 엔지니어 설득력 극대화 |

### Steal Sheet 3: 다크 모드 / 보안 대시보드 감성

| 레이어 | 전술 |
|--------|------|
| 인식 | 다크 배경(`#222222`)에 Primary Blue·Green·Red 시맨틱 컬러 배치한 위협 대시보드 스크린샷을 광고에 활용 |
| 고려 | SOC 팀 타깃 콘텐츠에 다크 UI 스타일 적용, "실제 운영 환경" 리얼리티 연출 |
| 결정 | 제품 데모 환경을 다크 모드 기본으로 설정, 보안 전문가 집단의 심리적 친밀감 유도 |
