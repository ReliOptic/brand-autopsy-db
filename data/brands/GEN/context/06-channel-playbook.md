# 06. 채널 플레이북 — Gen Digital (GEN)

## 채널 전략 개요

Gen Digital은 소비자 직접 도달(D2C) 중심의 디지털 채널 전략을 운영한다. 검색 기반 인텐트 포착, 소셜 인지도 확장, 이메일 리텐션의 3축 구조로 운영되며, 산하 브랜드별 채널 믹스를 독립적으로 최적화한다. (추정)

---

## 채널별 플레이북

### 채널 1: 검색 광고 (SEM / Google Ads)

| 항목 | 내용 |
|------|------|
| 역할 | 구매 의도 높은 사용자 직접 포착 |
| 주요 키워드 | "바이러스 백신 추천", "VPN 서비스", "신원도용 보호", "Norton 가격" |
| 입찰 전략 | 브랜드 키워드 방어 + 경쟁사 키워드 공략 병행 (추정) |
| 랜딩 페이지 | 제품별 전용 LP, 무료 체험 CTA 최상단 배치 |
| 핵심 메시지 | "지금 무료 체험 — 설치 30초, 즉시 보호" |
| KPI | CPA, 전환율, ROAS |
| 예산 분배 | 전체 디지털 예산의 35~40% 추정 |

---

### 채널 2: SEO / 오가닉 콘텐츠

| 항목 | 내용 |
|------|------|
| 역할 | 장기 브랜드 권위 구축, 낮은 CAC 획득 |
| 콘텐츠 유형 | 위협 교육 블로그, 비교 가이드, How-to 아티클 |
| 타깃 키워드 | 정보성 롱테일 ("피싱 이메일 구별법", "VPN이 필요한 이유") |
| 게시 빈도 | 주 3~5회 (추정) |
| 내부 링크 전략 | 교육 콘텐츠 → 제품 페이지 → 무료 체험 CTA |
| 핵심 메시지 | 제품 언급 최소화, 순수 교육 가치 제공 후 자연스러운 전환 |
| KPI | 오가닉 트래픽, 키워드 순위, 무료 체험 전환율 |

---

### 채널 3: 소셜 미디어 (Meta / Instagram / X / YouTube)

| 플랫폼 | 주 역할 | 콘텐츠 믹스 | 톤 |
|--------|--------|------------|---|
| Meta (Facebook) | 35세+ 가족형 타깃 리타게팅 | 카드뉴스, 영상 광고, 패밀리 플랜 프로모션 | 따뜻하고 안심시키는 |
| Instagram | MZ 프라이버시 인식 확장 | 인포그래픽, 리얼스, 스토리 | 세련되고 간결한 |
| X (Twitter) | 실시간 위협 뉴스재킹, PR | 스레드, 빠른 팁, 사고 대응 | 전문가적, 즉각적 |
| YouTube | 심층 교육, 제품 리뷰 | 튜토리얼, 비교 영상, 보안 팁 시리즈 | 친절한 전문가 |
| LinkedIn | B2B·투자자·파트너 관계 | 기업 성과, 업계 리포트 | 권위 있는 비즈니스 |

**소셜 콘텐츠 원칙:**
- 80% 교육/가치 콘텐츠, 20% 제품 프로모션 (추정)
- 위협 뉴스 발생 24시간 이내 반응 콘텐츠 발행
- 사용자 생성 콘텐츠(UGC) 및 리뷰 적극 큐레이션

### 채널 4: 이메일 마케팅

| 세그먼트 | 주 목적 | 주요 이메일 유형 | 발송 빈도 |
|---------|--------|----------------|---------|
| 무료 사용자 | 유료 전환 | 기능 비교, 제한 알림, 업그레이드 오퍼 | 월 2~3회 |
| 유료 구독자 | 리텐션·갱신 | 보안 리포트, 신기능 소개, 갱신 알림 | 월 1~2회 |
| 이탈 위험 | 재활성화 | 윈백 오퍼, "당신이 없는 동안 일어난 일" | 이탈 신호 감지 즉시 |
| 신규 가입 | 온보딩 | 설치 가이드, 첫 주 팁, 기능 소개 시리즈 | 가입 후 7일간 집중 |

**이메일 디자인 원칙:**
- 제목 라인: 50자 이내, 숫자·의문문 활용
- 배경: `#F2F4F7` (라이트), CTA 버튼: `#3352FF`
- 모바일 우선 단일 컬럼 레이아웃
- 개인화 토큰 필수 (이름, 현재 구독 플랜, 기기 수)

## 채널 Steal Sheet — 3개 레이어

### Steal Sheet 1 — 검색 의도 포착 + 즉시 전환 최적화

| 레이어 | 전략 |
|--------|------|
| 인사이트 | 보안 구매는 사건 발생 직후 72시간이 최고 전환 시점 (추정) |
| 액션 | 대형 사이버 사고 발생 시 관련 키워드 입찰가 즉시 상향 |
| 랜딩 | "지금 당신의 데이터가 유출되었는지 확인하세요" 전용 LP 상시 대기 |
| CTA | 무료 다크웹 스캔 → 결과 확인 → 유료 전환 퍼널 |
| KPI | 사고 발생 후 48시간 무료 체험 가입 수 |

### Steal Sheet 2 — 소셜 감성 + 위협 실시간 뉴스재킹

| 레이어 | 전략 |
|--------|------|
| 인사이트 | 보안 위협 뉴스는 72시간 내 관련 제품 검색 300~500% 급증 (추정) |
| 액션 | 소셜 리스닝 툴로 트렌딩 보안 키워드 실시간 모니터링 |
| 포맷 | X 스레드 "오늘의 위협 브리핑" + Instagram 카드뉴스 |
| 차별화 | 공포 조장 없이 "해결책 먼저" 프레임으로 차별화 |
| KPI | 뉴스재킹 포스트 인게이지먼트율, 프로필 방문 전환율 |

### Steal Sheet 3 — 이메일 갱신 이탈 방어 (Churn Defense)

| 레이어 | 전략 |
|--------|------|
| 인사이트 | 구독 갱신 30일 전이 이탈 위험 최고점 (추정) |
| 시퀀스 | D-30: "당신이 보호받은 1년" 개인화 보안 리포트 이메일 |
| | D-14: 갱신 시 추가 기기 무료 혜택 오퍼 |
| | D-7: "갱신하지 않으면 사라지는 것들" 체크리스트 |
| | D-1: 마지막 기회 + 긴급 할인 |
| 디자인 | 개인화 데이터 시각화 (차단된 위협 수, 스캔 횟수) 포함 |
| KPI | 갱신율, 갱신 시 업그레이드 전환율 |

## 채널별 금지어 / 필수어 요약

### 금지어 (전 채널 공통)

| # | 금지어 | 이유 |
|---|--------|------|
| 1 | "당신은 해킹당했습니다" (단정) | 허위 사실, 공포 조장 |
| 2 | "무료로 다 된다" | 유료 전환 저해 |
| 3 | "경쟁사보다 훨씬 낫습니다" (비교 광고 무근거) | 법적 리스크 |

| 5 | "복잡한 설정 필요 없음" (기능 제한 암시 없이) | 과장 오해 소지 |

| # | 필수어 | 적용 맥락 |
