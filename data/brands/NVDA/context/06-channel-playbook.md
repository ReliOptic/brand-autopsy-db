## 채널별 전략

| 항목 | 내용 |
|------|------|
| 역할 | 연간 브랜드 선언의 장. 신규 아키텍처·플랫폼·파트너십 공개 |

## 채널 시너지 맵

```
GTC 키노트 (씨앗)
    ├── YouTube: 키노트 풀영상 + 하이라이트 클립
    ├── X: 실시간 발표 업데이트 + 핵심 수치 인용
    ├── LinkedIn: 기업 임팩트 요약 + 파트너 반응
    ├── 공식 블로그: 기술 심층 분석 아티클
    └── Developer 채널: 코드 예제 + NGC 컨테이너 업데이트
```

**시너지 원칙**: GTC 한 번의 이벤트에서 6개 채널의 콘텐츠가 파생된다. 제작 비용 1, 배포 효과 6.

### Steal Sheet — 마케터가 훔쳐갈 3가지

1. **"단일 이벤트에서 6채널 콘텐츠를 동시에 파생시켜라."** GTC 키노트 하나가 YouTube 풀영상, X 하이라이트, LinkedIn 분석 포스트, 블로그 심층 글, 개발자 코드 예제, NGC 업데이트로 자동 분기된다. 제작 비용은 하나지만 배포 효과는 6배다. → 마케터 적용: 모든 핵심 이벤트(제품 출시, 연례 보고, 오프라인 행사)를 채널별 콘텐츠 분기 계획과 함께 설계하라. 이벤트 당일이 아닌 2주 전부터 채널별 파생 콘텐츠 캘린더를 확정하라.

2. **"위기 대응에서 침묵은 최악의 선택이다."** NVIDIA는 드라이버 버그가 발견되면 침묵 대신 24시간 내 인정+원인 설명+타임라인 공개로 대응한다. 이것이 오히려 개발자 신뢰를 높인다. → 적용: 위기 대응 매뉴얼에 "침묵 금지" 원칙을 명문화하라. 미완성이더라도 "조사 중"이라는 즉각 인정이 48시간 침묵보다 낫다.

3. **"창업자 X 계정을 제2의 공식 채널로 운용하라."** 젠슨 황의 X 계정 팔로워와 반응은 NVIDIA 공식 계정을 종종 능가한다. 창업자가 직접 쓰는 것처럼 보이는 투명한 소통이 기업 PR보다 더 강력하다. → 적용: 창업자/CEO의 개인 소셜을 브랜드 채널 중 하나로 공식 인정하고, 어조·빈도·주제를 느슨하게 가이드하되 자유도를 최대한 보장하라.
