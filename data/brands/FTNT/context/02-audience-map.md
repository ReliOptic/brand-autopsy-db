# 02. 타겟 오디언스 지형도 — FORTINET (FTNT)

## ICP (Ideal Customer Profile)

포티넷의 본진 ICP는 "글로벌 중대형 엔터프라이즈 CISO/네트워크 보안 아키텍트"에서 출발했다. 2010년대 후반부터 통신사·정부 기관·중견 MSP(Managed Service Provider)·OT(산업제어) 운영자로 동심원을 확장 중이다. 한국 시장에서는 금융권·공공기관·제조 대기업이 핵심 확장 링이다.

| 항목 | 정의 |
|------|------|
| 산업/신분 | 금융·통신·정부·헬스케어·제조·리테일 대기업 IT/보안 부서. 보안을 '비용 센터'가 아닌 '리스크 관리 인프라'로 인식하는 조직 |
| 인구 규모 (추정) | 전 세계 50만 개 이상 고객사 (공식). 한국 시장은 1,000여 개 대형 고객사 + 3,000여 개 중견·SMB 파트너 채널 경유 |
| 의사결정자 역할 | CISO, CTO, 네트워크 아키텍트, IT 인프라 총괄. 최종 예산 승인은 CFO/CEO까지 에스컬레이션되는 다단계 구조 |
| 의사결정 구조 | 평균 구매 사이클 6~18개월. PoC(기술 검증) → RFP → 벤더 평가 → 이사회 승인. 영향력자는 보안 엔지니어·컨설턴트·시스템 인테그레이터(SI) |
| 예산 범위 | SMB: 연 $10만~$50만 / 엔터프라이즈: 연 $100만~$1,000만+ / 정부·통신사: 별도 예산 사이클. 갱신 계약(FortiGuard 구독) 비중 증가 추세 |
| 도입 성숙도 | 신규 도입보다 '기존 솔루션 교체·통합' 니즈가 대부분. 멀티벤더 피로(Vendor Fatigue)가 포티넷으로의 이탈 주요 트리거 |

| 정보 소비 패턴 | Gartner 리포트, KISA 보안공지, 금감원 지침, FortiGuard 위협 인텔리전스 피드, ISAC 커뮤니티, CISO 피어 네트워크 (비공개 점심 미팅) |
| 구매 트리거 | ① 대형 금융사 랜섬웨어 사고 뉴스 → 이사회 긴급 지시 ② 기존 방화벽 EOS(End of Support) 도래 ③ FortiGate PoC 성능 결과가 기존 벤더 압도 ④ 포티넷 파트너 SI의 TCO 비교 분석 제출 |
| 주요 반론 | "포티넷은 중저가 이미지가 있다 — 이사회에 설명하기 어렵다", "지원 조직의 한국어 서비스 품질이 불안하다", "오픈 에코시스템과 통합이 잘 되나?" |

## 구매 여정별 고객 질문

| 단계 | 고객이 묻는 질문 | 감정 상태 | 우리가 제공할 콘텐츠 |
|------|-----------------|-----------|---------------------|
| Awareness | "요즘 어떤 위협이 실제로 위험한가? 우리 업계는 어디서 많이 뚫리나?" | 불안+정보 탐색 | FortiGuard 위협 인텔리전스 리포트, 업종별 사고 사례 백서, CISO 서밋 기조연설 |
| Consideration | "포티넷이 Palo Alto·Cisco 대비 실제로 성능이 좋은가? 통합이 진짜 되는가?" | 비교+의심 | NSS Labs/Gartner 비교 데이터시트, Security Fabric 데모 영상, 동종 업계 레퍼런스 케이스 |
| Decision | "TCO가 정말 낮은가? 도입 후 운영 복잡도는?" | 리스크 계산+손실 회피 | TCO 계산기, PoC 지원, 파트너 기술 검증, 계약 조건 협의 |
| Retention | "FortiGuard 갱신 가치가 있나? 새 기능은 언제 나오나?" | 습관+재평가 | FortiOS 업데이트 노트, 연간 위협 리포트, 고객 성공 사례, NSE 교육 무료 제공 |

## 트리거 이벤트 캘린더
- **1~2월**: 글로벌 사이버보안 예측 리포트 시즌 — FortiGuard 연간 위협 보고서 발표. "올해 가장 위험한 공격은?"
- **3월**: RSA Conference 준비 시즌 — 경쟁사 대비 포지셔닝 자료 업데이트 필요
- **4~5월**: 국내 IT·보안 전시회(정보보호 & 개인정보 보호 컨퍼런스, ISMS-P 갱신 시즌) — 컴플라이언스 피로가 교체 트리거
- **6월**: 상반기 예산 검토 시즌 — 기존 벤더 계약 재평가, PoC 요청 집중
- **9월**: 하반기 예산 배정 — 대형 엔터프라이즈 내년 인프라 투자 계획 확정 시점
- **11~12월**: 연말 예산 소진 + 갱신 계약 집중 시즌 — FortiGuard 구독 갱신 협상 피크
- **비정기**: 대형 랜섬웨어·공급망 공격 뉴스(SolarWinds, Log4Shell류) → 즉각 대응 콘텐츠 배포, 긴급 웨비나 개최

트리거 캘린더의 본질: **"보안은 공격 뉴스로 산다."** 포티넷은 위협 인텔리전스(FortiGuard)가 실시간 콘텐츠 트리거 역할을 한다 — 이것이 미디어-보안 파이프라인의 속도 무기다.
