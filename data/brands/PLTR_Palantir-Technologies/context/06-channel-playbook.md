# 06. 채널 플레이북 — 팔란티어 테크놀로지스 (Palantir Technologies)

## 채널 전략 개요

팔란티어는 전통적인 B2B 마케팅 채널 조합을 따르지 않는다. 광고 집행 최소화, CEO 직접 소통 최대화, 이벤트 기반 콘텐츠 생산이 핵심 구조다. 각 채널은 독립적인 콘텐츠 허브가 아닌, 단일 임무 서사를 각기 다른 형식으로 전달하는 레이어로 운용된다.

---

## 채널별 전략

### 채널 1 — 공식 웹사이트 (palantir.com)

| 항목 | 내용 |
|------|------|
| 역할 | 브랜드 포지셔닝 기준점·제품 정보 허브·채용 입구 |
| 주요 오디언스 | 정부·기업 의사결정자, 채용 후보자, 투자자 |
| 콘텐츠 유형 | 히어로 카피, 제품 소개(Gotham/Foundry/AIP), 케이스스터디, 채용 공고 |
| 톤 | 선언형·권위적·간결. 설명보다 입장 |
| KPI (추정) | 데모 요청 수, 채용 지원 수, 투자자 문의 |
| 금지 패턴 | 슬라이드쇼형 배너, 팝업 광고, 과도한 소셜 증명 배지 |

**운용 원칙:** 홈페이지 히어로 섹션은 제품 기능이 아닌 존재 이유(Why)를 먼저 선언한다. "Palantir builds the software that powers the world's most important institutions." 같은 선언이 기술 스펙보다 앞에 위치한다.

### 채널 2 — X (구 트위터)

| 항목 | 내용 |
|------|------|
| 역할 | CEO 직접 소통·입장 선언·커뮤니티 반응 창출 |
| 주요 오디언스 | 개인 투자자(PLTR 커뮤니티), 테크 업계 관계자, 미디어 |
| 콘텐츠 유형 | CEO 입장 발언, 계약 수주 공지, AIPCon 하이라이트, 뉴스 반응 |
| 톤 | 논쟁적·선언형·이분법적. 중립 표현 최소화 |
| 계정 구조 | @PalantirTech (브랜드) + @AlexKarp (CEO 개인) — CEO 계정의 영향력이 더 큼 (추정) |
| KPI (추정) | 리트윗·인용 수, 커뮤니티 토론 건수, 미디어 픽업 건수 |

**운용 원칙:** X는 설명 채널이 아닌 입장 채널이다. "우리는 미국이 이기기를 원한다"처럼 동의 또는 반박을 즉시 유발하는 문장을 설계한다. 논란은 노출의 연료다.

**Steal Sheet — X 채널 전용:**
- CEO 발언이 브랜드 계정보다 팔로워·인게이지먼트가 높다면 CEO를 콘텐츠 리소스의 우선순위로 배치하라
- 분기 실적 발표일 전후 24시간이 가장 높은 유기 도달 시점 (투자자 커뮤니티 활성화 기반, 추정)
- 리트윗을 유발하는 문장 구조: "[우리가 믿는 것] + [그것이 왜 옳은가] + [동의하지 않는다면 우리는 당신을 위한 회사가 아니다]"

## 채널 금지 패턴

- **대량 디지털 광고:** 팔란티어 포지셔닝(엘리트·임무 특화)과 구글/메타 배너 광고의 mass-market 이미지가 충돌
- **인플루언서 마케팅:** 외부 인플루언서를 통한 제품 홍보는 권위 희석. CEO가 유일한 인플루언서
- **콘텐츠 마케팅 자동화:** SEO 중심의 양산형 블로그 포스트는 팔란티어의 사상 리더십 포지션을 약화시킴
- **소셜 캠페인 해시태그 마케팅:** 버즈·바이럴 중심 캠페인은 임무 서사와 부조화

### Steal Sheet — 마케터가 훔쳐갈 3가지

1. **"이벤트 하나로 채널 전체를 가동하라."** AIPCon 단일 이벤트가 YouTube·LinkedIn·X·언론 보도·투자자 커뮤니티를 동시에 활성화한다. 이벤트가 콘텐츠 캘린더의 앵커가 되면 나머지 채널 콘텐츠가 자동으로 파생된다. → 마케터 적용: 연간 1~2개의 "앵커 이벤트"를 먼저 확정하고, 모든 채널 콘텐츠를 그 이벤트 전후로 배치하는 역방향 캘린더를 설계하라.

2. **"CEO의 논쟁적 발언이 유료 광고보다 도달이 높다."** Alex Karp의 단 하나의 인터뷰 발언이 CNN·Bloomberg·CNBC에 동시 보도되며, 팔란티어 브랜드 노출을 수십억 원의 광고 예산 없이 달성한다 (추정). → 적용: PR 예산을 광고로 분산하지 말고 CEO 미디어 트레이닝·인터뷰 배치에 집중하라. 논쟁을 두려워하지 않는 CEO는 브랜드의 가장 값싼 미디어 채널이다.

3. **"직접 영업을 콘텐츠로 만들어라."** 팔란티어의 고객 현장 배포 스토리는 직접 영업 과정에서 수집되고, 이것이 케이스스터디·AIPCon 시나리오·주주서한 사례로 재활용된다. 영업과 콘텐츠 마케팅이 분리된 파이프라인이 아닌 단일 루프로 연결된다. → 적용: 영업팀이 현장에서 수집하는 고객 문제·해결 과정·결과 수치를 콘텐츠팀과 실시간 공유하는 구조를 만들면, 마케팅 예산 없이 가장 설득력 있는 콘텐츠가 자동 생산된다.
