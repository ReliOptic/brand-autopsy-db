# 05 디자인 시스템 — PayPal (PYPL)

## 디자인 철학

PayPal의 디자인은 "신뢰 우선 시각 언어(Trust-First Visual Language)"를 기반으로 한다. 모든 시각적 결정은 사용자의 불안을 낮추고 행동에 대한 자신감을 높이는 방향으로 설계된다. (추정)

## 모션 & 애니메이션 원칙 (추정)

| 원칙 | 값 | 이유 |
|------|-----|------|
| 기본 전환 시간 | 200ms | 빠르지만 인식 가능 |
| 페이지 전환 | 300ms | 컨텍스트 유지 |
| 이징 함수 | ease-out | 자연스러운 감속 |
| 결제 완료 애니메이션 | 500ms | 만족감 강화 |
| 오류 진동 | 300ms shake | 즉각적 피드백 |

## 이미지 & 사진 가이드라인

### 사진 톤 (추정)

- 밝고 낙관적인 조명 (어두운 드라마 금지)
- 실제 사람, 다양성 반영
- 결제 행위보다 "결과"(자유, 성공, 연결) 시각화
- 스마트폰/노트북 기기 일상적으로 등장

### 금지 이미지 (추정)

- 지갑에서 현금 꺼내는 이미지 (디지털과 상충)
- 복잡한 서류/관료주의 이미지
- 불안하거나 걱정하는 표정
- 구식 금융기관 연상 이미지
