# 05. 디자인 시스템 해부 — 팔로알토 네트웍스 (Palo Alto Networks)

## 디자인 철학
팔로알토 네트웍스의 비주얼 언어는 "정밀함(Precision)과 신뢰(Trust)"를 시각적으로 번역한다. 어두운 배경 위에 선명한 주황-빨강 액센트가 위협의 긴박감과 탐지 능력을 동시에 표현하며, 데이터 시각화 중심의 레이아웃이 기술 권위를 뒷받침한다. 군더더기 없는 구조와 높은 정보 밀도가 CISO·엔지니어 대상 신뢰감을 형성한다.

- **다크 퍼스트**: `#141414` 다크 배경이 전체 디자인의 기조를 설정. 위협 대시보드·제품 UI는 다크 모드 우선

- **신호등 체계**: Green(`#00CC66`) = 안전, Yellow(`#FFCB06`) = 경고, Red(`#C84727`~`#FF2E00`) = 위험 — 보안 대시보드 전반에 일관 적용

### 타이포그래피 원칙
- 헤딩은 TT Hoves 볼드·세미볼드로 선명한 계층 구조 형성

- 코드 블록은 모노스페이스 폰트로 명확히 구분, 다크 배경에 밝은 텍스트

### Steal Sheet — 마케터가 훔쳐갈 3가지

1. **"컬러로 위협 레벨을 시각화하라."** PANW의 신호등 컬러 체계(그린-옐로우-레드)는 보안 대시보드뿐 아니라 마케팅 콘텐츠(인포그래픽·리포트·슬라이드)에도 일관 적용된다. 독자가 텍스트를 읽기 전에 색상만으로 리스크 수준을 직관적으로 파악할 수 있다 (추정: 공식 리포트 디자인 관찰 기반). → 적용: 보고서·대시보드·이메일에서 데이터의 심각도를 컬러 코드로 즉각 전달하면, 바쁜 경영진이 핵심 메시지를 빠르게 파악할 수 있다.

2. **"다크 배경을 프리미엄·기술 권위의 신호로 활용하라."** `#141414` 다크 배경은 단순한 심미적 선택이 아니라 "우리는 심각한 위협을 다루는 진지한 기술 회사"라는 포지셔닝 신호다. 소비재 브랜드의 밝고 친근한 팔레트와 명확히 차별화된다 (추정: 업계 비주얼 포지셔닝 관찰 기반). → 적용: 엔터프라이즈·전문가 대상 브랜드라면 다크 배경과 고채도 액센트 조합이 신뢰도와 기술 권위를 동시에 전달하는 효과적인 선택이다.

3. **"폰트 하나에 브랜드 개성을 집중하라."** TT Hoves라는 독립 커스텀 폰트는 PANW의 시각 자산 중 가장 일관되게 브랜드 인지도를 형성하는 요소다. 헬베티카·아리알 계열 범용 폰트와의 차별화가 즉각적으로 인지된다 (추정: 공식 웹사이트 CSS 분석 기반). → 적용: 범용 시스템 폰트 대신 브랜드 전용 폰트 하나에 투자하면, 광고·문서·웹사이트 전반에서 브랜드 일관성이 극적으로 높아진다.
