### 컬러 역할 해석

| 역할 군 | 포함 HEX | 디자인 의도 |

| 주요 브랜드 컬러 | `#0032EF`, `#0032FF`, `#003F7A` | 신뢰·기술·기업 정체성 표현 |
| 에너지·혁신 강조 | `#A9DE32`, `#55F0FF` | 기술 혁신·광자 에너지 시각화 (추정) |
| 중립 배경 시스템 | `#F0F0F0`~`#E3E3E3` | 콘텐츠 가독성·클린 레이아웃 지원 |
| 텍스트 계층 | `#2D2D2D`, `#646464`, `#707070` | 제목→본문→보조 정보 위계 |
| 상태·피드백 색상 | `#F07D28`, `#FA574B`, `#37A6C4` | UI 경고·오류·정보 상태 표현 |

### 컬러 사용 원칙

1. **블루 우선**: `#0032EF`를 CTA 버튼, 링크, 주요 헤드라인에 사용한다.
2. **그린 절제 사용**: `#A9DE32`는 강조 배지, 성공 상태, 포인트 아이콘에만 한정한다.
3. **시안 계열은 기술적 맥락**: `#55F0FF`, `#37A6C4`는 다이어그램·인포그래픽·데이터 시각화에 사용한다.
4. **회색 계층 준수**: 배경 `#F3F3F3` → 섹션 구분 `#E3E3E3` → 텍스트 `#707070` 순서를 유지한다.
5. **오렌지·레드는 기능적 사용만**: 마케팅 장식보다 UI 상태 표현에 한정한다.

## 타이포그래피 시스템

### 폰트 패밀리

| 폰트명 | 분류 | 용도 | 출처 |
|--------|------|------|------|
| Franklin Gothic | 산세리프 (Humanist) | 주요 헤드라인, 브랜드 타이틀 | (공식) |
| Franklin Gothic Demi Condensed | 산세리프 (Condensed) | 서브헤드, 배너 텍스트 | (공식) |
| ITCFranklinGothicCompressed | 산세리프 (Compressed) | 대형 디스플레이 텍스트 | (공식) |
| ITCFranklinGothicCompressedThin | 산세리프 (Compressed Light) | 경량 디스플레이 레이어 | (공식) |
| Neue Haas (neuehaas) | 산세리프 (Geometric) | 본문, UI 텍스트 | (공식) |
| Arial | 산세리프 (Fallback) | 시스템 폰트 폴백 | (공식) |
| icomoon | 아이콘 폰트 | UI 아이콘 시스템 | (공식) |
| swiper-icons | 아이콘 폰트 | 슬라이더 컨트롤 아이콘 | (공식) |

### 타이포그래피 해석

- **Franklin Gothic 계열 독점 사용**: 브랜드 일관성을 위해 헤드라인 전반에 Franklin Gothic 변형을 사용한다. 압축형(Compressed) 변형은 기술 정밀함과 산업적 강인함을 시각적으로 전달한다.
- **Neue Haas (Helvetica 계열)**: 본문 가독성 최적화. 기술 문서·데이터시트에서의 장문 읽기를 지원한다.
- **아이콘 폰트 활용**: 이미지 대신 벡터 아이콘 폰트(icomoon)를 사용해 성능과 확장성을 확보한다 (추정).

## 디자인 원칙

| 원칙 | 설명 |
|------|------|
| 정밀성(Precision) | 여백, 정렬, 간격 모두 그리드 기반으로 일관성 유지 |
| 기술적 권위(Authority) | 과도한 장식 배제, 데이터·다이어그램 중심 레이아웃 |
| 가독성 우선(Readability) | 회색 계층을 통한 콘텐츠 위계 명확화 |
| 에너지 포인트(Energy Accent) | 그린·시안 계열로 광자·에너지 개념을 시각적으로 암시 |
| 스케일(Scale) | 압축형 폰트와 대형 수치 타이포그래피로 기술 스케일 표현 |

## 컴포넌트 설계 방향 (추정)

- **버튼**: `#0032EF` 배경, 흰색 텍스트, 4px 라운드 또는 직각 (기술 브랜드 특성)
- **카드**: 흰색 배경, `#E3E3E3` 테두리, 16~24px 패딩
- **표**: `#F0F0F0` 헤더 배경, `#2D2D2D` 텍스트, 1px `#C8C8C8` 구분선
- **배지/태그**: `#A9DE32` 배경, `#2D2D2D` 텍스트 (성공/신규), `#55F0FF` 배경 (기술 하이라이트)
- **경고 배너**: `#F07D28` 좌측 보더 또는 배경 틴트
