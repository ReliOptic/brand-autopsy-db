# 02. 오디언스 맵 — Amphenol (APH)

## 오디언스 계층 구조

```
1차 의사결정자 (Primary Decision Makers)
├── 하드웨어 엔지니어 / 설계 엔지니어
├── 조달 관리자 (Procurement Manager)
└── 시스템 아키텍트

2차 영향자 (Secondary Influencers)
├── 품질보증(QA) 엔지니어

3차 이해관계자 (Tertiary Stakeholders)
├── 기관 투자자 (IR 타겟)
├── OEM 파트너사 임원
└── 규제기관 / 인증기관
```

## 페르소나 정의

### 페르소나 1: 박재현 — 시스템 설계 엔지니어

| 항목 | 내용 |
|------|------|
| 나이 / 직급 | 38세 / 시니어 하드웨어 엔지니어 |
| 소속 | 국내 대형 방산·항공 1차 협력사 |
| 학력 | 전기전자공학 학사, 임베디드 시스템 석사 |
| 연봉 | 7,500만 원 (추정) |
| 주요 KPI | 설계 오류율 0%, 납기 준수율 99%, 인증 통과율 |

**핵심 고민**
> "MIL-SPEC 요건을 충족하면서 납기 12주 이내에 맞출 수 있는 커넥터 공급사가 얼마나 될까?"

**정보 수집 채널**
- IEEE Xplore, Connector Industry 전문지
- 공급사 데이터시트 직접 다운로드
- LinkedIn 기술 그룹, 국내 전자부품 유통사 기술 세미나

**구매 결정 요인 (우선순위)**
1. 기술 규격 충족 여부 (임피던스, 온도 범위, 진동 내성)
2. 납기 신뢰성 및 재고 가용성
3. 인증서류 (MIL-DTL, IPC, RoHS) 완비 여부
4. 국내 기술 지원 가능 여부

**페인 포인트**
- 글로벌 공급난 시 대체 소싱 경로 부재
- 영문 데이터시트만 제공되어 사내 검토 지연
- 샘플 요청 후 응답까지 2~3주 소요

**Amphenol 인식**
> "기술력은 최고지만 가격이 비싸고 국내 지원이 약하다. 그래도 방산 프로젝트에선 선택지가 없다." (추정)

### 페르소나 2: 이수미 — 글로벌 조달 관리자

| 항목 | 내용 |
|------|------|
| 나이 / 직급 | 44세 / 글로벌 소싱 팀장 |
| 소속 | 삼성전자 네트워크 사업부 (유사 프로필 — 추정) |
| 학력 | 경영학 학사, SCM 자격증 (CPSM) |
| 연봉 | 1억 1,000만 원 (추정) |
| 주요 KPI | 부품 단가 절감율, 공급 다변화 지수, On-Time Delivery |

**핵심 고민**
> "Amphenol 단일 소싱 의존도가 너무 높다. 2차 벤더 육성과 가격 협상력 확보가 동시에 필요하다."

**정보 수집 채널**
- Gartner SCM 리포트
- 공급사 QBR (분기 사업 리뷰) 데이터
- IHS Markit 부품 시장 분석

**구매 결정 요인 (우선순위)**
1. 총소유비용 (TCO) 및 볼륨 할인 구조
2. 공급 연속성 보증 (BCP 문서)
3. ESG 컴플라이언스 (분쟁광물, 환경 인증)
4. ERP 연동 및 EDI 지원

**페인 포인트**
- 가격 투명성 부족, 글로벌 vs 로컬 가격 차이
- 장기 공급 계약 시 가격 에스컬레이션 조항 복잡
- 대체 공급사 검증에 6~9개월 소요

**Amphenol 인식**
> "프리미엄 공급사지만 협상 여지가 좁다. 납기와 품질로 우리를 묶어두는 전략이 보인다." (추정)

### 페르소나 3: 마이클 탠 (Michael Tan) — AI 인프라 스타트업 CTO

| 항목 | 내용 |
|------|------|
| 나이 / 직급 | 35세 / Co-founder & CTO |
| 소속 | 싱가포르 기반 AI 가속 칩 스타트업 |
| 학력 | 컴퓨터공학 학사 (NUS), 스탠퍼드 AI 랩 출신 |
| 연봉 | $220,000 USD + 스톡옵션 (추정) |
| 주요 KPI | 제품 출시 속도, PCIe 5.0/CXL 호환성, 전력 효율 |

**핵심 고민**
> "112G PAM4 고속 인터커넥트를 소량으로 신속하게 프로토타입할 수 있는 공급사가 필요하다. MOQ가 문제다."

**정보 수집 채널**
- IEEE DesignCon, Hot Chips 학회
- GitHub 하드웨어 커뮤니티
- Amphenol TCS 기술 웹세미나

**구매 결정 요인 (우선순위)**
1. 최신 고속 규격 지원 (PCIe 5.0, CXL 2.0, 800G Ethernet)
2. 소량 프로토타입 대응 (낮은 MOQ)
3. 레퍼런스 디자인 및 시뮬레이션 데이터 제공
4. 납기 속도 (4주 이내)

**페인 포인트**
- 대기업 중심 영업 구조로 스타트업 응대 느림
- 고급 커넥터 기술 정보가 NDA 뒤에 잠겨있음
- 국내 유통사 재고 없어 직접 수입 필요

**Amphenol 인식**
> "AI 데이터센터 커넥터의 사실상 표준. 하지만 스타트업에게는 높은 벽이다." (추정)

## 구매 여정 (Buyer Journey)

```
인식 (Awareness)
└── 트리거: 신규 설계 프로젝트 착수, 기존 공급사 납기 이슈, 규격 업그레이드 필요

고려 (Consideration)
└── 행동: 데이터시트 비교, 샘플 요청, 기술 세미나 참석, 배포사 문의

평가 (Evaluation)
└── 기준: 규격 충족 여부, 가격/납기, 국내 기술지원, 인증서류

구매 (Purchase)
└── 경로: 공식 유통사, 직접 계약 (대량), 글로벌 e-procurement 플랫폼

사후 관리 (Retention)
└── 요소: 정기 QBR, 신규 제품 우선 샘플링, 기술 지원 응답속도
```
