# 05. 디자인 시스템 — Aflac (AFL)

## 디자인 철학

Aflac의 비주얼 언어는 **"친근한 신뢰"** 를 구현한다. 보험이라는 무거운 카테고리에서 오리 마스코트를 통해 접근 장벽을 낮추되, 컬러·타이포그래피·레이아웃에서는 금융 브랜드의 안정감을 유지한다. 밝고 따뜻하지만 가볍지 않다.

## 타이포그래피

### 서체 시스템 (추정)

| 역할 | 서체 | 스타일 | 용도 |
|------|------|--------|------|
| 헤드라인 | Gotham Bold / 유사 sans-serif | Bold, 700 | 광고 헤드라인, 히어로 텍스트 |
| 서브헤드 | Gotham Medium | Medium, 500 | 섹션 제목, 카드 제목 |
| 본문 | Open Sans / Roboto | Regular, 400 | 웹 본문, 설명 텍스트 |
| 숫자 강조 | Gotham Black | Black, 900 | "$1,000 지급" 등 수치 강조 |
| 법적 텍스트 | Open Sans | Regular, 400, 10px | 약관, 면책 문구 |

### 타이포그래피 원칙
- 헤드라인: 대문자 또는 타이틀 케이스, 짧고 강렬하게
- 본문: 최대 2-3문장, 한 단락 = 하나의 아이디어
- 숫자는 항상 크게 — 비용 절감 수치가 시각적 앵커

## 로고 시스템

### 기본 로고 (공식)
- "Aflac" 워드마크 + 오리 심볼 조합
- 가로형(수평) 배치가 기본
- 최소 크기: 디지털 80px 너비, 인쇄 25mm

### 로고 변형
| 변형 | 배경 | 로고 컬러 |
|------|------|-----------|
| 기본형 | 흰 배경 | Blue + Yellow |
| 반전형 | 파란 배경 | White + Yellow |
| 단색형 | 모노크롬 환경 | All White 또는 All Navy |

### 금지 사용
- 로고 비율 변형 금지
- 오리 심볼 단독 사용 시 브랜드 색상 준수
- 배경과 대비 낮은 환경 배치 금지

## 마스코트 — Aflac Duck

### 마스코트 가이드라인 (추정)
- 오리는 항상 긍정적·도움이 되는 역할로 등장
- 환자나 고통받는 사람을 조롱하는 맥락 절대 금지
- 오리가 "AFLAC!" 이라고 외치는 사운드는 브랜드 오디오 시그니처
- 디지털 애니메이션: 부드럽고 친근한 모션, 과도한 코믹 연출 지양

---

## 이미지 스타일

### 사진 촬영 방향 (추정)
- 실제 사람, 실제 감정 — 스톡 사진 느낌 지양
- 밝은 자연광, 따뜻한 색온도
- 가족 단위, 직장 환경, 의료 현장 (긍정적 결말 맥락)
- 다양성 반영: 인종, 연령, 가족 구성

### 금지 이미지 (추정)
- 병원 침대에서 고통받는 장면 (공포 유발)
- 쌓인 청구서 클로즈업 (절망감)
- 과도하게 연출된 행복 (비진정성)
