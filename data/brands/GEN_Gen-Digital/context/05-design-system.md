# 05. 디자인 시스템 — Gen Digital (GEN)

## 디자인 철학

Gen Digital의 디자인 언어는 **"신뢰할 수 있는 기술의 따뜻함"** 을 구현한다. 차갑고 기계적인 사이버 보안 미학에서 벗어나, 접근 가능하고 인간적이면서도 전문성을 시각적으로 전달한다. 산하 브랜드별로 독립적인 시각 정체성을 유지하면서도, Gen 모기업 레벨에서는 통합된 디지털 플랫폼 감성을 추구한다. (추정)

## 타이포그래피 시스템

### 폰트 패밀리 (공식 — CSS 데이터 직접 추출)

| 역할 | 폰트 | 비고 |
|------|------|------|
| 주 UI 폰트 | Inter | 웹 기본, 현대적 가독성 |
| 보조 본문 | Source Sans Pro (Regular/SemiBold/Bold) | 긴 텍스트 가독성 최적화 |
| 시스템 폴백 | Roboto (Regular/Medium/Bold) | Android/Chrome 환경 |
| 레거시 폴백 | Arial, Helvetica, Verdana | 구형 환경 호환 |
| 세리프 (강조용) | Baskerville | 신뢰감·권위 표현 특수 사용 |

### 타이포 스케일 (추정)

| 단계 | 크기 | 폰트 | 용도 |
|------|------|------|------|
| Display | 56~72px | Inter Bold | 히어로 헤드라인 |
| H1 | 40~48px | Inter Bold | 페이지 제목 |
| H2 | 32~36px | Inter SemiBold | 섹션 제목 |
| H3 | 24~28px | Inter Medium | 서브섹션 |
| Body Large | 18~20px | Source Sans Pro Regular | 리드 카피 |
| Body | 16px | Source Sans Pro Regular | 일반 본문 |
| Caption | 12~14px | Source Sans Pro Regular | 보조 텍스트 |

## 디자인 Steal Sheet — 3개 레이어

### Steal Sheet 1 — 컬러 시스템 전용 랜딩 페이지

| 레이어 | 전략 |
|--------|------|
| 히어로 | `#0B0B0B` 다크 배경 + `#3352FF` 글로우 그라디언트 |
| 신뢰 섹션 | `#F2F4F7` 배경 + `#47BCC3` 아이콘으로 "안전함" 감성 |
| CTA 버튼 | `#3352FF` 주 CTA + `#47BCC3` 보조 CTA로 선택 구조화 |
| 결과 | 전환율 최적화: 다크 히어로는 주목도, 라이트 섹션은 신뢰감 |

### Steal Sheet 2 — 타이포그래피 권위 구축

| 레이어 | 전략 |
|--------|------|
| 헤드라인 | Inter Bold 대형 + `#0400F5` 딥 블루로 강한 인상 |
| 신뢰 수치 | Display 크기 숫자 ("5억+ 사용자") + Source Sans Pro 설명 |
| 본문 | 16px Source Sans Pro Regular, 행간 1.6 — 읽기 편한 최적 구성 |
| 특수 강조 | Baskerville 이탤릭으로 고객 추천사 — 신뢰·권위 감성 |

### Steal Sheet 3 — 다크/라이트 듀얼 모드 전략

| 레이어 | 전략 |
|--------|------|
| 다크 모드 | `#0B0B0B` 배경 + `#BDC7FF` 텍스트 + `#3352FF` 강조 — 기술적 전문성 |
| 라이트 모드 | `#F2F4F7` 배경 + `#242424` 텍스트 + `#47BCC3` 포인트 — 친근함 |
| 적용 규칙 | 위협 관련 콘텐츠 = 다크, 솔루션/혜택 콘텐츠 = 라이트 |
| 심리 효과 | 문제(어둠) → 해결(밝음)의 시각적 여정 구현 |
