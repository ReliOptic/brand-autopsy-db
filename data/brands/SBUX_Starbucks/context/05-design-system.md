## 3. 타이포그래피

| 용도 | 서체 | 특성 |
|------|------|------|
| 헤드라인 | Sodo Sans (커스텀) | 스타벅스 전용 서체, 친근하고 현대적 (공식) |
| 본문 | Lander (커스텀) | 가독성 중심, 자연스러운 흐름 (공식) |
| 강조 | Pike (커스텀 세리프) | Reserve·프리미엄 콘텐츠 전용 (공식) |
| 디지털 대체 | Arial / Helvetica Neue | 커스텀 폰트 미지원 환경 (추정) |
| 한국어 | Noto Sans KR / 본고딕 | 한국 디지털 환경 최적화 (추정) |

### 타이포 위계

| 레벨 | 사이즈 | 사용처 |
|------|--------|--------|
| Display | 56~72px | 히어로 배너, 캠페인 타이틀 |
| H1 | 40~48px | 페이지 메인 타이틀 |
| H2 | 28~36px | 섹션 헤더 |
| Body Large | 18~20px | 리드 카피, 설명 텍스트 |
| Body | 14~16px | 일반 본문 |
| Caption | 12px | 주석, 법적 고지 |

## 4. 로고 시스템

| 버전 | 용도 | 사용 지침 |
|------|------|-----------|
| Full Lockup (Siren + Wordmark) | 소규모 마케팅, 앱 스플래시 | 공식 브랜드 외부 노출 (공식) |
| Siren Mark Only | 메인 로고, 컵, 패키지, 매장 사이니지 | 인지도 충분한 모든 터치포인트 (공식) |
| Reserve Mark | Reserve·Roastery 전용 | 일반 스타벅스와 시각적 분리 (공식) |
| Monochrome | 1도 인쇄, 엠보싱 | 단색 환경 전용 (추정) |

### 로고 보호 영역 (Clear Space)
- Siren Mark 높이의 25% 여백을 사방에 유지 (추정)
- 배경색: Dark Green, White, 자연광 이미지 위만 허용 (추정)

## 5. 매장 디자인 언어

| 요소 | 내용 |
|------|------|
| 소재 | 재생목재, 천연석, 황동, 세라믹 — 자연 소재 우선 (공식) |
| 조명 | 따뜻한 텅스텐 계열, 3,000K 이하 (추정) |
| 음향 | Starbucks 큐레이션 플레이리스트, 65dB 이하 환경 유지 (추정) |
| 향 | 커피 아로마 보존 위해 강한 향수·방향제 사용 금지 (공식) |
| 가구 | 1~2인 소파+바 시트 혼합, 장시간 체류 유도 (추정) |
| 신호 체계 | 픽업 스테이션 시인성, 모바일 오더 분리 동선 (공식) |

### 매장 유형별 디자인 차별화

| 유형 | 특성 |
|------|------|
| 표준 매장 | 브랜드 그린 + 자연 소재, 효율 동선 |
| Drive-Thru 전용 | 픽업 속도 최적화, 미니멀 인테리어 |
| Reserve Bar | 다크 우드·골드 강조, 오픈 바 구조 |
| Reserve Roastery | 산업용 로스터 노출, 멀티층 구조, 랜드마크 설계 |
| 지역 문화 반영점 | 현지 아트·소재 통합, 브랜드 가이드 내 로컬화 허용 |

## 8. Steal Sheet — 디자인 시스템 레이어

| # | 전략 | 실행 방법 | 적용 대상 |
|---|------|-----------|-----------|
| 1 | **시즌 컬러를 레이어로 운영** | `#00704A` 기반을 고정하고 시즌 팔레트를 오버레이로 교체 → 브랜드 일관성 유지하며 신선감 부여 | 패키징, SNS 템플릿 |
| 2 | **소재로 프리미엄 신호 전달** | Reserve 매장에 황동·대리석 집중, 일반 매장 재생목재 유지 → 시각적 계층 분리 | 매장 인테리어 |
| 3 | **컵이 마케팅 미디어** | 시즌 컵 디자인을 아티스트 협업으로 제작 → 수집 욕구 자극, SNS 인증샷 유도 | 패키징, SNS 캠페인 |
