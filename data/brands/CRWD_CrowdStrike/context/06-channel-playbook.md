# 06. 채널 플레이북 — CrowdStrike (CRWD)

## 채널 전략 개요

CrowdStrike는 **고관여 엔터프라이즈 B2B** 구매 사이클에 최적화된 채널 믹스를 운영한다. 인바운드(콘텐츠·SEO·위협 인텔리전스)로 신뢰를 구축하고, 아웃바운드(직접 영업·파트너 채널)로 딜을 클로징하는 이중 구조다. 단일 채널 의존 없이, 각 채널이 퍼널의 특정 단계를 전담한다.

---

## 채널별 상세 플레이북

### 채널 1 — 직접 영업 (Enterprise Direct Sales)

**역할**: 대형 엔터프라이즈($1M+ ACV) 딜 클로징

**운영 방식**
- Named Account 기반 커버리지 모델 (산업별 버티컬 SE 배치)
- Executive Briefing Center(EBC) — 고객 현장 방문 또는 CrowdStrike HQ 초청 브리핑
- Proof of Concept(PoC) 30일 무상 제공 후 전환 제안
- 경쟁 딜에서는 Competitive Intelligence 팀이 배틀카드 실시간 지원

**KPI**: ACV, Win Rate vs. 주요 경쟁사, Sales Cycle Length, PoC 전환율

**콘텐츠 지원**
- 세일즈 덱 (Executive Summary 버전 + Technical Deep Dive 버전 분리)
- IR 케이스 스터디 (동종업계 익명 사례)
- ROI 계산기 (TCO vs. 침해 비용 비교)

---

### 채널 2 — 파트너 / MSSP 채널

**역할**: Mid-Market 및 SMB 커버리지, 지역·산업 특화 영업

**운영 방식**
- CrowdStrike Elevate Partner Program — 티어별(Select/Advantaged/Elite) 마진 및 지원 차등
- MSSP 전용 Falcon 라이선스 모델 (멀티테넌트 관리 콘솔)
- 파트너 포털: 세일즈 자료, 기술 인증 트레이닝, 딜 등록 시스템
- 분기별 파트너 QBR (Quarterly Business Review) 및 인센티브 프로그램

**KPI**: 파트너 소싱 파이프라인 기여율, 파트너 인증 완료 수, 채널 ACV 성장률

**콘텐츠 지원**
- 파트너 전용 배틀카드 (경쟁사 대응 논리)
- 공동 마케팅 자금(MDF) 지원 캠페인 템플릿
- 파트너 브랜디드 웨비나 키트

---

### 채널 3 — 이벤트 / 컨퍼런스

**역할**: 브랜드 신뢰도 구축 + 파이프라인 생성 + 위협 인텔리전스 권위 각인

**주요 이벤트**
- **Fal.Con (자체 연례 컨퍼런스)**: 고객·파트너·업계 리더 대상, 신제품 발표 + 위협 인텔리전스 브리핑(공식)
- **RSA Conference**: 대형 부스 + C-Suite 타겟 Executive Roundtable
- **Black Hat / DEF CON**: 기술 청중 타겟 연구 발표 + Briefings 스폰서
- **Gartner Security Summit**: CISO 타겟 1:1 미팅 + 스폰서십

**운영 원칙**
- 이벤트 전: 사전 아웃리치로 미팅 슬롯 50% 이상 사전 확보
- 이벤트 중: 라이브 위협 인텔리전스 데모 + 실시간 Adversary 업데이트 공개
- 이벤트 후: 48시간 내 팔로업 시퀀스 실행

**KPI**: 미팅 수, MQL 생성, 파이프라인 영향액, 미디어 픽업 수

---

### 채널 4 — 디지털 마케팅 (Paid + Organic)

**Paid 채널**
- Google Search: "EDR solution", "endpoint security platform", "ransomware protection" 등 구매 의도 키워드
- LinkedIn Ads: CISO/CISO/VP Security 직함 타겟, 보안 업계 종사자 타겟
- Programmatic Display: 보안 미디어(Dark Reading, SecurityWeek, SC Media) 타겟 플레이스먼트
- Retargeting: 웹사이트 방문자, 위협 보고서 다운로더 대상 리타겟

**Organic 채널**
- SEO: 위협 인텔리전스 키워드 + 제품 비교 키워드 + 기술 튜토리얼
- Content Hub (crowdstrike.com/blog): 주 2–3회 기술 콘텐츠 발행
- YouTube: 위협 인텔리전스 브리핑, Falcon 데모, Fal.Con 세션 녹화

**KPI**: Organic Traffic Growth, CAC by Channel, MQL Volume, Conversion Rate by Source

---

### 채널 5 — 소셜 미디어

#### LinkedIn (핵심 채널)

**목적**: B2B 의사결정자 도달 + 위협 인텔리전스 권위 강화 + 채용 브랜딩

**콘텐츠 믹스 (추정)**
- 40% 위협 인텔리전스 업데이트 (적 활동 알림, 취약점 브리핑)
- 30% 제품/기능 업데이트 (신규 Falcon 모듈, 파트너십)
- 20% 고객 사례/수상 (Gartner 평가, 고객 인용)
- 10% 문화/채용 (보안 전문가 대상)

**게시 빈도**: 주 5–7회 (추정)

**성과 KPI**: Follower Growth, Engagement Rate, Inbound Lead Attribution

#### X (Twitter / 구 Twitter)

**목적**: 실시간 위협 알림 + 보안 커뮤니티 참여 + 저널리스트 관계

**콘텐츠 믹스**
- 실시간 위협 인텔리전스 ("BREAKING: CrowdStrike identified…")
- 기술 블로그 공유
- 업계 행사 라이브 트윗
- 보안 커뮤니티 대화 참여

**게시 빈도**: 일 2–5회 (추정)

#### YouTube

**목적**: 기술 청중 육성 + Falcon 제품 데모 + Fal.Con 콘텐츠 장기 보존

**채널 구성**
- Threat Intelligence Briefings (Adversary 분석 영상)
- Falcon Platform Demos (제품 기능 워크스루)
- Fal.Con Session Recordings (연례 컨퍼런스 전체 영상)
- CrowdStrike University (교육 콘텐츠)

---

### 채널 6 — 이메일 / 마케팅 자동화

**목적**: 리드 육성 + 파이프라인 가속 + 고객 리텐션

**시퀀스 구조**

| 퍼널 단계 | 트리거 | 이메일 시퀀스 |
|---------|--------|------------|
| 인식 | 위협 보고서 다운로드 | 웰컴 → 추가 인텔리전스 리소스 → 웨비나 초대 (3–5일 간격) |
| 검토 | 제품 페이지 방문 | 케이스 스터디 → ROI 계산기 → PoC 제안 (7일 간격) |
| 결정 | PoC 요청 또는 데모 신청 | SDR 콜 준비 → 경쟁사 비교 자료 → 레퍼런스 고객 소개 |
| 리텐션 | 기존 고객 | 분기별 위협 브리핑 → 신규 모듈 얼리 액세스 → QBR 초대 |

**KPI**: Open Rate, Click Rate, MQL 전환율, 이메일 기여 파이프라인

### Steal Sheet — 3가지

1. **"인바운드를 신뢰 자산으로 설계하라."** CrowdStrike의 위협 보고서 다운로드는 리드 캡처이면서 동시에 제품 가치 증명이다. 당신 브랜드의 퍼널 최상단에도 "읽는 것만으로 이미 우리 가치를 경험하는" 콘텐츠 자산이 있어야 한다 — 무료로 배포하는 인텔리전스가 가장 비용 효율적인 파이프라인 드라이버다.

2. **"자체 이벤트는 카테고리 소유권 선언이다."** Fal.Con 컨퍼런스는 CrowdStrike가 사이버보안 카테고리의 의제를 설정하는 연간 행사다. 자체 커뮤니티 이벤트를 가진 브랜드는 경쟁사 이벤트에서 뒤따르지 않아도 된다 — 고객이 당신의 이벤트로 모이는 순간 카테고리 리더십이 확정된다.

3. **"채널을 퍼널 단계에 묶어라."** CrowdStrike는 채널마다 역할이 다르다 — X는 인식, LinkedIn은 관심, 이메일은 육성, 직접 영업은 클로징. 모든 채널에서 같은 메시지를 쏘는 것은 예산 낭비다. 각 채널의 청중 심리와 구매 준비도를 파악하고 해당 단계에 맞는 콘텐츠를 배치하라.
