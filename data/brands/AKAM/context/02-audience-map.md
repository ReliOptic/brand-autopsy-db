## 페르소나

### 페르소나 1: 박지훈 — 엔터프라이즈 CISO

| 항목 | 내용 |
|------|------|
| 직함 | Chief Information Security Officer |
| 소속 산업 | 금융서비스 / 대형 은행 |
| 나이 | 44세 |
| 경력 | IT 보안 18년, CISSP·CISM 보유 |
| 보고 라인 | CTO 직속 보고, 이사회 분기 보고 |
| 연간 예산 | 보안 인프라 예산 300억~500억 원 (추정) |

**핵심 목표**
- 제로트러스트 아키텍처로 내부 위협 및 랜섬웨어 차단
- 규제 준수(FSC, ISO 27001, PCI-DSS) 유지
- 경영진에게 사이버 리스크를 비즈니스 언어로 보고

**페인포인트**
- 하루 수백만 건의 봇 트래픽과 API 공격 대응 인력 부족
- 레거시 방화벽과 클라우드 네이티브 보안 툴 간 통합 복잡성
- DDoS 공격 시 서비스 중단으로 인한 평판 리스크

**구매 트리거**
- 동종업계 대형 침해 사고 발생 후 이사회 긴급 지시
- 새로운 금융 규제 시행 6개월 전 대응 필요성

**선호 채널**
- Gartner MQ·Forrester Wave 리포트
- CISO 전용 컨퍼런스 (RSA Conference, Gartner Security Summit)
- 동료 CISO 레퍼런스 콜

**Akamai 관련 관심 제품**
- Guardicore Segmentation
- App & API Protector (WAAP)
- MFA·Zero Trust Network Access (ZTNA)

### 페르소나 2: 이수연 — 미디어 플랫폼 VP Engineering

| 항목 | 내용 |
|------|------|
| 직함 | VP of Engineering / Platform |
| 소속 산업 | OTT 스트리밍 / 미디어 |
| 나이 | 37세 |
| 경력 | 백엔드·인프라 엔지니어링 12년 |
| 팀 규모 | 인프라 팀 30명 직접 관리 |
| 핵심 KPI | P99 레이턴시, 스트림 버퍼링 비율, CDN 비용/GB |

**핵심 목표**
- 라이브 스포츠 이벤트 시 수천만 동시접속 처리
- 스트리밍 품질(4K HDR) 유지하면서 CDN 비용 20% 절감
- 멀티 CDN 전략으로 벤더 종속성 탈피

**페인포인트**
- 예측 불가능한 트래픽 스파이크(월드컵, 올림픽)에서 CDN 장애
- 글로벌 PoP 확장 시 지역별 레이턴시 격차
- 광고 삽입(Ad Insertion) 레이턴시로 인한 시청자 이탈

**구매 트리거**
- 경쟁사의 CDN 장애로 자사 서비스 차별화 기회 인식
- 새로운 스트리밍 시장(동남아·중동) 진출 계획

**선호 채널**
- 기술 블로그 (InfoQ, The New Stack)
- GitHub 레포·오픈소스 기여 확인
- AWS re:Invent·KubeCon 등 기술 컨퍼런스

**Akamai 관련 관심 제품**
- Adaptive Media Delivery
- mPulse Real User Monitoring
- EdgeWorkers (엣지 컴퓨팅)

### 페르소나 3: 김민준 — 공공기관 IT 인프라 과장

| 항목 | 내용 |
|------|------|
| 직함 | IT 인프라 과장 / 정보화전략팀 |
| 소속 산업 | 중앙정부 부처 / 공공기관 |
| 나이 | 41세 |
| 경력 | 공공 IT 행정 15년, 정보처리기사 보유 |
| 예산 구조 | 국가예산 기반, 연간 입찰·조달 프로세스 |
| 규제 환경 | 국가정보원 보안 가이드라인, 전자정부법 |

**핵심 목표**
- 전자정부 서비스 99.9% 이상 가용성 유지
- 선거·재난 등 트래픽 급증 이벤트 대응
- 국산·검증 솔루션 우선 원칙 속 글로벌 기술 도입 정당화

**페인포인트**
- 조달 프로세스 복잡성으로 신기술 도입 지연
- 보안 인증(CC인증·국정원 검토) 요구로 글로벌 벤더 접근 장벽
- 내부 기술 인력 부족으로 복잡한 솔루션 운영 어려움

**구매 트리거**
- 타 부처·지자체 사이버 공격 사례 발생 후 예산 승인
- 정부 디지털 전환 중장기 계획 수립 시점

**선호 채널**
- 나라장터·공공 조달 플랫폼
- 행안부·국정원 공식 세미나
- 기존 SI 파트너(삼성SDS, LG CNS) 레퍼런스

**Akamai 관련 관심 제품**
- DDoS 방어 (Prolexic)
- Bot Manager
- Edge DNS

## 구매 의사결정 구조 (B2B)

```
[비즈니스 오너 / C-Level] — 예산 승인 + 전략 방향
        ↓
[IT/보안 리더 (CISO·CTO·VP Eng)] — 기술 요구사항 정의
        ↓
[평가팀 (아키텍트·보안엔지니어)] — POC·벤치마크 수행
        ↓
[조달·구매팀] — 계약 협상·SLA 검토
        ↓
[최종 승인] → 계약 체결
```

**평균 영업 사이클**: 6~18개월 (엔터프라이즈 기준, 추정)

## 고객 여정 단계별 메시지

| 단계 | 고객 상태 | 핵심 메시지 | 콘텐츠 형식 |
|------|---------|-----------|-----------|
| 인식(Awareness) | "이런 위협이 있는 줄 몰랐다" | State of the Internet 리포트 배포 | 리서치 보고서, 인포그래픽 |
| 고려(Consideration) | "솔루션을 비교하고 있다" | Gartner MQ Leader 위치 증명 | 기술 백서, 비교 가이드 |
| 결정(Decision) | "POC를 진행 중이다" | ROI 계산기, 레퍼런스 케이스 | 고객 사례 연구, POC 지원 |
| 유지(Retention) | "갱신 시점이 다가온다" | 신규 위협 대응 업그레이드 | QBR 자료, 신제품 로드맵 |
| 옹호(Advocacy) | "동료에게 추천하고 싶다" | 공동 성공 사례 발표 기회 | 고객 사례 발표, 파트너 프로그램 |
