## 1. 채널 전략 개요

Amazon은 **"자체 플랫폼 최우선 + 외부 채널 유입 최소화"** 전략을 취한다.
소비자가 Google·소셜을 통해 진입하더라도 최종 전환은 Amazon.com 내에서 완결된다. (추정)

```
외부 인지 채널          중간 채널               자체 전환 채널
──────────             ──────────              ──────────────
TV 광고         →      Google 검색     →        Amazon.com
소셜 미디어     →      Amazon 광고     →        Amazon App
인플루언서      →      이메일 뉴스레터  →        Alexa 쇼핑
구전·리뷰       →      Prime Video    →        Amazon Go
```

## 2. 채널별 전략 상세

### 2-1. Amazon.com / Amazon App (자체 플랫폼)

| 항목 | 내용 |
|------|------|
| 역할 | 최종 전환 허브 — 모든 마케팅의 종착지 |
| 핵심 기능 | 검색, 개인화 추천, 리뷰, 1-Click 구매 |
| 트래픽 원 | 직접 접속 60%+ (추정) |
| 최적화 지표 | 전환율, 장바구니 이탈율, 평균 주문금액 |
| 특이점 | 플랫폼 자체가 채널이자 광고 매체 |

**Steal Sheet — 자체 플랫폼 채널 3종**
1. 검색창을 마케팅 채널로 활용 — Sponsored Products로 검색 결과 상단 점령, 외부 광고 없이 구매 의도 고객 포착
2. "오늘의 딜" 페이지를 독립 채널처럼 운영 — 반복 방문 습관 형성, 앱 알림 옵트인 증가
3. Subscribe & Save를 채널 락인 도구로 사용 — 반복 구매를 자동화하여 경쟁 채널 진입 차단

### 2-2. Amazon Advertising (자체 광고 플랫폼)

| 항목 | 내용 |
|------|------|
| 광고 유형 | Sponsored Products, Sponsored Brands, Sponsored Display, DSP |
| 매출 규모 | 약 $470억+ (2024 광고 수익, 공식) |
| 강점 | 구매 의도 데이터 기반, 경쟁자 대비 최고 전환율 |
| 타겟팅 | 키워드, 제품, 카테고리, 오디언스 행동 |
| ROI 특성 | Google·Meta 대비 구매 전환 우위 (추정) |

**Steal Sheet — Amazon 광고 채널 3종**
1. Sponsored Products는 소규모 예산으로 시작해도 즉각적 가시성 제공 — 신규 브랜드 진입 장벽 낮음
2. Amazon DSP로 Amazon 外 웹사이트에서도 리타게팅 — 자체 구매 데이터를 외부 채널에 투사
3. Brand Store 페이지를 D2C 사이트처럼 활용 — Amazon 내 브랜드 허브 구축으로 충성도 강화

### 2-3. 이메일 마케팅

| 항목 | 내용 |
|------|------|
| 유형 | 트랜잭션 이메일, 추천 뉴스레터, 딜 알림, 구독 갱신 |
| 개인화 수준 | 구매 이력·탐색 데이터 기반 초개인화 (추정) |
| 오픈율 | 트랜잭션 이메일 50%+, 마케팅 이메일 20-30% (추정) |
| 핵심 트리거 | 장바구니 이탈, 가격 하락, 재고 복구, 리뷰 요청 |

**Steal Sheet — 이메일 채널 3종**
1. 가격 하락 알림 이메일 — 위시리스트 상품 가격 변동 시 즉시 발송, 구매 의도 포착 타이밍 최적화
2. "당신이 본 상품과 유사한" 추천 이메일 — 탐색 데이터를 이메일로 전환, 수동적 발견 유도
3. Prime Day 사전 예고 이메일 시리즈 — D-7부터 카운트다운, 구매 기대감을 이메일 채널로 구축

### 2-4. 소셜 미디어

| 플랫폼 | 전략 | 콘텐츠 유형 | 팔로워 규모 |
|--------|------|------------|------------|
| Instagram | 제품·라이프스타일 비주얼 | 이미지, 릴스, 쇼핑 태그 | 수백만 (추정) |
| X (Twitter) | 고객 서비스, 딜 알림 | 텍스트, 빠른 반응 | 수백만 (추정) |
| Facebook | 광범위 오디언스 딜 공유 | 이미지, 동영상 | 수천만 (추정) |
| TikTok | MZ세대 제품 발견 | 숏폼 비디오, 인플루언서 콜라보 | 성장 중 (추정) |
| YouTube | Prime Video 트레일러, 브랜드 광고 | 롱폼, 트레일러 | 수백만 (추정) |
| LinkedIn | AWS B2B, 기업 PR | 아티클, 케이스 스터디 | 수백만 (추정) |

**Steal Sheet — 소셜 채널 3종**
1. TikTok Shop 연동 — Amazon 대항마로 성장 중인 TikTok 커머스를 역으로 활용, "TikTok에서 발견 → Amazon에서 구매" 흐름 유도
2. Instagram 쇼핑 태그 — 소셜 발견을 Amazon PDP로 직결하는 원터치 경로 구축
3. X(트위터) 고객 서비스 공개 대응 — 빠른 해결 사례를 공개 노출하여 브랜드 신뢰 증거로 전환

## 5. 채널 통합 시너지 맵

```
Prime Video 시청
      ↓ 시청 중 광고 노출
Amazon 광고 클릭
      ↓
Amazon.com PDP
      ↓ 구매 미완료
Alexa 리마인더 / 이메일 장바구니 알림
      ↓
재방문 → 구매 완료
      ↓
리뷰 요청 이메일
      ↓
UGC 생성 → PDP 신뢰도 상승 → 신규 고객 전환
```

Amazon의 채널 전략 핵심은 **자체 생태계 내 순환 구조 완성**이다. 외부 채널은 유입 역할에 그치며, 실제 가치는 자체 플랫폼 루프에서 창출된다. (추정)
