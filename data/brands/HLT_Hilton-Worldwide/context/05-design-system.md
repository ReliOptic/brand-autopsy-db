# 05 디자인 시스템 — Hilton Worldwide (HLT)

## 디자인 철학

Hilton의 디자인 언어는 **"따뜻한 현대성(Warm Modernity)"** 으로 요약된다. 클래식 호스피탈리티의 신뢰감과 현대적 미니멀리즘을 결합하며, 브랜드 포트폴리오별로 온도와 정교함의 다이얼을 조정한다.

## UI 컴포넌트 원칙

### 버튼

- **Primary CTA**: `#003580` 배경, `#FFFFFF` 텍스트, border-radius 4px (공식)
- **Secondary**: 투명 배경, `#003580` 테두리·텍스트
- **Destructive/Warning**: `#D32F2F` (`#D32F2F`) — 취소·경고 액션 전용

### 카드 & 레이아웃

- 호텔 카드: 16:9 이미지 비율, 하단 가격+CTA 레이아웃
- 여백 시스템: 8px 그리드 기반 (추정)
- 모서리 처리: 4~8px radius (과도한 라운드 지양)

### 아이콘

- 선 굵기: 1.5~2px 스트로크 (추정)
- 스타일: 기하학적 미니멀 — 과도한 장식 없음
- 힐튼 호너스 전용 아이콘셋: 티어 배지, 포인트 코인, 디지털 키 (공식)

## Steal Sheet — 디자인 레이어별 3대 탈취 전략

### 비주얼 아이덴티티 레이어에서 탈취

1. **Gold Accent 차별화**: `#C9A84C` 골드를 경쟁사(Marriott 실버·IHG 블루 계열)가 모방하기 전에 "Honors Gold = 보상의 컬러"로 고객 인식 고착화
2. **따뜻한 사진 스타일 독점**: Hyatt·Marriott의 쿨톤 인테리어 사진 대비 Hilton 워밍 팔레트를 일관 적용 — 소셜 피드에서 즉각 식별 가능
3. **Hampton Orange 에너지**: `#E8611A` 오렌지를 미드스케일 세그먼트 인식 앵커로 활용 — Holiday Inn 그린 대비 에너지·활기 포지셔닝

### UI/UX 레이어에서 탈취

1. **디지털 키 UX 선도**: 앱 내 객실 선택→디지털 키→체크아웃 3탭 완결 플로우를 경쟁사 대비 1~2단계 단순화 (추정 목표)
2. **포인트 가시화**: 홈 화면 상단에 포인트 잔액·다음 무료 숙박까지 필요 포인트 상시 노출 — Bonvoy 앱 대비 동기부여 UI 우위
3. **다국어 온보딩**: 아시아 시장(한국·일본·중국) 언어별 UX 미세조정 — IHG·Hyatt 대비 현지화 투자 우위 (추정)

### 공간 디자인 레이어에서 탈취

1. **로비 소셜 공간화**: Canopy·Tempo 로비를 "제3의 공간"으로 설계 — Airbnb가 제공 불가한 커뮤니티 경험 제공
2. **웰니스 룸 패키지**: 수면 특화 어메니티(조명·온도·침구 통합 제어) 표준화 — Hyatt의 웰니스 포지셔닝 견제
3. **F&B 로컬 디자인 통합**: 레스토랑·바를 호텔 브랜딩과 분리된 독립 로컬 콘셉트로 설계 — "호텔 레스토랑" 낙인 탈피
