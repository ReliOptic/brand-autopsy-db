# 02 오디언스 맵 — Morgan Stanley (MS)

## 오디언스 계층 구조

```
Primary (직접 클라이언트)
├── 기관 투자자 (Institutional)
├── 초고자산가 UHNW ($10M+)
└── 고자산가 HNW ($1M–$10M)

Secondary (의사결정 영향자)
├── 기업 CFO·재무팀
├── 연금·기금 수탁자
└── Family Office 운용역

Tertiary (에코시스템)
├── 규제기관·정책입안자
├── 금융 미디어·애널리스트
└── 잠재 FA 채용 인재
```

## 페르소나 상세

### 페르소나 1: "글로벌 기관 CIO" — David Chen

| 항목 | 내용 |
|------|------|
| 나이 | 52세 |
| 직책 | 아시아계 미국계 대형 연기금 CIO |
| 운용 AUM | $45B |
| 지역 | 뉴욕 / 싱가포르 듀얼 오피스 |
| 학력 | MIT 금융공학 석사 / CFA |

**핵심 니즈**
- 멀티에셋 포트폴리오의 알파 소스 다변화
- 지정학적 리스크 헤지 (중국·미국 탈동조화)
- ESG 수탁 책임 이행 보고서

**페인포인트**
- 유동성 위기 시 대형 블록 거래 실행 파트너 부재
- 규제 변화(Basel IV·SEC 기후공시) 대응 역량 요구
- 리서치 품질 대비 수수료 투명성 요구

**브랜드 접점**
- Morgan Stanley Research (블루페이퍼·GIC 리포트)
- Institutional Sales 데스크 직접 커버리지
- MS Matrix (기관 전용 포털)

**구매 트리거**
- 신규 섹터 딜 소싱 / IPO 참여 기회
- 맞춤 구조화 상품 (CLO·TRS)

**채널 선호**
- 전화·이메일 직접 소통 (담당 MD급 이상)
- 분기 리뷰 미팅 (on-site)

### 페르소나 2: "UHNW 2세대 상속인" — Sophia Müller-Park

| 항목 | 내용 |
|------|------|
| 나이 | 38세 |
| 직책 | 유럽계 제조업 가문 3세 / Family Office 오너 |
| 순자산 | $280M+ |
| 지역 | 런던 / 서울 |
| 학력 | LSE 경제학 BA / INSEAD MBA |

**핵심 니즈**
- 세대 간 자산 이전 전략 (Tax-efficient trust 구조)
- 임팩트 투자·ESG 테마 포트폴리오
- 아트·부동산·PE 대체투자 통합 뷰

**페인포인트**
- 부모 세대 FA와의 세대 갈등 (디지털 리터러시 격차)
- 다국적 자산에 대한 크로스-보더 세금 복잡성
- "단순히 부를 지키는 것" 이상의 임팩트 원함

**브랜드 접점**
- Morgan Stanley Private Wealth Management (PWM)
- Investing with Impact 플랫폼 (공식)
- FA 개인 관계망

**구매 트리거**
- 상속 이벤트 발생 시
- 임팩트 투자 신규 펀드 런칭

**채널 선호**
- FA 1:1 미팅 (디지털+오프라인 혼합)
- MS 앱 내 포트폴리오 대시보드

### 페르소나 3: "스타트업 창업자 Exit 후 신흥 HNW" — Jason Kwon

| 항목 | 내용 |
|------|------|
| 나이 | 41세 |
| 직책 | 시리즈D 테크 스타트업 창업자 (IPO 후 exit) |
| 유동 자산 | $12M (post-exit, lock-up 해제) |
| 지역 | 샌프란시스코 |
| 학력 | Stanford CS / Y Combinator 출신 |

**핵심 니즈**
- IPO 락업 해제 후 집중 포지션 리스크 분산
- 세금 최적화된 주식 처분 전략 (10b5-1 플랜)
- 벤처 LP 투자 지속을 위한 유동성 관리

**페인포인트**
- 전통 금융 언어에 대한 거부감 ("너무 올드스쿨")
- 기술 분야 이해도가 부족한 FA에 대한 불신
- 빠른 디지털 서비스 기대 vs. 전통 WM 프로세스 속도

**브랜드 접점**
- E*TRADE (초기 진입) → MS Private Wealth (업셀)
- Morgan Stanley at Work (주식 보상 관리 플랫폼) (공식)
- Shareworks by Morgan Stanley (스톡옵션 관리)

**구매 트리거**
- 창업사 M&A·IPO 이벤트
- 스톡옵션 베스팅 완료 시점
- 세금 신고 시즌 (Q1)

**채널 선호**
- 디지털 퍼스트 (앱·온라인)
- 콘텐츠 마케팅 (파운더 커뮤니티 뉴스레터)

## 고객 여정 맵 (UHNW 기준)

```
인지(Awareness)
→ Financial Times / WSJ 리서치 인용 노출
→ 동료 네트워크 추천

고려(Consideration)
→ MS Thought Leadership 리포트 다운로드
→ FA 소개 미팅 요청

평가(Evaluation)
→ 투자 제안서(IPS) 검토
→ 경쟁사(Goldman, JPM) 비교

결정(Decision)
→ FA 신뢰 관계 형성
→ 초기 포트폴리오 이전

유지(Retention)
→ 분기 리뷰 미팅
→ 이벤트 초청 (MS Ideas Conference 등)

확장(Expansion)
→ 추가 자산 이전
→ 자녀·가족 소개 (세대 이전)
```
