# 06 채널 플레이북 — Morgan Stanley (MS)

## 채널 전략 개요

Morgan Stanley의 채널 전략은 **"프리미엄 게이트키핑(Premium Gatekeeping)"** 원칙을 따른다. 브랜드의 접근성을 의도적으로 조절해 채널별 오디언스 품질을 유지하며, FA(재무 어드바이저) 관계망이 모든 채널의 수렴점이다.

```
채널 깔때기 구조

Public (브랜드 인지)
├── morganstanley.com
├── LinkedIn
├── Thoughts on the Market 팟캐스트
└── PR / 언론 커버리지

Qualified (리드 육성)
├── Insights 뉴스레터
├── 웨비나 / 이벤트
└── MS Research 포털

Engaged (관계 심화)
├── FA 1:1 커뮤니케이션
├── MS Matrix (기관 포털)
└── E*TRADE (디지털 리테일)

Retained (유지·확장)
├── MS 앱 (포트폴리오 관리)
├── Shareworks (주식 보상)
└── MS at Work (직장 복지)
```

## 채널별 상세 플레이북

### 채널 1: 공식 웹사이트 (morganstanley.com) (공식)

| 항목 | 내용 |
|------|------|
| 주요 목적 | 브랜드 허브, 리서치 배포, FA 연결 |
| 주요 오디언스 | 모든 세그먼트 (진입 게이트) |
| 콘텐츠 우선순위 | 리서치 리포트, 투자 전망, 서비스 소개 |
| CTA 구조 | "FA 찾기" > "리서치 보기" > "계좌 개설" |
| 업데이트 빈도 | 리서치: 일간 / 서비스: 분기 |
| KPI | 리서치 다운로드 수, FA 문의 연결률 |

**운영 원칙**
- 첫 화면(Above the Fold): 시장 인사이트 또는 주요 테마 1개
- 검색 최적화: 투자 관련 롱테일 키워드 (예: "2025 금리 전망 투자 전략")
- 접근성: WCAG 2.1 AA 준수 (공식 기업 정책)

### 채널 2: LinkedIn (공식 계정 운영)

| 항목 | 내용 |
|------|------|
| 팔로워 | 약 5M+ (추정, 2024년 기준) |
| 주요 목적 | B2B 브랜딩, 인재 채용, 사고 리더십 |
| 주요 오디언스 | 기관 클라이언트, 잠재 FA, 금융 전문가 |
| 게시 빈도 | 주 4–6회 (추정) |
| 콘텐츠 믹스 | 리서치 인사이트 40% / 임직원 스토리 25% / 채용 20% / ESG 15% |

**콘텐츠 포맷 우선순위**
1. 문서형 캐러셀 (리서치 요약 슬라이드)
2. 단문 인사이트 포스트 (데이터 포인트 1개 + 함의)
3. 임원 개인 계정 연동 포스트 (CEO·CIO 직접 게시)
4. 동영상 클립 (Thoughts on the Market 팟캐스트 영상 버전)

**금지 사항**
- 경쟁사 직접 언급 또는 비교
- 비공개 딜 정보 암시
- 지나치게 캐주얼한 밈·유머 콘텐츠

### 채널 4: E*TRADE 디지털 플랫폼 (공식 인수 브랜드)

| 항목 | 내용 |
|------|------|
| 주요 목적 | 리테일 투자자 획득, MS WM으로 업셀 |
| 주요 오디언스 | 자기주도 리테일 투자자, 스타트업 창업자 |
| 핵심 기능 | 주식 거래, 옵션, ETF, 퇴직계좌 |
| MS 통합 수준 | MS 브랜드와 E*TRADE 브랜드 병행 운영 (공식) |
| 업셀 경로 | E*TRADE Pro → Morgan Stanley Private Wealth |

**채널 전략**
- 리테일 진입 → 자산 성장 → FA 연결 3단계 퍼널
- "Morgan Stanley at Work" 연동으로 기업 복지 채널 확보
- 자동화 투자(Core Portfolios) 상품으로 진입 장벽 낮춤

- 심플: 복잡한 리포트를 FA가 클라이언트 언어로 번역 제공
