# 02. 오디언스 맵 — Linde plc (LIN)

## 오디언스 계층 구조

```
Primary (직접 구매·계약)
├── 대형 산업 고객 (철강·화학·정유)
├── 의료기관 (병원·제약사)
└── 에너지 전환 파트너 (수소·CCUS 프로젝트)

Secondary (영향력·추천)
├── 엔지니어링 컨설턴트 / EPC사
├── 규제기관·정부 (에너지정책)
└── ESG 투자자·애널리스트

Tertiary (인식·채용)
├── 공학·화학 전공 취업 준비생
└── 일반 대중 (지속가능성 관심층)
```
