## 배틀카드

### 배틀카드 1 — vs Abbott FreeStyle Libre (당뇨 CGM 경쟁)

**경쟁 상황:** 병원 내분비과 의사가 CGM 처방 시 MDT Guardian vs Libre 비교

| 구분 | MDT Guardian 4 | Abbott FreeStyle Libre 3 |
|------|---------------|--------------------------|
| 착용 방식 | 오버패치 필요 | 단독 부착 가능 |
| 보정 필요 여부 | 불필요 (공식) | 불필요 (공식) |
| 인슐린 펌프 연동 | MiniMed 780G 완전 통합 (공식) | 별도 펌프 없음 |
| 착용 기간 | 최대 7일 | 최대 14일 (공식) |
| 알람 커스터마이징 | 고급 설정 가능 | 기본 알람만 |
| 대상 환자 | 인슐린 펌프 사용자, 제1형 당뇨 | 인슐린 비사용 제2형 당뇨 |

**세일즈 피치:** "폐쇄루프 자동 인슐린 전달이 필요한 환자에게 Guardian 4는 시스템 전체의 뇌입니다. Libre는 단독 센서입니다."

**우리가 지는 상황:** 단순 혈당 모니터링만 원하는 제2형 당뇨 환자 → Libre 권장이 맞음. 무리한 경쟁 자제.

### 배틀카드 2 — vs Boston Scientific WATCHMAN (심장 경쟁)

**경쟁 상황:** 심방세동 환자의 뇌졸중 예방 옵션 논의 시 MDT vs BSX

| 구분 | MDT (Amulet 미보유, OAC 지원) | BSX WATCHMAN FLX |
|------|-------------------------------|------------------|
| 접근 전략 | 항응고제(OAC) 병용 최적화 | 좌심방이 폐색술 |
| 근거 수준 | 기존 약물 치료 풍부한 데이터 | PINNACLE FLX RCT 완료 (공식) |
| 시술 복잡도 | 약물 조정 (낮음) | 경도관 시술 (중간) |
| 장기 추적관리 | CareLink 원격 모니터링 | 별도 모니터링 필요 |
| MDT 강점 | 통합 심장 생태계 (CRM + 모니터링) | — |

**세일즈 피치:** "Medtronic의 심장 생태계는 WATCHMAN 이후에도 필요합니다. CRT-D·ICD와 함께 원스톱 모니터링을 제공하는 파트너는 저희뿐입니다."

**우리가 지는 상황:** OAC 금기 환자 → BSX WATCHMAN이 임상적으로 유리. 무리한 경쟁 자제.

### 배틀카드 3 — vs Stryker Mako (척추·수술 로봇 경쟁)

**경쟁 상황:** 병원 수술 로봇 도입 검토 위원회에서 MDT Hugo vs Stryker Mako

| 구분 | MDT Hugo RAS | Stryker Mako |
|------|-------------|--------------|
| 적응증 | 복강경 연부조직 수술 | 정형외과(무릎·고관절) |
| 시장 진입 시기 | 2021 CE 마크 (공식) | 2006 출시, 시장 선점 (공식) |
| 브랜드 인지도 | 신규 — 구축 중 | 글로벌 1위 정형외과 로봇 |
| 데이터 플랫폼 | Touch Surgery 통합 | Stryker OS |
| MDT 강점 | 연부조직 수술 전문성, 글로벌 서비스망 | — |
| 약점 | 정형외과 로봇 경험치 부족 | 소프트티슈 적응증 없음 |

**세일즈 피치:** "Hugo는 복강경·비뇨기·부인과 수술의 디지털 전환을 위한 플랫폼입니다. Mako가 뼈를 다룬다면, Hugo는 조직을 다룹니다. 둘은 경쟁이 아닌 상호보완입니다."

**우리가 지는 상황:** 정형외과 중심 병원의 로봇 도입 → Mako가 압도적 강점. 협업 프레이밍 전환.

## 경쟁 포지셔닝 맵 (추정)

```
         [혁신성 高]
              |
   BSX ●      |        ● MDT
              |
[전문성 低]----+----[전문성 高]
              |
   SYK ●      |    ● ABT
              |
         [혁신성 低]
```
- MDT: 높은 전문성 + 높은 혁신성 → 프리미엄 풀포트폴리오 플레이어
- SYK: 높은 전문성 + 낮은 혁신성 → 정형외과 도메인 지배자
- BSX: 낮은 전문성 + 높은 혁신성 → 니치 혁신 어태커
- ABT: 낮은 전문성 + 낮은 혁신성 → 소비자 접근성 특화

## Steal Sheet — 경쟁 레이어별 3가지 탈취 전략

### Layer 1: Abbott CGM 시장에서 빼앗아올 것
1. **제1형 당뇨 + 펌프 패키지 딜** — Guardian 4 + MiniMed 780G 번들 제안, Libre 단독 처방 환자 전환 유도
2. **내분비과 KOL 교육 투자** — Libre 처방 의사 대상 폐쇄루프 임상 데이터 심포지엄 개최
3. **환자 앱 경험 개선** — CareLink 앱 UX를 LibreLink 수준으로 업그레이드, 전환 마찰 제거

### Layer 2: Boston Scientific EP 시장에서 빼앗아올 것
1. **통합 EP + CRM 패키지** — EP 카테터 + ICD + 원격 모니터링 원스톱 계약, BSX 단품 경쟁 불리하게 만들기
2. **수련병원 펠로우십 독점 파트너십** — EP 펠로우 교육 프로그램을 MDT 기기로 표준화
3. **실시간 원격 지원 차별화** — 수술 중 MDT 전문가 원격 지원 서비스 → BSX 대비 유지보수 우위

### Layer 3: Stryker 척추 시장에서 빼앗아올 것
1. **척추 + 신경조절 번들 전략** — 척추 수술 후 만성통증 환자에게 SCS(척수자극기) 연계 처방 경로 구축
2. **중소병원 척추 임플란트 가격 공세** — Stryker 프리미엄 대비 10~15% 할인 계약 + 교육 지원 패키지
3. **디지털 수술 계획 툴 제공** — Stryker 전산화 계획 툴 없는 중소병원에 MDT 무상 소프트웨어 공급
