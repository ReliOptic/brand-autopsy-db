# 06 채널 플레이북 — Nike (NKE)

## 채널 전략 개요

Nike의 채널 전략은 2021년 이후 **DTC(Direct-to-Consumer) 우선** 체계로 전환되었다. Foot Locker 등 주요 홀세일 파트너 물량을 축소하고, Nike.com·Nike 앱·SNKRS 앱·NRC 앱을 통한 직접 판매 및 관계 구축에 집중한다 (공식).

---

## 채널 생태계 맵

```
                        Nike 브랜드 생태계
                               │
          ┌────────────────────┼────────────────────┐
          │                    │                    │
     디지털 직접            오프라인 직접          파트너 채널
     ──────────            ────────────          ──────────
     Nike.com              Nike 플래그십          선택적 홀세일
     Nike 앱               Nike By You           (Dick's, JD 등)
     SNKRS 앱              Nike Live (소형점)
     NRC / NTC 앱
     SNS (IG/TK/YT)
```

## 채널별 상세 플레이북

### 1. Nike.com / Nike 앱 (DTC 핵심)

| 항목 | 내용 |
|------|------|
| 역할 | 풀프라이스 판매, 멤버십 락인, 개인화 |
| 타깃 페르소나 | 박지훈 (퍼포먼스), 이수연 (라이프스타일) |
| 핵심 콘텐츠 | 제품 상세 페이지, 스타일 가이드, 선수 스토리 |
| 개인화 | 구매 이력·운동 데이터 기반 추천 (NRC 연동) |
| KPI | 전환율, 멤버십 가입수, 재구매율 |
| 금지 행동 | 할인 프로모션 남발, 비멤버 동일 혜택 |

---

### 2. SNKRS 앱

| 항목 | 내용 |
|------|------|
| 역할 | 한정판 드로우, 스니커 문화 커뮤니티, 희소성 관리 |
| 타깃 페르소나 | 김도현 (스트리트 컬처 팬) |
| 핵심 콘텐츠 | 드로우 공지, 릴리즈 스토리, 디자이너 인터뷰 |
| 드로우 구조 | 무작위 추첨 + 멤버 조기 접근 (SNKRS Pass) |
| KPI | 앱 DAU, 드로우 참여율, 출시 24h 완판율 |
| 금지 행동 | 과도한 재고 공급으로 희소성 희석 |

---

### 3. Nike Run Club (NRC) / Nike Training Club (NTC) 앱

| 항목 | 내용 |
|------|------|
| 역할 | 행동 데이터 수집, 커뮤니티 형성, 장기 락인 |
| 타깃 페르소나 | 박지훈 (퍼포먼스 러너) |
| 핵심 콘텐츠 | AI 코칭, 러닝 플랜, 챌린지, 코치 오디오 |
| 데이터 활용 | 러닝 데이터 → 제품 추천 → Nike.com 연결 |
| KPI | MAU, 세션 완료율, Nike.com 전환 연결율 |
| 금지 행동 | 앱 내 과도한 제품 광고 (경험 훼손) |

---

### 4. Instagram

| 항목 | 내용 |
|------|------|
| 역할 | 브랜드 감성 구축, 신제품 발견, 인플루언서 앰플리파이 |
| 타깃 페르소나 | 이수연 (라이프스타일), 김도현 (스트리트) |
| 포맷 믹스 | 피드 40% / 릴스 40% / 스토리 20% |
| 업로드 빈도 | 피드 주 4–5회, 스토리 매일 (추정) |
| 핵심 콘텐츠 | 선수 착용 컷, 출시 티저, OOTD 스타일링 |
| 인플루언서 전략 | 메가(선수) + 마이크로(러닝·피트니스 커뮤니티) 혼합 |
| KPI | 도달수, 저장수, 스토리 링크 클릭수 |

---

### 5. TikTok

| 항목 | 내용 |
|------|------|
| 역할 | Z세대 신규 인지 확보, 바이럴 문화 연결 |
| 타깃 페르소나 | 김도현 (21세), 이수연의 MZ 주변 |
| 포맷 | 15–60초 숏폼, 챌린지 해시태그 |
| 톤 | 브랜드보다 서브컬처 참여자에 가까운 어조 |
| 핵심 콘텐츠 | 선수 NG·비하인드, 신발 개봉기, 댄스·운동 챌린지 |
| KPI | 영상 완주율, 팔로워 증가, UGC 생성량 |
| 금지 행동 | 과도하게 다듬어진 광고형 영상 |

---

### 6. YouTube

| 항목 | 내용 |
|------|------|
| 역할 | 장편 스토리텔링, 다큐, 제품 기술 설명 |
| 타깃 페르소나 | 박지훈 (기술 탐색), 이수연 (영감 콘텐츠) |
| 포맷 | 60–120초 광고 + 5–20분 다큐 |
| 핵심 콘텐츠 | 선수 미니 다큐, 기술 혁신 영상, 캠페인 풀버전 |
| KPI | 조회수, 시청 지속 시간, 구독자 증가 |

---

### 7. 오프라인 — Nike 플래그십 스토어

| 항목 | 내용 |
|------|------|
| 역할 | 브랜드 체험, 개인화 서비스, 커뮤니티 이벤트 |
| 타깃 페르소나 | 세 페르소나 모두, 특히 고관여 구매자 |
| 주요 서비스 | Nike By You 커스터마이징, 발 스캔 핏 분석, 멤버 전용 행사 |
| 위치 전략 | 도심 핵심 상권 (서울 명동·강남 등) |
| 경험 원칙 | 스태프는 판매원이 아닌 운동 코치로 포지션 |
| KPI | 객단가, 멤버십 신규 가입, NPS |

## 채널별 콘텐츠 캘린더 원칙

| 시즌 | 핵심 채널 | 캠페인 방향 |
|------|-----------|------------|
| 1월–3월 (신년) | NRC 앱, Instagram | 목표 설정, 새 시즌 러닝 스타트 |
| 4월–6월 (봄 마라톤) | YouTube, NRC, Nike.com | 퍼포먼스 러닝, 대회 시즌 |

| 9월–11월 (Back to Sport) | 전 채널 | 시즌 최대 캠페인, 신제품 출시 |
| 12월 (홀리데이) | SNKRS, Nike.com | 한정판 기프트 세트, 드로우 |

## 채널 통합 고객 여정

```
인지 (TikTok/YouTube/Instagram)
    ↓
탐색 (Nike.com / 앱 / 오프라인 방문)
    ↓
전환 (Nike.com DTC / SNKRS 드로우 / 플래그십 구매)
    ↓
락인 (NRC 앱 가입 → 운동 데이터 → 재구매 추천)
    ↓
옹호 (UGC 공유 / 커뮤니티 챌린지 참여)
```

## Steal Sheet — 채널 플레이북 레이어

| # | 훔칠 요소 | 적용 방법 |
|---|-----------|-----------|
| 1 | **앱을 락인 도구로 활용** — NRC 데이터가 재구매를 유발 | 행동 데이터를 수집하는 앱을 설계하고, 그 데이터로 제품 추천 루프를 연결하라 |
| 2 | **채널별 페르소나 분리** — SNKRS는 김도현만, NRC는 박지훈만 | 하나의 채널에 모든 페르소나를 밀어넣지 말고 채널-페르소나를 1:1로 매핑하라 |
| 3 | **희소성 채널을 별도 생태계로 운영** | 드로우·한정판은 일반 이커머스와 완전히 분리된 UX·커뮤니케이션으로 설계하라 |
