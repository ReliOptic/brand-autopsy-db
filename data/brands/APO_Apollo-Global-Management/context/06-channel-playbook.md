## 1. 채널 전략 개요

Apollo의 채널 전략은 **"기관 퍼스트(Institutional First)"** 원칙 아래, 직접 관계 채널이 디지털 채널보다 우선한다. 대중 마케팅보다 **고밀도 접점 전략**을 통해 의사결정자와의 1:1 신뢰 구축에 집중한다.

### 채널 우선순위 매트릭스

| 채널 | 우선순위 | 주요 목적 | 대상 오디언스 |
|------|---------|---------|------------|
| 직접 LP 로드쇼 | 최우선 | 관계 구축, 투자 유치 | 대형 기관 LP |
| 기관 컨퍼런스 | 최우선 | 네트워킹, 브랜드 권위 | 전체 기관 투자자 |
| 공식 웹사이트 | 높음 | 정보 허브, 신뢰 기반 | 모든 오디언스 |
| LinkedIn | 높음 | 사고 리더십, 채용 | 금융 전문가 |
| 미디어 관계(PR) | 높음 | 브랜드 권위, 위기 관리 | 일반 금융 시장 |
| 어닝콜/IR | 높음 | 투자자 관계, 투명성 | 주주, 분석가 |
| 트위터(X) | 중간 | 마켓 코멘터리 | 금융 커뮤니티 |
| 팟캐스트 | 중간 | 심층 콘텐츠, 리치 확장 | 전문 투자자 |
| 유튜브 | 낮음 | 교육 콘텐츠 | 차세대 LP |

## 2. 채널별 상세 플레이북

### 채널 1: 직접 LP 로드쇼 (Direct LP Roadshow)

**목적**: 신규 펀드 레이징 및 기존 LP 관계 심화

**실행 원칙**
- 시니어 파트너 직접 참여 필수 (부문 대표 이상)
- 맞춤형 프레젠테이션 — LP별 포트폴리오 구성에 최적화
- 1:1 또는 소그룹(5인 이하) 미팅 선호
- 미팅 후 48시간 내 팔로업 자료 발송

**메시지 우선순위**
1. 영구 자본 구조적 우위
2. 크레딧 오리지네이션 독점 역량
3. Athene 통합 안정성
4. 운용 팀 연속성 및 거버넌스 강화

**KPI**
- 미팅당 투자 약정 전환율 (추정 목표: 15-25%)
- 재방문 LP 비율 (추정 목표: 80%+)

### 채널 3: 공식 웹사이트 (apollo.com)

**구조 원칙 (추정)**

```
홈 (브랜드 포지셔닝)
├── 전략 (크레딧 / PE / 부동산 / 인프라)
├── 포트폴리오 (주요 투자 사례)
├── 인사이트 (리서치·백서·코멘터리)
├── Athene (보험 통합 서비스)
├── Investor Relations (IR 자료)
└── About (팀·거버넌스·ESG)
```

**SEO 콘텐츠 전략**
- "대체 자산", "사모 크레딧", "영구 자본" 관련 롱테일 키워드
- 분기별 시장 전망 보고서로 유기적 유입 확보
- 경쟁사 비교 랜딩 페이지 (Apollo vs 시장 평균) (추정)

**전환 경로**
- IR 문의 → 담당자 연결 → 미팅 예약
- 인사이트 구독 → 이메일 뉴스레터 → 딥 인게이지먼트
