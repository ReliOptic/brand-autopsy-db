# 02 오디언스 맵 — Amcor (AMCR)

## 타깃 오디언스 개요

Amcor는 순수 B2B 기업으로, 직접 소비자(B2C)가 아닌 기업 구매자와 의사결정자를 대상으로 한다. 오디언스는 산업별, 역할별, 지역별로 세분화된다.

---

## 핵심 산업 세그먼트

| 세그먼트 | 비중(추정) | 주요 제품군 |
|---------|-----------|-----------|
| 식품·음료 | 45% | 플렉시블 파우치, 병뚜껑, 식품 트레이 |
| 의약품·헬스케어 | 25% | 블리스터 팩, 의료기기 포장, 멸균 포장 |
| 홈케어·퍼스널케어 | 15% | 튜브, 스탠드업 파우치, 캡슐 포장 |
| 담배 | 10% | 경질 포장, 필름 |
| 기타 산업재 | 5% | 산업용 포장 |

**고통 포인트**
- 규제 변화(EU 포장재 규정, 단일사용 플라스틱 금지)에 대응하는 포장 전환 압박

**정보 채널**
- Packaging Digest, Packaging World 구독
- Pack Expo, FACHPACK 참가
- LinkedIn 전문가 네트워크

**Amcor와의 접점**
- 신제품 개발 시 공동 R&D 참여
- 연간 공급 계약 갱신 협상
- 지속가능성 솔루션 공동 개발

## 의사결정 구조 (DMU: Decision-Making Unit)

```
경영진 (최종 승인)
    └── CPO / VP Packaging (전략 결정)
            ├── Packaging Director (사양 결정) ← 페르소나 1
            ├── R&D / Innovation Manager (기술 평가)
            ├── Procurement Manager (가격·공급 협상) ← 페르소나 3
            ├── Regulatory Affairs Manager (컴플라이언스) ← 페르소나 2
            └── Sustainability Manager (ESG 부합성)
```
