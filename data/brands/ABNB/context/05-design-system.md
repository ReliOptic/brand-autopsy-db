# 05 디자인 시스템 — Airbnb (ABNB)

## 디자인 철학

Airbnb의 디자인 언어는 "따뜻하고 명료한 환대(Warm Clarity)"를 기반으로 한다.
DLS(Design Language System)를 통해 일관된 사용자 경험을 전 플랫폼에 적용한다. (공식)

### Airbnb Cereal VF 특성 (공식)

- Variable Font 방식으로 단일 파일에 다중 웨이트 포함
- 자체 개발 폰트로 브랜드 독자성 확보 (2018년 공개)
- 산세리프 기반, 친근하고 현대적인 인상
- 웨이트: Book / Medium / Bold / ExtraBold / Black

## Steal Sheet — 디자인 시스템 레이어별 3가지 훔칠 전략

### Layer 1: 컬러 전략
1. **단일 브랜드 레드의 집중 사용** — `#FF385C`를 CTA에만 써서 클릭 유도 신호로 각인시킨다. 브랜드 컬러를 "희소하게" 쓸수록 시선이 자동 집중된다.
2. **그라디언트로 브랜드 개성 강화** — 단색보다 레드→퍼플 그라디언트가 기억에 더 잘 남는다. 앱 아이콘·온보딩 첫 화면에 집중 배치해 첫인상을 확보한다.
3. **뉴트럴 배경으로 콘텐츠 집중** — `#F7F7F7` 배경은 숙소 사진을 자연스럽게 돋보이게 만든다. 배경을 비우면 콘텐츠가 말한다.

### Layer 2: 타이포그래피 전략
1. **자체 폰트 개발로 브랜드 자산화** — Airbnb Cereal은 라이센스 비용 절감 + 브랜드 고유성 확보 + Variable Font로 파일 크기 최적화까지 달성. 장기적으로 자체 폰트는 강력한 브랜드 자산이다.
2. **Book 웨이트로 따뜻함 연출** — 본문에 Bold 대신 Book을 써서 권위적이지 않은, 친근한 대화 느낌을 만든다.
3. **타입 스케일 토큰화** — 디자인 토큰으로 관리하면 다크 모드·접근성 폰트 크기 조정을 코드 한 줄로 처리한다.

### Layer 3: 컴포넌트 전략
1. **카드에 위시리스트 하트 즉각 노출** — 예약 전 "저장" 행동을 유도해 재방문 고리를 만든다. 하트 탭 한 번이 재방문 알림 채널 확보로 이어진다.
2. **검색 바를 첫 화면 중앙에** — 목적지를 입력하는 순간 사용자가 여행을 상상하기 시작한다. 검색이 곧 여행 시작의 의식(ritual)이다.
3. **Lottie 마이크로 애니메이션으로 감정 연결** — 체크인 확정 시 반짝이는 애니메이션 하나가 "예약 완료"를 축하 이벤트로 만든다. 기능적 피드백을 감정적 순간으로 전환한다.
