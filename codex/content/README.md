the canonical archive under `data/brands/`.

Purpose:

- demonstrate editorial quality targets before merging into the mainline archive
- compare `gold-reference-style` writing against the current repository outputs
- keep experimental content isolated from production research files

Current pilot set:

- `ABNB_Airbnb`
- `MMM_3M`
- `AFL_Aflac`
